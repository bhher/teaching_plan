### 📘 이론

#### 1.1 C 언어란?
- **절차지향 프로그래밍 언어**
- 1972년 데니스 리치가 개발
- 시스템 프로그래밍에 적합
- 운영체제, 임베디드 시스템 등에 사용

#### 1.2 C 언어의 특징
- **컴파일 언어**: 소스코드를 기계어로 변환 후 실행
- **저수준 언어**: 메모리 직접 제어 가능
- **빠른 실행 속도**: 컴파일된 바이너리 실행
- **이식성**: 다양한 플랫폼에서 사용 가능

#### 1.3 C vs Java 비교

| 항목 | C | Java |
|------|---|------|
| 타입 | 컴파일 언어 | 컴파일 + 인터프리터 |
| 메모리 관리 | 수동 (malloc/free) | 자동 (GC) |
| 포인터 | 직접 사용 | 참조만 사용 |
| 실행 속도 | 빠름 | 상대적으로 느림 |
| 용도 | 시스템 프로그래밍 | 애플리케이션 개발 |

#### 1.4 컴파일 과정
```
소스코드(.c) → 전처리 → 컴파일 → 어셈블리 → 링크 → 실행파일(.exe)
```

**단계별 설명:**
1. **전처리**: `#include`, `#define` 처리
2. **컴파일**: C 코드를 어셈블리 코드로 변환
3. **어셈블**: 어셈블리 코드를 기계어로 변환
4. **링크**: 라이브러리와 연결

#### 1.5 개발 환경 설정
- **컴파일러**: GCC (MinGW-w64)
- **에디터**: VS Code
- **확장**: C/C++ Extension

**출력:**
```

```

#### 실습 2: 여러 줄 출력


### 🎯 핵심 정리
- `#include <stdio.h>`: 표준 입출력 라이브러리 포함
- `int main()`: 프로그램의 시작점
- `printf()`: 화면에 출력하는 함수
- `return 0`: 프로그램 정상 종료

## 📅 2주차 – 변수와 자료형

### 📌 학습 목표
- C 언어의 기본 자료형 이해
- 변수 선언과 초기화 방법 습득
- 서식 지정자 사용법 익히기

### 📘 이론

#### 2.1 자료형 종류

**정수형:**
- `int`: 정수 (보통 4바이트, -2,147,483,648 ~ 2,147,483,647)
- `long`: 긴 정수 (8바이트)
- `short`: 짧은 정수 (2바이트)
- `char`: 문자 (1바이트, -128 ~ 127)

**실수형:**
- `float`: 단정밀도 실수 (4바이트)
- `double`: 배정밀도 실수 (8바이트)

**부호:**
- `signed`: 부호 있음 (기본값)
- `unsigned`: 부호 없음 (0 이상만)

#### 2.2 변수 선언과 초기화

```c
// 선언만
int age;
float height;

// 선언과 동시에 초기화
int age = 20;
float height = 170.5;
char grade = 'A';
```

#### 2.3 서식 지정자 (Format Specifier)

| 서식 지정자 | 의미 | 예시 |
|------------|------|------|
| `%d` | 정수 (int) | `printf("%d", 10);` |
| `%f` | 실수 (float) | `printf("%f", 3.14);` |
| `%lf` | 실수 (double) | `printf("%lf", 3.14);` |
| `%c` | 문자 (char) | `printf("%c", 'A');` |
| `%s` | 문자열 | `printf("%s", "Hello");` |
| `%ld` | 긴 정수 (long) | `printf("%ld", 100L);` |

### 🧪 실습

#### 실습 1: 기본 자료형 사용

```c
#include <stdio.h>

int main() {
    int age = 20;
    float height = 170.5;
    char grade = 'A';
    
    printf("나이: %d\n", age);
    printf("키: %.1f\n", height);
    printf("등급: %c\n", grade);
    
    return 0;
}
```

**출력:**
```
나이: 20
키: 170.5
등급: A
```

#### 실습 2: 여러 변수 출력

```c
#include <stdio.h>

int main() {
    int num1 = 10, num2 = 20;
    float avg = 15.5;
    
    printf("첫 번째 수: %d\n", num1);
    printf("두 번째 수: %d\n", num2);
    printf("평균: %.2f\n", avg);
    
    return 0;
}
```

#### 실습 3: 실수 출력 정밀도 제어

```c
#include <stdio.h>

int main() {
    float pi = 3.14159;
    
    printf("기본: %f\n", pi);
    printf("소수점 2자리: %.2f\n", pi);
    printf("소수점 4자리: %.4f\n", pi);
    
    return 0;
}
```

**출력:**
```
기본: 3.141590
소수점 2자리: 3.14
소수점 4자리: 3.1416
```

### 🎯 핵심 정리
- 변수는 사용 전에 반드시 선언해야 함
- 자료형에 맞는 서식 지정자 사용
- 실수 출력 시 `.숫자`로 소수점 자리수 제어 가능
