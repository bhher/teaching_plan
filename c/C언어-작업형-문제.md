# C언어 작업형 평가 문제

## 문제 1: 배열의 최댓값과 최솟값 찾기

### 문제 설명
5개의 정수를 입력받아 배열에 저장한 후, 최댓값과 최솟값을 찾아 출력하는 프로그램을 작성하세요.

### 입력 형식
- 첫 번째 줄에 5개의 정수가 공백으로 구분되어 입력됩니다.
- 각 정수는 -1000 이상 1000 이하입니다.

### 출력 형식
- 첫 번째 줄에 "최댓값: [값]" 형식으로 출력합니다.
- 두 번째 줄에 "최솟값: [값]" 형식으로 출력합니다.

### 예시 입력
```
10 5 20 8 15
```

### 예시 출력
```
최댓값: 20
최솟값: 5
```

### 힌트
- 배열을 선언하고 반복문으로 입력받으세요.
- 최댓값과 최솟값을 찾을 때는 첫 번째 값을 초기값으로 설정하고 비교하세요.

### 배점 (총 5점)
1. 컴파일 오류 없음 (1점)
2. 배열 선언 및 입력 처리 (1점)
3. 최댓값 찾기 로직 (1점)
4. 최솟값 찾기 로직 (1점)
5. 출력 형식 정확성 (1점)

## 문제 2: 평균 이상인 값의 개수

### 문제 설명
10개의 정수를 입력받아 배열에 저장한 후, 평균을 계산하고 평균 이상인 값의 개수를 출력하는 프로그램을 작성하세요.

### 입력 형식
- 첫 번째 줄에 10개의 정수가 공백으로 구분되어 입력됩니다.
- 각 정수는 0 이상 100 이하입니다.

### 출력 형식
- 첫 번째 줄에 "평균: [값]" 형식으로 출력합니다. (소수점 둘째 자리까지)
- 두 번째 줄에 "평균 이상인 값의 개수: [개수]" 형식으로 출력합니다.

### 예시 입력
```
85 90 75 60 95 80 70 65 88 92
```

### 예시 출력
```
평균: 80.00
평균 이상인 값의 개수: 5
```

### 힌트
- 평균을 계산할 때는 합계를 구한 후 개수로 나누세요.
- 실수 나눗셈을 위해 형변환을 사용하세요: `(float)sum / 10`
- 평균 이상인 값을 찾을 때는 반복문과 조건문을 사용하세요.

### 배점 (총 5점)
1. 컴파일 오류 없음 (1점)
2. 배열 선언 및 입력 처리 (1점)
3. 평균 계산 로직 (1점)
4. 평균 이상인 값 개수 계산 로직 (1점)
5. 출력 형식 정확성 (1점)

## 문제 3: 문자열에서 특정 문자 개수 세기

### 문제 설명
문자열과 하나의 문자를 입력받아, 해당 문자열에 그 문자가 몇 번 나타나는지 세는 프로그램을 작성하세요. (대소문자 구분)

### 입력 형식
- 첫 번째 줄에 문자열이 입력됩니다. (최대 100자)
- 두 번째 줄에 찾을 문자 하나가 입력됩니다.

### 출력 형식
- "[문자]가 나타나는 횟수: [개수]" 형식으로 출력합니다.

### 예시 입력
```
Hello World
l
```

### 예시 출력
```
l가 나타나는 횟수: 3
```

### 예시 입력 2
```
Programming
m
```

### 예시 출력 2
```
m가 나타나는 횟수: 2
```

### 힌트
- 문자열은 문자 배열로 선언하세요: `char str[101];`
- `strlen()` 함수를 사용하여 문자열의 길이를 구하세요.
- 반복문으로 각 문자를 확인하고 조건문으로 비교하세요.

### 배점 (총 5점)
1. 컴파일 오류 없음 (1점)
2. 문자열 및 문자 입력 처리 (1점)
3. 문자열 길이 구하기 (1점)
4. 문자 개수 세기 로직 (1점)
5. 출력 형식 정확성 (1점)

## 문제 4: 배열에서 짝수와 홀수 분리하기

### 문제 설명
8개의 정수를 입력받아 배열에 저장한 후, 짝수와 홀수를 분리하여 각각 출력하는 프로그램을 작성하세요. 짝수는 먼저 출력하고, 그 다음 홀수를 출력합니다.

### 입력 형식
- 첫 번째 줄에 8개의 정수가 공백으로 구분되어 입력됩니다.
- 각 정수는 -100 이상 100 이하입니다.

### 출력 형식
- 첫 번째 줄에 "짝수: " 뒤에 짝수들을 공백으로 구분하여 출력합니다.
- 두 번째 줄에 "홀수: " 뒤에 홀수들을 공백으로 구분하여 출력합니다.
- 짝수나 홀수가 없으면 해당 줄에 "없음"을 출력합니다.

### 예시 입력
```
1 2 3 4 5 6 7 8
```

### 예시 출력
```
짝수: 2 4 6 8
홀수: 1 3 5 7
```

### 예시 입력 2
```
1 3 5 7 9 11 13 15
```

### 예시 출력 2
```
짝수: 없음
홀수: 1 3 5 7 9 11 13 15
```

### 힌트
- 나머지 연산자(`%`)를 사용하여 짝수/홀수를 판별하세요.
- 짝수와 홀수를 각각 다른 배열에 저장하거나, 출력 형식을 조정하세요.
- 출력 시 마지막 공백을 주의하세요.

### 배점 (총 5점)
1. 컴파일 오류 없음 (1점)
2. 배열 선언 및 입력 처리 (1점)
3. 짝수 분리 로직 (1점)
4. 홀수 분리 로직 (1점)
5. 출력 형식 정확성 (1점)
