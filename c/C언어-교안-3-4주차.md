## 📅 3주차 – 입력 & 출력

### 📌 학습 목표
- `scanf()` 함수 사용법 습득
- 입력 버퍼 개념 이해
- 다양한 자료형 입력 처리

### 📘 이론

#### 3.1 scanf() 함수
- **기능**: 표준 입력(키보드)에서 데이터 읽기
- **형식**: `scanf("서식", &변수)`
- **주의**: 변수 앞에 `&` (주소 연산자) 필수!

#### 3.2 서식 지정자와 변수 타입

| 서식 지정자 | 변수 타입 | 예시 |
|------------|----------|------|
| `%d` | int | `scanf("%d", &num);` |
| `%f` | float | `scanf("%f", &height);` |
| `%lf` | double | `scanf("%lf", &value);` |
| `%c` | char | `scanf("%c", &ch);` |
| `%s` | char 배열 | `scanf("%s", name);` |

#### 3.3 입력 버퍼 개념
- 입력 버퍼: 키보드 입력을 임시 저장하는 공간
- `\n` (엔터)도 버퍼에 저장됨
- 연속 입력 시 주의 필요

### 🧪 실습

#### 실습 1: 정수 입력

```c
#include <stdio.h>

int main() {
    int num;
    
    printf("정수를 입력하세요: ");
    scanf("%d", &num);
    printf("입력값: %d\n", num);
    
    return 0;
}
```

#### 실습 2: 여러 값 입력

```c
#include <stdio.h>

int main() {
    int age;
    float height;
    
    printf("나이와 키를 입력하세요: ");
    scanf("%d %f", &age, &height);
    printf("나이: %d, 키: %.1f\n", age, height);
    
    return 0;
}
```

#### 실습 3: 문자 입력

```c
#include <stdio.h>

int main() {
    char grade;
    
    printf("등급을 입력하세요: ");
    scanf(" %c", &grade);  // 공백 주의!
    printf("등급: %c\n", grade);
    
    return 0;
}
```

**주의사항:**
- 문자 입력 전에 공백을 넣어야 이전 입력의 `\n`을 무시함
- `scanf(" %c", &grade);` ← 공백 중요!

#### 실습 4: 입력 버퍼 문제 해결

```c
#include <stdio.h>

int main() {
    int num;
    char ch;
    
    printf("정수 입력: ");
    scanf("%d", &num);
    
    // 버퍼 비우기
    while(getchar() != '\n');
    
    printf("문자 입력: ");
    scanf("%c", &ch);
    printf("정수: %d, 문자: %c\n", num, ch);
    
    return 0;
}
```

### 🎯 핵심 정리
- `scanf()` 사용 시 변수 앞에 `&` 필수
- 여러 값 입력 시 공백으로 구분
- 문자 입력 시 버퍼 문제 주의

## 📅 4주차 – 연산자

### 📌 학습 목표
- 산술, 대입, 비교, 논리 연산자 이해
- 증감 연산자 사용법 습득
- 연산자 우선순위 파악

### 📘 이론

#### 4.1 산술 연산자

| 연산자 | 의미 | 예시 | 결과 |
|--------|------|------|------|
| `+` | 덧셈 | `5 + 3` | 8 |
| `-` | 뺄셈 | `5 - 3` | 2 |
| `*` | 곱셈 | `5 * 3` | 15 |
| `/` | 나눗셈 | `5 / 3` | 1 (정수) |
| `%` | 나머지 | `5 % 3` | 2 |

**주의:**
- 정수끼리 나눗셈은 정수 결과
- 실수 결과를 원하면 하나라도 실수여야 함: `5.0 / 3` → 1.666...

#### 4.2 대입 연산자

```c
int a = 10;
a += 5;  // a = a + 5 (15)
a -= 3;  // a = a - 3 (12)
a *= 2;  // a = a * 2 (24)
a /= 4;  // a = a / 4 (6)
a %= 5;  // a = a % 5 (1)
```

#### 4.3 비교 연산자

| 연산자 | 의미 | 예시 | 결과 |
|--------|------|------|------|
| `==` | 같다 | `5 == 3` | 0 (거짓) |
| `!=` | 다르다 | `5 != 3` | 1 (참) |
| `>` | 크다 | `5 > 3` | 1 (참) |
| `<` | 작다 | `5 < 3` | 0 (거짓) |
| `>=` | 크거나 같다 | `5 >= 3` | 1 (참) |
| `<=` | 작거나 같다 | `5 <= 3` | 0 (거짓) |

**C 언어에서:**
- 참(True): 0이 아닌 값 (보통 1)
- 거짓(False): 0

#### 4.4 논리 연산자

| 연산자 | 의미 | 예시 |
|--------|------|------|
| `&&` | AND (그리고) | `(a > 0) && (b > 0)` |
| `\|\|` | OR (또는) | `(a > 0) \|\| (b > 0)` |
| `!` | NOT (아니다) | `!(a > 0)` |

#### 4.5 증감 연산자

```c
int a = 10;

a++;    // 후위 증가: a = 11 (나중에 증가)
++a;    // 전위 증가: a = 12 (먼저 증가)

a--;    // 후위 감소: a = 11
--a;    // 전위 감소: a = 10
```

**차이점:**
```c
int a = 10, b;

b = a++;  // b = 10, a = 11 (값을 먼저 대입 후 증가)
b = ++a;  // a = 12, b = 12 (증가 후 값을 대입)
```

#### 4.6 연산자 우선순위

1. `()` (괄호)
2. `++`, `--` (증감)
3. `*`, `/`, `%` (곱셈, 나눗셈, 나머지)
4. `+`, `-` (덧셈, 뺄셈)
5. `<`, `>`, `<=`, `>=` (비교)
6. `==`, `!=` (동등 비교)
7. `&&` (논리 AND)
8. `||` (논리 OR)
9. `=` (대입)

### 🧪 실습

#### 실습 1: 산술 연산

```c
#include <stdio.h>

int main() {
    int a = 10, b = 3;
    
    printf("a + b = %d\n", a + b);
    printf("a - b = %d\n", a - b);
    printf("a * b = %d\n", a * b);
    printf("a / b = %d\n", a / b);
    printf("a %% b = %d\n", a % b);
    
    return 0;
}
```

**출력:**
```
a + b = 13
a - b = 7
a * b = 30
a / b = 3
a % b = 1
```

#### 실습 2: 비교 연산

```c
#include <stdio.h>

int main() {
    int a = 10, b = 3;
    
    printf("a > b: %d\n", a > b);
    printf("a < b: %d\n", a < b);
    printf("a == b: %d\n", a == b);
    printf("a != b: %d\n", a != b);
    
    return 0;
}
```

#### 실습 3: 논리 연산

```c
#include <stdio.h>

int main() {
    int age = 20;
    int score = 85;
    
    printf("나이가 18 이상이고 점수가 60 이상: %d\n", 
           (age >= 18) && (score >= 60));
    printf("나이가 65 이상이거나 점수가 90 이상: %d\n", 
           (age >= 65) || (score >= 90));
    
    return 0;
}
```

#### 실습 4: 증감 연산자

```c
#include <stdio.h>

int main() {
    int a = 10;
    
    printf("초기값: %d\n", a);
    printf("a++: %d\n", a++);  // 출력 후 증가
    printf("현재값: %d\n", a);
    printf("++a: %d\n", ++a);  // 증가 후 출력
    printf("현재값: %d\n", a);
    
    return 0;
}
```

### 🎯 핵심 정리
- 정수 나눗셈은 정수 결과 (실수 결과 원하면 실수 사용)
- 비교 연산 결과: 참(1), 거짓(0)
- 증감 연산자 전위/후위 차이 이해
