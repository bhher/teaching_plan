# 티스토리 블로그 태그 목록

## 📌 태그 사용 가이드

- 각 포스트마다 **5-10개**의 태그를 적절히 조합하여 사용
- 너무 많은 태그는 오히려 역효과가 있을 수 있으니 핵심 태그만 선택
- 검색 빈도가 높은 태그와 특정 주제 태그를 조합
- 티스토리는 태그를 쉼표(,)로 구분하여 입력

## 🎨 태그 조합 전략

### 1. 핵심 기술 태그 (필수)
- HTML, CSS, React, Java 중 해당하는 것

### 2. 주제 태그 (필수)
- 해당 포스트의 핵심 주제 (예: Flexbox, useState, 객체지향)

### 3. 난이도/대상 태그
- 기초, 입문, 실전 등

### 4. 카테고리 태그
- 프론트엔드, 백엔드, 풀스택

### 5. 검색 최적화 태그
- 웹개발, 프로그래밍, 코딩 등 인기 검색어

---

## 💡 태그 사용 팁

1. **일관성 유지**: 비슷한 주제의 포스트는 유사한 태그 사용
2. **검색어 고려**: 사용자가 검색할 만한 키워드 포함
3. **너무 길지 않게**: 태그는 간결하게 (2-4단어 이내)
4. **한글/영문 혼용**: 검색 범위를 넓히기 위해 필요시 혼용
5. **트렌드 반영**: 최신 기술 용어 포함
