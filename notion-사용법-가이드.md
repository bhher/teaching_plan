## Notion이란?

Notion은 **올인원 워크스페이스(All-in-one Workspace)** 도구로, 다음 기능들을 하나의 플랫폼에서 사용할 수 있습니다:

- 📝 문서 작성 및 편집
- 📊 데이터베이스 관리
- 📋 프로젝트 관리
- 🗂️ 지식 관리
- 👥 팀 협업
- 🔗 위키 및 문서화

### 주요 장점
- **유연성**: 블록 기반 구조로 자유롭게 구성 가능
- **통합성**: 여러 도구를 하나로 통합
- **협업**: 실시간 협업 및 댓글 기능
- **템플릿**: 다양한 템플릿 제공
- **무료 플랜**: 개인 사용자에게 충분한 무료 기능 제공

## 시작하기

### 1. 계정 생성
1. [notion.so](https://www.notion.so) 접속
2. "Start for free" 클릭
3. 이메일 또는 Google 계정으로 가입
4. 이메일 인증 완료

### 2. 워크스페이스 생성
- 개인 워크스페이스: 혼자 사용
- 팀 워크스페이스: 여러 사람과 협업

### 3. 첫 페이지 만들기
- 좌측 사이드바에서 "+ New page" 클릭
- 페이지 제목 입력
- 템플릿 선택 (또는 빈 페이지로 시작)

## 기본 사용법

### 페이지 생성 방법
1. **사이드바에서 생성**
   - 좌측 사이드바 "+ New page" 클릭
   
2. **현재 페이지 내에서 생성**
   - `/page` 입력 후 Enter
   - 또는 "Turn into" → "Page"

3. **하위 페이지 생성**
   - 기존 페이지를 드래그하여 다른 페이지 아래로 이동
   - 또는 페이지 제목 위에 마우스 오버 → "⋮" → "Add a page inside"

### 텍스트 편집
- 기본적으로 WYSIWYG(What You See Is What You Get) 방식
- 마우스로 클릭하여 바로 편집 가능
- **Bold**: `Ctrl + B` (Mac: `Cmd + B`)
- **Italic**: `Ctrl + I` (Mac: `Cmd + I`)
- **Underline**: `Ctrl + U` (Mac: `Cmd + U`)
- **Strikethrough**: `Ctrl + Shift + X` (Mac: `Cmd + Shift + X`)

### 마크다운 단축키
Notion에서는 마크다운 스타일의 단축키를 사용하여 빠르게 포맷을 적용할 수 있습니다:

- `'#' + 'SpaceBar'`: 제목1
- `'##' + 'SpaceBar'`: 제목2
- `'###' + 'SpaceBar'`: 제목3
- `'숫자' + '.' + 'SpaceBar'`: 번호 목록
- `'-' + 'SpaceBar'`: 글머리 기호 목록
- `'[' + ']' + 'SpaceBar'`: 할 일 목록

> 💡 **사용 예시**: 새 줄에서 `#`을 입력하고 스페이스바를 누르면 제목1 블록이 생성됩니다.

### 블록 삭제
- 블록 왼쪽에 마우스 오버 → "⋮" 클릭 → "Delete"
- 또는 블록 선택 후 `Backspace` 또는 `Delete` 키

## 블록 사용법

Notion은 **블록(Block)** 기반으로 구성됩니다. 모든 콘텐츠는 블록으로 만들어집니다.

### 블록 추가 방법
1. **슬래시 명령어 사용** (`/`)
   - 원하는 위치에 `/` 입력
   - 블록 타입 선택 (예: `/heading`, `/todo`, `/table`)

2. **마우스 사용**
   - 블록 왼쪽에 "+" 표시 클릭
   - 원하는 블록 타입 선택

### 주요 블록 타입

#### 텍스트 블록
- **Heading 1/2/3**: 제목 (`/h1`, `/h2`, `/h3`)
- **Text**: 일반 텍스트
- **Bulleted List**: 불릿 리스트 (`/bullet`)
- **Numbered List**: 번호 매기기 리스트 (`/numbered`)
- **Toggle List**: 접기/펼치기 리스트 (`/toggle`)
- **Quote**: 인용구 (`/quote`)
- **Callout**: 강조 박스 (`/callout`)
- **Code**: 코드 블록 (`/code`)

#### 미디어 블록
- **Image**: 이미지 (`/image`)
  - 이미지 드래그 앤 드롭 가능
  - URL로도 추가 가능
- **Video**: 비디오 (`/video`)
- **Audio**: 오디오 (`/audio`)
- **File**: 파일 첨부 (`/file`)
- **Embed**: 임베드 (`/embed`)
  - YouTube, Google Maps, Figma 등 다양한 서비스 지원

#### 콘텐츠 블록
- **Divider**: 구분선 (`/divider`)
- **Table**: 표 (`/table`)
- **Database**: 데이터베이스 (`/database`)
  - Table, Board, Timeline, Calendar, Gallery, List 등
- **Link to Page**: 페이지 링크 (`/link`)
- **Mention**: 멘션 (`@`)
- **Equation**: 수식 (`/equation`)
- **Bookmark**: 북마크 (`/bookmark`)

#### 기능 블록
- **To-do List**: 체크리스트 (`/todo`)
- **Toggle**: 접기/펼치기 (`/toggle`)
- **Template Button**: 템플릿 버튼 (`/template`)
- **Synced Block**: 동기화 블록 (`/sync`)

### 블록 변환
- 블록 왼쪽 "⋮" 클릭 → "Turn into" → 원하는 블록 타입 선택
- 또는 블록 선택 후 `/turn` 입력

### 블록 이동
- 블록 왼쪽 "⋮" 아이콘을 드래그하여 이동
- 또는 블록 선택 후 `Ctrl + Shift + ↑/↓` (Mac: `Cmd + Shift + ↑/↓`)

## 데이터베이스 활용

데이터베이스는 Notion의 핵심 기능 중 하나입니다.

### 데이터베이스 생성
1. `/database` 입력
2. 원하는 뷰 타입 선택:
   - **Table**: 표 형태
   - **Board**: 칸반 보드
   - **Timeline**: 타임라인
   - **Calendar**: 캘린더
   - **Gallery**: 갤러리
   - **List**: 리스트

### 속성(Property) 추가
- 각 데이터베이스 항목은 여러 속성을 가질 수 있습니다:
  - **Text**: 텍스트
  - **Number**: 숫자
  - **Select**: 선택 (드롭다운)
  - **Multi-select**: 다중 선택
  - **Date**: 날짜
  - **Person**: 사람
  - **Files & Media**: 파일/미디어
  - **Checkbox**: 체크박스
  - **URL**: 링크
  - **Email**: 이메일
  - **Phone**: 전화번호
  - **Formula**: 수식
  - **Relation**: 관계 (다른 데이터베이스 연결)
  - **Rollup**: 롤업 (관계된 데이터 집계)
  - **Created time**: 생성 시간
  - **Created by**: 생성자
  - **Last edited time**: 마지막 수정 시간
  - **Last edited by**: 마지막 수정자

### 뷰(View) 관리
- 같은 데이터베이스를 다양한 방식으로 볼 수 있습니다
- 뷰 추가: 데이터베이스 상단 "Add a view" 클릭
- 필터, 정렬, 그룹화 설정 가능

### 데이터베이스 예시 활용
- 📋 **프로젝트 관리**: 할 일 목록, 마일스톤 추적
- 📚 **지식 베이스**: 문서 모음, 위키
- 📊 **CRM**: 고객 정보 관리
- 📅 **일정 관리**: 이벤트, 미팅 스케줄
- 💰 **예산 관리**: 수입/지출 추적
- 📖 **독서 목록**: 읽은 책, 읽을 책 관리

## 페이지 구성

### 사이드바 구성
- **Favorites**: 즐겨찾기 (별표 표시한 페이지)
- **Private**: 개인 페이지
- **Shared**: 공유된 페이지
- **Workspace**: 팀 워크스페이스 페이지

### 페이지 설정
- 페이지 오른쪽 상단 "⋮" → "Settings & members"
- 또는 페이지 우측 상단 톱니바퀴 아이콘 클릭

### 페이지 속성
- **Cover**: 커버 이미지 추가
- **Icon**: 이모지 또는 이미지 아이콘 추가
- **Comments**: 댓글 활성화/비활성화
- **Font**: 폰트 변경 (Default, Serif, Mono)
- **Small text**: 작은 텍스트 모드

### 페이지 템플릿
- 페이지 설정 → "Templates"에서 템플릿 추가
- 자주 사용하는 페이지 구조를 템플릿으로 저장
- `/template` 블록으로 템플릿 버튼 생성 가능

### 페이지 연결
- **링크 연결**: `/link` 또는 `[[` 입력
- **데이터베이스 관계**: Relation 속성 사용
- **Mention**: `@` 입력하여 다른 페이지/사용자 멘션

## 협업 기능

### 멤버 초대
1. 페이지 우측 상단 "Share" 클릭
2. 이메일 주소 입력 또는 링크 공유
3. 권한 설정:
   - **Full access**: 전체 권한
   - **Can edit**: 편집 권한
   - **Can comment**: 댓글 권한
   - **Can view**: 읽기 권한

### 댓글 추가
- 블록 선택 → 우클릭 → "Comment"
- 또는 블록 왼쪽에 댓글 아이콘 클릭
- `@멤버`로 특정 멤버에게 알림

### 변경 사항 추적
- 페이지 우측 상단 "Updates" 아이콘 클릭
- 멘션, 댓글, 페이지 변경 사항 확인

### 페이지 히스토리
- 페이지 설정 → "Page history"에서 버전 히스토리 확인
- 이전 버전으로 되돌리기 가능 (Pro 플랜 이상)

## 고급 기능

### 템플릿 버튼
- `/template` 입력
- 템플릿 버튼 클릭 시 미리 정의된 구조의 콘텐츠 자동 생성

### 동기화 블록 (Synced Block)
- `/sync` 입력
- 같은 블록을 여러 페이지에서 공유
- 한 곳에서 수정하면 모든 곳에 반영

### 수식 (Formula)
- 속성에서 Formula 타입 선택
- 프로그래밍처럼 수식 작성 가능
- 예: `prop("수량") * prop("단가")`

### 롤업 (Rollup)
- Relation 속성과 함께 사용
- 관련 데이터베이스의 값을 집계
- 예: 프로젝트별 할 일 개수 계산

### 관계 (Relation)
- 두 데이터베이스를 연결
- 양방향 관계 설정 가능
- 예: 프로젝트 ↔ 할 일, 책 ↔ 저자

### 백링크 (Backlinks)
- 페이지 하단에 자동으로 표시
- 현재 페이지를 링크한 다른 페이지들 목록
- 페이지 간 관계 파악에 유용

### 워크스페이스 스페이스
- Unlimited 플랜에서 사용 가능
- 완전히 독립적인 워크스페이스 생성
- 팀/프로젝트별로 분리 관리

## 팁과 활용법

### 1. 효율적인 페이지 구조
- 상위 페이지를 카테고리로 구성
- 하위 페이지로 세부 내용 작성
- 데이터베이스로 항목 관리

### 2. 단축키 활용
- `/` 명령어를 자주 사용하여 빠르게 블록 추가
- `Ctrl + P`로 페이지 빠르게 찾기

### 3. 템플릿 활용
- 자주 사용하는 구조를 템플릿으로 저장
- 프로젝트 시작 시 템플릿 사용으로 시간 절약

### 4. 데이터베이스 뷰 활용
- 필터와 정렬로 원하는 정보만 보기
- 여러 뷰로 같은 데이터를 다각도로 분석

### 5. 태그 활용
- Multi-select 속성으로 태그 시스템 구축
- 필터링으로 관련 항목만 빠르게 찾기

### 6. 에임베드 활용
- 외부 도구를 Notion 페이지에 통합
- Google Sheets, Figma, Miro 등

### 7. 모바일 앱 활용
- iOS, Android 앱 제공
- 언제 어디서나 접근 가능

### 8. 백업
- 중요 데이터는 정기적으로 내보내기
- Settings → Export

### 9. 검색 활용
- `Ctrl + P` (Mac: `Cmd + P`)로 강력한 검색
- 페이지, 블록, 데이터베이스 항목 모두 검색

### 10. 공개 페이지
- 페이지를 웹에 공개 가능
- 포트폴리오, 블로그 등으로 활용

## 자주 묻는 질문 (FAQ)

### Q: 무료 플랜에서도 충분한가요?
A: 개인 사용자라면 무료 플랜으로도 충분히 활용 가능합니다. 팀 협업이 많거나 고급 기능이 필요하면 유료 플랜을 고려해보세요.

### Q: 오프라인에서 사용할 수 있나요?
A: 모바일 앱에서는 오프라인 모드가 지원됩니다. 데스크톱에서는 인터넷 연결이 필요합니다.

### Q: 데이터를 내보낼 수 있나요?
A: 네, Settings → Export에서 Markdown, HTML, PDF 등으로 내보낼 수 있습니다.

### Q: 다른 도구와 연동이 가능한가요?
A: API를 통해 다른 도구와 연동할 수 있습니다. Zapier, Make 등을 통한 자동화도 가능합니다.

### Q: 페이지를 비공개로 만들 수 있나요?
A: 네, 페이지 공유 설정에서 권한을 조절할 수 있습니다. 기본적으로 개인 페이지는 비공개입니다.
