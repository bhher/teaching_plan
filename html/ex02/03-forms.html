폼 요소 및 유효성 검사 예제

1. 기본 텍스트 입력

3자 이상 20자 이하로 입력하세요

2. 체크박스와 라디오 버튼

3. 선택 박스 (select)

Ctrl(또는 Cmd) 키를 누른 채로 여러 개 선택할 수 있습니다

4. 텍스트 영역 (textarea)

5. 버튼 (button)

버튼 타입:
  • type="submit": 폼 제출 (기본값)
  • type="reset": 폼 초기화
  • type="button": 일반 버튼 (JavaScript와 함께 사용)

6. 유효성 검사 예제

required 속성으로 필수 입력을 강제합니다
정규식 패턴으로 형식을 검사합니다
최소 3자, 최대 20자
1부터 100까지의 숫자만 입력 가능

7. 폼 접근성 체크리스트

  • ✅ 모든 입력 필드에 <label> 연결
  • ✅ for 속성과 id 속성 매칭
  • ✅ 필수 필드에 required 속성 사용
  • ✅ 필수 필드에 시각적 표시 (*)
  • ✅ 도움말 텍스트 제공
  • ✅ 적절한 input type 사용
  • ✅ 유효성 검사 메시지가 명확한가?
  • ✅ 키보드만으로 모든 기능 사용 가능한가?
