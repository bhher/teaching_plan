테이블 태그 및 접근성 예제

1. 기본 테이블 구조

가장 간단한 테이블 구조입니다.

홍길동 25 개발자

2. 테이블 섹션 (thead, tbody, tfoot)

테이블을 논리적으로 구분하여 구조를 명확히 합니다.

2024년 월별 매출 현황
월 매출액 비고
1월 1,000,000원 -
2월 1,200,000원 신제품 출시
3월 1,500,000원 -
합계 3,700,000원 -

코드 예시

<table>
  <caption>테이블 제목</caption>
  <thead>
    <tr>
      <th scope="col">제목 1</th>
    </tr>
  </thead>
  <tbody>
    <tr>
      <td>데이터</td>
    </tr>
  </tbody>
  <tfoot>
    <tr>
      <td>합계</td>
    </tr>
  </tfoot>
</table>

3. 셀 병합 (colspan, rowspan)

열 병합 (colspan)

이름 연락처
홍길동 010-1234-5678 hong@example.com
김철수 010-9876-5432

행 병합 (rowspan)

이름 1학기 2학기
중간고사 기말고사
홍길동 90 85
김철수 95 92

코드 예시

<!-- 열 병합 -->
<td colspan="2">2개 열을 합침</td>

<!-- 행 병합 -->
<td rowspan="2">2개 행을 합침</td>

4. 접근성: scope 속성

헤더 셀이 어떤 셀에 대한 헤더인지 명시합니다.

학생 성적표
이름 국어 영어 수학
홍길동 90 85 95
김철수 88 92 90
접근성 팁:
  • scope="col": 열 헤더
  • scope="row": 행 헤더
  • 스크린 리더가 테이블 구조를 더 잘 이해할 수 있습니다.

5. 복잡한 테이블: headers 속성

복잡한 테이블에서 셀이 어떤 헤더에 속하는지 명시합니다.

시간표
시간 월 화 수
09:00 수학 영어 국어
10:00 과학 체육 음악

코드 예시

<th id="mon" scope="col">월</th>
<th id="t1" scope="row">09:00</th>
<td headers="mon t1">수학</td>

6. 테이블 접근성 체크리스트

  • ✅ <th> 태그 사용 (헤더 셀)
  • ✅ <caption> 사용 (테이블 제목)
  • ✅ scope 속성 사용 (간단한 테이블)
  • ✅ headers 속성 사용 (복잡한 테이블)
  • ✅ thead, tbody, tfoot 사용
  • ❌ 테이블을 레이아웃 용도로 사용하지 않기
주의: 테이블은 데이터를 표시하는 용도로만 사용해야 합니다. 레이아웃을 만들기 위해 테이블을 사용하는 것은 권장하지 않습니다. 레이아웃은 CSS를 사용하세요.
