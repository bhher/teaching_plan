미디어 요소 및 임베드 예제

1. 오디오 (audio)

오디오 파일을 재생할 수 있습니다.

기본 오디오 플레이어

참고: 실제 오디오 파일이 없어 재생되지 않습니다.

코드 예시

<audio controls>
  <source src="audio.mp3" type="audio/mpeg">
  <source src="audio.ogg" type="audio/ogg">
  브라우저가 오디오 태그를 지원하지 않습니다.
</audio>

오디오 속성

  • controls: 재생 컨트롤 표시
  • autoplay: 자동 재생 (권장하지 않음)
  • loop: 반복 재생
  • muted: 음소거
  • preload: 미리 로드 (none, metadata, auto)
주의: autoplay 속성은 사용자 경험을 해칠 수 있으므로 신중하게 사용해야 합니다. 대부분의 브라우저는 자동 재생을 제한합니다.

2. 비디오 (video)

비디오 파일을 재생할 수 있습니다.

기본 비디오 플레이어

참고: 실제 비디오 파일이 없어 재생되지 않습니다.

코드 예시

<video controls width="640" height="360">
  <source src="video.mp4" type="video/mp4">
  <source src="video.webm" type="video/webm">
  브라우저가 비디오 태그를 지원하지 않습니다.
</video>

비디오 속성

  • controls: 재생 컨트롤 표시
  • width, height: 비디오 크기
  • poster: 썸네일 이미지
  • autoplay, loop, muted, preload: 오디오와 동일

썸네일 이미지 사용

<video controls poster="thumbnail.jpg">
  <source src="video.mp4" type="video/mp4">
</video>

3. 비디오 자막 (track)

비디오에 자막을 추가할 수 있습니다.

<video controls>
  <source src="video.mp4" type="video/mp4">
  <track kind="subtitles" src="subtitles-ko.vtt" srclang="ko" label="한국어" default>
  <track kind="subtitles" src="subtitles-en.vtt" srclang="en" label="English">
</video>
접근성: 자막은 청각 장애인과 다양한 언어 사용자를 위한 중요한 기능입니다. 가능하면 자막을 제공하세요.

4. iframe 기본 사용법

다른 웹 페이지를 현재 페이지에 임베드할 수 있습니다.

예제 웹사이트 임베드

코드 예시

<iframe 
  src="https://www.example.com" 
  title="예제 웹사이트"
  width="800" 
  height="450">
  브라우저가 iframe을 지원하지 않습니다.
</iframe>
접근성: title 속성은 필수입니다. 스크린 리더가 iframe의 목적을 이해할 수 있도록 합니다.

5. YouTube 비디오 임베드

YouTube 비디오를 웹페이지에 임베드할 수 있습니다.

YouTube 비디오 예제

사용 방법

  1. YouTube 비디오 페이지에서 "공유" 버튼 클릭
  2. "퍼가기" 탭 선택
  3. 제공된 코드 복사
  4. 웹페이지에 붙여넣기

코드 예시

<iframe 
  width="560" 
  height="315" 
  src="https://www.youtube.com/embed/VIDEO_ID" 
  title="YouTube video player" 
  frameborder="0" 
  allow="accelerometer; autoplay; clipboard-write; encrypted-media; gyroscope; picture-in-picture" 
  allowfullscreen>
</iframe>

7. iframe 보안: sandbox 속성

sandbox 속성으로 iframe의 기능을 제한할 수 있습니다.

<iframe 
  src="https://example.com" 
  sandbox="allow-scripts allow-same-origin"
  title="제한된 iframe">
</iframe>

sandbox 값

  • allow-scripts: JavaScript 실행 허용
  • allow-same-origin: 같은 출처로 취급
  • allow-forms: 폼 제출 허용
  • allow-popups: 팝업 허용
보안 주의: 신뢰할 수 없는 사이트를 iframe으로 임베드할 때는 sandbox 속성을 사용하여 보안을 강화하세요.

8. 미디어 접근성 체크리스트

  • ✅ 비디오에 자막 제공 (<track> 태그)
  • ✅ 오디오에 대체 텍스트나 전사본 제공
  • ✅ 자동 재생하지 않기 (사용자 경험 고려)
  • ✅ 컨트롤 제공 (controls 속성)
  • ✅ iframe에 title 속성 필수
  • ✅ 대체 콘텐츠 제공 (태그 내 텍스트)
  • ✅ 키보드 접근 가능한지 확인
