## HTML 실습 문제 모음 (난이도: 하)

### 문제 1 – 나만의 명함 페이지

- **문제 설명**  
  화면에 한 장짜리 “명함”처럼 보이는 간단한 페이지를 만든다.

- **요구사항**
  - 이름을 가장 눈에 띄게(큰 글자) 표시한다.
  - 이름 아래에 한 줄 소개 문장을 문단으로 작성한다.
  - 프로필 이미지를 1개 포함한다.
  - 이메일 주소를 클릭하면 메일 프로그램이 열리도록 링크를 만든다.
    - 형식: `mailto:example@example.com`
  - 휴대폰 번호를 텍스트로 표시한다.

- **힌트**
  - 이메일 링크도 일반 링크(`a`)와 동일하게 사용하지만, `href`에 `mailto:` 를 붙인다.
  - 이미지는 준비한 파일을 쓰거나, 임시 이미지 URL(placeholder)을 검색해서 사용해도 된다.

### 문제 2 – 좋아하는 것 설문 폼

- **문제 설명**  
  사용자가 “좋아하는 것”을 선택해서 제출할 수 있는 간단한 폼을 만든다.

- **요구사항**
  - 폼 안에 다음 입력 요소를 포함한다:
    - 이름 입력칸 (`input type="text"`)
    - 좋아하는 색을 입력하는 칸 (`input type="text"` 또는 `color`는 자유)
    - 좋아하는 취미를 여러 개 선택할 수 있는 체크박스(`checkbox`) 3개 이상
    - “다음” 버튼 하나 (`button` 또는 `input type="submit"`)
  - 이름과 좋아하는 색 칸은 **placeholder**를 사용해 어떤 내용을 쓰는지 힌트를 준다.
  - 이름은 **필수 입력**(`required`)으로 지정한다.

- **힌트**
  - 체크박스들은 같은 그룹이라도 `name`을 같게 할 필요는 없다.  
    각각의 항목이 “있다/없다” 정보이기 때문이다.
  - `<label>`을 사용해 각 입력칸이 무엇을 의미하는지 명확하게 표시하자.

### 문제 3 – 이미지 갤러리 + 링크

- **문제 설명**  
  사진(또는 그림) 3장을 보여주는 “작은 갤러리” 페이지를 만든다.

- **요구사항**
  - 페이지 제목을 텍스트로 표시한다.
  - 이미지를 3개 나란히(또는 세로로) 배치한다.
  - 각 이미지를 클릭하면, **관련된 외부 사이트**(예: 해당 사진과 관련된 블로그 글, 위키 문서 등)로 이동하도록 링크를 건다.
  - 이미지를 불러올 수 없을 때를 대비해, 각 이미지마다 **적절한 `alt` 텍스트**를 작성한다.

- **힌트**
  - 링크 안에 이미지를 넣는 형태로 구현한다. (이미지를 클릭하면 링크 이동)
  - 이미지가 너무 크면 `width` 속성이나 CSS로 크기를 조정할 수 있다.
  - 외부 링크는 `https://` 로 시작하는 전체 주소를 사용해야 한다.

### 문제 4 – 간단 로그인 폼 만들기

- **문제 설명**  
  아이디와 비밀번호를 입력받는 로그인 폼을 만든다.

- **요구사항**
  - 아이디, 비밀번호 두 칸을 갖는 폼을 만든다.
    - 아이디: `input type="text"`
    - 비밀번호: `input type="password"`
  - 각 입력칸 앞에 `label`을 사용해 무엇을 입력하는 칸인지 표시한다.
  - 아이디와 비밀번호 둘 다 **required** 속성을 사용해 필수 입력으로 만든다.
  - “자동 로그인” 체크박스를 하나 추가한다.
  - “로그인” 버튼을 하나 만든다.
  - 폼 아래에 “회원가입”으로 이동하는 링크를 하나 만든다. (예: `signup.html`로 이동)

- **힌트**
  - `label`의 `for`와 `input`의 `id`를 맞추면, 라벨을 클릭해도 입력칸에 포커스가 생긴다.
  - 자동 로그인은 실제 기능을 구현하지 않아도 되고, 체크박스 UI 연습이 목적이다.

### 문제 5 – 간단 설문조사 페이지

- **문제 설명**  
  사용자의 기본 정보를 입력받는 짧은 설문조사 페이지를 만든다.

- **요구사항**
  - 이름, 이메일, 나이 입력칸을 만든다.
    - 이름: `type="text"` (required)
    - 이메일: `type="email"` (required)
    - 나이: `type="number"` 또는 `text` (선택)
  - “수강 중인 과목”을 여러 개 선택할 수 있는 체크박스를 3개 이상 만든다.
  - “수강 만족도”를 1개만 선택하게 하는 라디오 버튼 그룹(`radio`)을 만든다.
    - 예: “만족 / 보통 / 불만족”
  - 제출 버튼을 하나 만든다.
  - 페이지 상단에는 텍스트로 설문조사 제목과 간단한 설명 문장을 작성한다.

- **힌트**
  - 라디오 버튼은 **같은 그룹끼리 `name` 속성을 동일하게** 설정해야 한다.
  - `placeholder`에는 예시 값을 적어 두면, 사용자가 어떤 형식으로 써야 할지 이해하기 쉽다.

### 문제 6 – 간단 이력서 상단 영역

- **문제 설명**  
  이력서 맨 위에 들어갈 “기본 정보 + 연락처” 영역을 만드는 연습을 한다.

- **요구사항**
  - 이름(텍스트), 직업/직무(텍스트)를 화면에 크게 표시한다.
  - 바로 아래에:
    - 프로필 이미지를 1개 표시한다.
    - 이메일, 전화번호, 개인 홈페이지(또는 GitHub, 블로그) 링크를 텍스트로 나열한다.
  - 이메일은 `mailto:`, 홈페이지/블로그는 일반 `https://` 링크로 만든다.
  - 연락처 정보는 문단 또는 목록 형태 중 하나를 선택해 정리한다.

- **힌트**
  - 나중에 CSS를 입힐 것을 생각하면서, **정보를 잘 나눠서 마크업**해 두는 것이 중요하다.
  - 이미지와 텍스트가 너무 붙어 있으면 `<br>` 이나 `<p>`를 사용해 적당히 간격을 준다.
