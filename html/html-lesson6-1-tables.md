## HTML 6-1차시 – 테이블 태그 기초

### 0. 학습 목표

- 표를 만드는 기본 태그인 `table`, `tr`, `th`, `td`의 역할을 이해한다.
- 간단한 표(시간표, 점수표 등)를 직접 만들 수 있다.
- `thead`, `tbody`, `tfoot`을 사용해 표 구조를 나눌 수 있다.
- 기본적인 `colspan`, `rowspan` 사용 예시를 이해한다. (간단 버전)

---

### 1. 테이블이란?

#### 1) 개념

- **Table(표)**: 행(row)과 열(column)로 이루어진 데이터 표
- HTML에서 표를 만들 때는 다음 태그들을 사용한다:
  - `<table>` : 표 전체
  - `<tr>` : 한 줄(행, table row)
  - `<th>` : 제목 셀(table header)
  - `<td>` : 일반 셀(table data)

#### 2) 간단 예시

```html
<table>
  <tr>
    <th>이름</th>
    <th>나이</th>
  </tr>
  <tr>
    <td>홍길동</td>
    <td>20</td>
  </tr>
  <tr>
    <td>김철수</td>
    <td>25</td>
  </tr>
</table>
```

> 첫 번째 줄에는 `<th>` 를 사용해 **열의 제목**을 표시,  
> 그 아래 줄들에는 `<td>` 로 실제 데이터를 적는다.

---

### 2. 테이블 기본 구조 이해하기

#### 2.1 태그 역할 정리

| 태그    | 의미                | 설명                                  |
|---------|---------------------|---------------------------------------|
| `table` | 표 전체             | 표 시작~끝을 감싸는 태그             |
| `tr`    | table row (행)      | 가로 한 줄                            |
| `th`    | table header (헤더) | 제목 셀, 기본적으로 굵게/가운데 정렬 |
| `td`    | table data (데이터) | 일반 데이터 셀                        |

#### 2.2 표 예시 – 점수표

```html
<h2>시험 점수표</h2>
<table border="1">
  <tr>
    <th>이름</th>
    <th>국어</th>
    <th>영어</th>
  </tr>
  <tr>
    <td>홍길동</td>
    <td>90</td>
    <td>85</td>
  </tr>
  <tr>
    <td>김철수</td>
    <td>80</td>
    <td>95</td>
  </tr>
</table>
```

> `border="1"` 은 테두리를 보기 위해 임시로 사용하는 속성입니다.  
> 실제 서비스에서는 보통 CSS로 테두리를 스타일링합니다.

---

### 3. `thead`, `tbody`, `tfoot` – 표 구조 나누기

#### 3.1 왜 나누나요?

- 표가 길어졌을 때:
  - **머리쪽(헤더)** 과 **몸통(데이터)**, **합계(푸터)** 를 구분하면  
    코드가 더 보기 쉽고, 스타일 지정도 편리해진다.

#### 3.2 구조 예시

```html
<table border="1">
  <thead>
    <tr>
      <th>이름</th>
      <th>국어</th>
      <th>영어</th>
    </tr>
  </thead>

  <tbody>
    <tr>
      <td>홍길동</td>
      <td>90</td>
      <td>85</td>
    </tr>
    <tr>
      <td>김철수</td>
      <td>80</td>
      <td>95</td>
    </tr>
  </tbody>

  <tfoot>
    <tr>
      <th>평균</th>
      <td>85</td>
      <td>90</td>
    </tr>
  </tfoot>
</table>
```

> 입문 단계에서는 **`thead` + `tbody` 정도만 알고** 있어도 충분합니다.  
> 합계/요약이 있을 때 `tfoot`을 추가로 사용하면 좋습니다.

### 4. 간단한 `colspan`, `rowspan` 소개

> 이 부분은 “이런 것도 있다” 정도로 **간단히 체험**하는 수준으로 다룹니다.

#### 4.1 `colspan` – 가로(열) 병합

- 여러 열을 **가로로 합쳐서** 하나의 셀로 만들 때 사용

```html
<table border="1">
  <tr>
    <th colspan="2">이름과 나이</th>
  </tr>
  <tr>
    <td>홍길동</td>
    <td>20</td>
  </tr>
</table>
```

#### 4.2 `rowspan` – 세로(행) 병합

- 여러 행을 **세로로 합쳐서** 하나의 셀로 만들 때 사용

```html
<table border="1">
  <tr>
    <th rowspan="2">반</th>
    <th>이름</th>
  </tr>
  <tr>
    <td>홍길동</td>
  </tr>
</table>
```

> 실제로는 조금 더 복잡한 계산이 필요하지만,  
> “하나의 칸이 위/아래 or 좌/우로 넓어질 수 있다” 정도만 이해해도 괜찮습니다.

### 5. 실습 – 나만의 간단 시간표 만들기

#### 5.1 목표

- **월~금 / 1~3교시** 정도의 작은 시간표를 HTML 테이블로 만든다.

#### 5.2 요구사항

- `<h2>`로 “나의 시간표” 제목을 작성한다.
- `<table>` 안에:
  - 첫 번째 줄:
    - 첫 칸: 비워두거나 “교시”
    - 나머지 칸: “월”, “화”, “수”, “목”, “금”
  - 그 아래 줄들:
    - 1교시, 2교시, 3교시 행을 만들고,
    - 각 칸에 과목 이름이나 활동 이름을 자유롭게 적는다.
- 최소 3행(1~3교시), 6열(교시 + 월~금) 이상 사용해 보기.

예시 구조(정답 아님, 참고용 글 설명):

```txt
        월   화   수   목   금
1교시   국어 영어 수학 과학 체육
2교시   ...  ...  ...  ...  ...
3교시   ...  ...  ...  ...  ...
```

#### 5.3 힌트

- 헤더 줄(요일)은 `<th>`를 사용하면 굵고 가운데 정렬되어 더 잘 보인다.
- 실제 표 모양은 나중에 CSS로 예쁘게 바꿀 수 있으니,  
  지금은 **행/열 구조를 정확히 쓰는 것**에 집중한다.

### 6. 추가 실습 아이디어 (강사용)

1. **좋아하는 과일/음식 표 만들기**
   - 열: “이름 / 좋아하는 정도(1~5) / 비고”  
   - 행: 학생들이 각자 3개 이상 작성

2. **회원 목록 표 만들기**
   - 열: “아이디 / 이름 / 이메일”  
   - 헤더 줄은 `<thead>`, 나머지는 `<tbody>`로 구분

---

### 7. 마무리 정리

- `table` = 표 전체, `tr` = 한 줄(행), `th` = 제목 셀, `td` = 데이터 셀
- `thead`, `tbody`, `tfoot`으로 표의 머리/몸통/발을 나눌 수 있다.
- `colspan`, `rowspan`으로 셀을 가로/세로로 합칠 수 있다.
- 오늘은 **구조(행/열)를 정확히 표현하는 연습**에 집중하고,  
  모양(색, 줄 간격, 정렬 등)은 이후 CSS에서 다룬다.
