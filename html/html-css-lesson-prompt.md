## HTML + CSS 연계 교안 작성용 프롬프트

아래 프롬프트를 그대로 복사해서, **주제만 바꿔서** 사용하면 됩니다.

---

> HTML + CSS 연계 교안을 만들어줘.  
>   
> **대상:**  
> - HTML 기본 태그를 조금 알고, CSS는 거의 처음인 입문자  
>   
> **주제:**  
> - (예: 프로필 카드 디자인, 간단한 블로그 레이아웃, 간단한 로그인 페이지 스타일링 등)  
>   
> **포함 내용(이론):**  
> - HTML 구조 복습 (사용할 태그만 간단히)  
> - CSS 적용 방법 3가지: inline, `<style>`, 외부 CSS 파일 (개념 설명, 이번 차시에서 실제로 쓸 방식 1가지 선택)  
> - 선택자 개념 (태그 선택자, 클래스 선택자, id 선택자)  
> - 박스 모델 기초 (width, height, padding, margin, border)  
> - 글자/색상/배경색 스타일링 기초 (font-size, color, background-color, text-align 등)  
>   
> **실습:**  
> - HTML 구조 먼저 만들기  
> - CSS를 순차적으로 입혀서:
>   - 글자 크기/색상 변경  
>   - 여백(padding/margin) 조정  
>   - 테두리(border) 적용  
>   - 레이아웃(가로 배치/세로 배치) 간단히 조정  
>   
> **필수 조건:**  
> - 강의용 md 파일로 정리  
> - “0. 학습 목표 / 1. 이론(슬라이드 흐름) / 2. 실습 단계 / 3. 코드 스켈레톤 / 4. 과제” 구조로 작성  
> - 정답 코드는 **스켈레톤 정도까지만** (완성본은 주석으로만 설명)  
> - 설명은 실제 수업에서 말하듯이, 쉬운 표현으로 작성  
> - HTML과 CSS 코드를 각각 어떤 파일에 넣는지(예: `index.html`, `style.css`) 명시
