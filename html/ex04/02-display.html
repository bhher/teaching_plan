Display 속성 예제

1. Block (블록)

특징:

  • 항상 새 줄에서 시작
  • 너비는 부모 요소의 100% (기본값)
  • 높이는 콘텐츠에 따라 결정
  • margin, padding, width, height 사용 가능
Block 요소 1
Block 요소 2
Block 요소 3
div { display: block; /* 기본값 */ width: 200px; height: 100px; margin: 10px; padding: 15px; }
블록 요소 예시: <div>, <p>, <h1>~<h6>, <section>, <article>, <ul>, <li> 등

2. Inline (인라인)

특징:

  • 같은 줄에 배치
  • 너비와 높이는 콘텐츠에 따라 결정
  • width, height 사용 불가
  • 상하 margin 사용 불가 (좌우만 가능)
Inline 요소 1 Inline 요소 2 Inline 요소 3 Inline 요소 4
span { display: inline; /* 기본값 */ /* width, height 사용 불가 */ margin: 0 10px; /* 좌우만 가능 */ padding: 5px 10px; /* 가능하지만 위아래는 영향 없음 */ }
주의: 인라인 요소에 width나 height를 설정해도 적용되지 않습니다.
인라인 요소 예시: <span>, <a>, <strong>, <em>, <img> 등

3. Inline-block (인라인 블록)

특징:

  • 인라인처럼 같은 줄에 배치
  • 블록처럼 width, height 사용 가능
  • margin, padding 모두 사용 가능
Inline-block 1
Inline-block 2
Inline-block 3
div { display: inline-block; width: 150px; height: 80px; margin: 10px; padding: 15px; }
사용 예시: 버튼을 가로로 나열, 카드 레이아웃, 네비게이션 메뉴

4. None vs Visibility: hidden

display: none

보이는 요소
숨겨진 요소 (display: none)
다음 요소 (공간 차지 안 함)

visibility: hidden

보이는 요소
숨겨진 요소 (visibility: hidden)
다음 요소 (공간은 차지함)
/* display: none */ .hidden1 { display: none; /* 공간 차지 안 함, 레이아웃에 영향 없음 */ } /* visibility: hidden */ .hidden2 { visibility: hidden; /* 공간은 차지, 레이아웃에 영향 있음 */ }
차이점:
  • display: none: 공간도 차지하지 않음
  • visibility: hidden: 공간은 차지하지만 보이지 않음

5. Display 비교표

특징 Block Inline Inline-block
줄 배치 새 줄 같은 줄 같은 줄
width/height ✅ 사용 가능 ❌ 사용 불가 ✅ 사용 가능
margin (상하) ✅ 사용 가능 ❌ 사용 불가 ✅ 사용 가능
padding ✅ 사용 가능 ⚠️ 좌우만 효과 ✅ 사용 가능
