박스 모델 예제

1. Padding (내부 여백)

콘텐츠와 테두리 사이의 공간입니다. 배경색이 적용됩니다.

padding: 20px; (모든 방향 20px)
padding: 10px 30px; (위아래 10px, 좌우 30px)
padding: 10px 20px 30px 40px; (위 오른쪽 아래 왼쪽)
개별 속성 사용 (top: 20px, right: 40px, bottom: 10px, left: 30px)
padding: 20px; /* 모든 방향 */ padding: 10px 30px; /* 위아래 좌우 */ padding: 10px 20px 30px; /* 위 좌우 아래 */ padding: 10px 20px 30px 40px; /* 위 오른쪽 아래 왼쪽 */ /* 개별 속성 */ padding-top: 20px; padding-right: 40px; padding-bottom: 10px; padding-left: 30px;

2. Margin (외부 여백)

테두리 밖의 공간입니다. 배경색이 적용되지 않습니다.

margin: 20px; (모든 방향 20px)
margin: 10px 30px; (위아래 10px, 좌우 30px)
margin: 0 auto; (중앙 정렬)
width: 300px
margin: 20px; /* 모든 방향 */ margin: 10px 30px; /* 위아래 좌우 */ margin: 0 auto; /* 중앙 정렬 */ /* 개별 속성 */ margin-top: 10px; margin-right: 20px; margin-bottom: 30px; margin-left: 40px;

3. Border (테두리)

패딩을 감싸는 선입니다. 배경색이 적용됩니다.

border: 3px solid #e74c3c;
방향별 테두리 (top: 5px solid, right: 3px dashed, bottom: 4px dotted, left: 2px solid)
border: 3px solid #e74c3c; /* 단축 속성 */ /* 개별 속성 */ border-width: 3px; border-style: solid; border-color: #e74c3c; /* 방향별 지정 */ border-top: 5px solid red; border-right: 3px dashed blue; border-bottom: 4px dotted green; border-left: 2px solid purple;

4. Box-sizing

요소의 크기 계산 방식을 지정합니다.

content-box (기본값)

width: 200px
padding: 20px
border: 5px
실제 너비: 250px
계산: 200px + 40px(padding) + 10px(border) = 250px

border-box (권장)

width: 200px
padding: 20px
border: 5px
실제 너비: 200px
계산: 200px (padding과 border 포함)
/* content-box (기본값) */ .content-box { box-sizing: content-box; width: 200px; padding: 20px; border: 5px solid #333; } /* 실제 너비: 200 + 40 + 10 = 250px */ /* border-box (권장) */ .border-box { box-sizing: border-box; width: 200px; padding: 20px; border: 5px solid #333; } /* 실제 너비: 200px */ /* 전역 설정 (권장) */ * { box-sizing: border-box; }
팁: 대부분의 경우 border-box를 사용하는 것이 편리합니다. 전역 설정으로 모든 요소에 적용하는 것을 권장합니다.

5. Margin Collapse (마진 겹침)

인접한 블록 요소의 상하 마진이 겹쳐서 하나의 마진으로 합쳐지는 현상입니다.

Box 1
margin-bottom: 30px
Box 2
margin-top: 20px
실제 간격: 30px (큰 값이 적용됨)
.box1 { margin-bottom: 30px; } .box2 { margin-top: 20px; } /* 실제 간격: 30px (큰 값) */
주의: 마진 겹침은 블록 요소의 상하 마진에만 발생합니다. 좌우 마진이나 인라인 요소에는 발생하지 않습니다.

6. 박스 모델 전체 구조

Content (콘텐츠)
실제 텍스트나 이미지가 표시되는 영역
↑ Padding (내부 여백)
↑ Border (테두리)
↑ Margin (외부 여백)
┌─────────────────────────────────┐ │ Margin (외부 여백) │ │ ┌───────────────────────────┐ │ │ │ Border (테두리) │ │ │ │ ┌─────────────────────┐ │ │ │ │ │ Padding (내부 여백) │ │ │ │ │ │ ┌───────────────┐ │ │ │ │ │ │ │ Content │ │ │ │ │ │ │ │ (콘텐츠) │ │ │ │ │ │ │ └───────────────┘ │ │ │ │ │ └─────────────────────┘ │ │ │ └───────────────────────────┘ │ └─────────────────────────────────┘

7. 개발자 도구로 박스 모델 확인하기

  1. F12로 개발자 도구 열기
  2. 요소 선택 도구 (Ctrl+Shift+C)로 요소 선택
  3. Elements 탭에서 선택된 요소 확인
  4. Computed 탭에서 박스 모델 시각화 확인
연습: 위의 예제 박스들을 개발자 도구로 선택하여 실제 margin, padding, border 값을 확인해보세요!
