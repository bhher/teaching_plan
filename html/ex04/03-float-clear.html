3. Clear 속성

Float의 영향을 제거합니다.

Float: left
Float: right

Float 요소들이 위에 있습니다. Clear 속성을 사용하면 Float의 영향을 받지 않습니다.

clear: both;
(양쪽 float 영향 제거)

Clear 요소 이후의 텍스트는 정상적으로 배치됩니다.

/* Clear 속성 */ .clear-left { clear: left; /* 왼쪽 float 제거 */ } .clear-right { clear: right; /* 오른쪽 float 제거 */ } .clear-both { clear: both; /* 양쪽 float 제거 (가장 많이 사용) */ }

4. Float 레이아웃 (과거 방식)

과거에는 Float를 사용하여 레이아웃을 만들었습니다.

Sidebar
float: left
width: 200px
Main Content
margin-left: 220px
/* 과거 Float 레이아웃 */ .sidebar { float: left; width: 200px; } .main { margin-left: 220px; /* sidebar 너비 + 간격 */ } /* 또는 */ .main { float: right; width: calc(100% - 200px); }
주의: Float는 레이아웃을 만들기 위한 도구가 아닙니다. 현대적 웹 개발에서는 Flexbox나 Grid를 사용하는 것을 권장합니다.

5. 현대적 대체 방법

Flexbox (권장)

Sidebar
Main Content
/* Flexbox */ .container { display: flex; gap: 20px; } .sidebar { width: 200px; } .main { flex: 1; /* 남은 공간 모두 차지 */ }

Grid (권장)

Sidebar
Main Content
/* Grid */ .container { display: grid; grid-template-columns: 200px 1fr; gap: 20px; }
권장: Flexbox와 Grid는 Float보다 훨씬 강력하고 예측 가능합니다.
