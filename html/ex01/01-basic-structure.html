HTML 기본 구조 예제

이 문서는 HTML의 기본 구조를 보여줍니다.

1. DOCTYPE 선언

문서의 첫 줄에 <!DOCTYPE html>이 있습니다.

2. HTML 요소

<html lang="ko">로 문서의 언어를 한국어로 지정했습니다.

3. HEAD 요소

HEAD 영역에는 메타데이터가 포함됩니다:

  • charset: 문자 인코딩 (UTF-8)
  • viewport: 반응형 웹 설정
  • description: 페이지 설명
  • title: 브라우저 탭 제목

4. BODY 요소

이 영역의 모든 내용이 화면에 표시됩니다.
