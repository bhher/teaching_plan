# HTML 기초 구조 및 시맨틱 마크업 예제

이 폴더는 HTML 기초 구조와 시맨틱 마크업을 학습하기 위한 예제 파일들을 포함합니다.

## 예제 파일 목록

### 1. `01-basic-structure.html`
- HTML 문서의 기본 구조 (DOCTYPE, head, body, meta)
- 메타 태그의 역할과 사용법

### 2. `02-semantic-tags.html`
- 시맨틱 태그 사용법
- header, nav, main, section, article, footer 태그 예제
- 각 태그의 역할과 사용 시나리오

### 3. `03-text-headings-paragraphs.html`
- 제목 태그 (h1~h6)의 계층 구조
- 문단 태그 (p) 사용법
- 텍스트 강조 태그 (strong, em)
- 줄바꿈 (br), 인용문 (blockquote, q) 태그

### 4. `04-links.html`
- 링크 태그 (a)의 다양한 사용법
- 외부 링크, 내부 링크, 이메일 링크, 전화 링크
- 접근성을 고려한 링크 텍스트 작성법

### 5. `05-images-accessibility.html`
- 이미지 태그 (img) 사용법
- alt 속성의 중요성과 올바른 작성법
- 접근성을 고려한 이미지 사용 가이드

### 6. `06-complete-example.html`
- 모든 내용을 종합한 완성 예제
- 실제 블로그 포스트 형태의 웹페이지
- 시맨틱 태그를 활용한 완전한 HTML 문서 구조

## 사용 방법

1. 각 HTML 파일을 웹 브라우저에서 열어보세요
2. 브라우저의 개발자 도구(F12)를 사용하여 소스 코드를 확인하세요
3. 각 예제의 코드를 수정해보며 학습하세요

## 학습 체크리스트

각 예제를 학습한 후 다음 사항을 확인하세요:

- [ ] DOCTYPE 선언을 이해했는가?
- [ ] head와 body의 역할을 구분할 수 있는가?
- [ ] 시맨틱 태그를 적절히 사용할 수 있는가?
- [ ] 제목 태그의 계층 구조를 올바르게 사용할 수 있는가?
- [ ] 링크와 이미지를 접근성을 고려하여 작성할 수 있는가?

## 참고 사항

- 모든 예제는 HTML5 표준을 따릅니다
- 접근성(accessibility)을 고려한 마크업을 지향합니다
- 시맨틱 태그 사용을 권장합니다
