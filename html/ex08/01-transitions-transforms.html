트랜지션과 트랜스폼 예제

1. 트랜지션 (Transition) 기본

CSS 속성 값이 변경될 때 부드럽게 전환됩니다.

기본 트랜지션

마우스를 올려보세요 (배경색 변경)
.button { background-color: #3498db; transition: background-color 0.3s; } .button:hover { background-color: #e74c3c; }

여러 속성 트랜지션

마우스를 올려보세요 (배경색, 크기, 모서리 변경)
.box { transition: background-color 0.3s, transform 0.5s, border-radius 0.3s; } .box:hover { background-color: #9b59b6; transform: scale(1.1); border-radius: 20px; }

타이밍 함수

마우스를 올려보세요 (ease 타이밍)
transition: transform 1s ease; /* 기본값 */ transition: transform 1s linear; /* 일정한 속도 */ transition: transform 1s ease-in; /* 느리게 시작 */ transition: transform 1s ease-out; /* 느리게 끝 */ transition: transform 1s ease-in-out; /* 느리게 시작하고 끝 */

지연 시간 (Delay)

마우스를 올려보세요 (0.3초 지연)
.box { transition: all 0.5s ease 0.3s; /* 0.3초 지연 */ }

2. 트랜스폼 (Transform) - 이동

요소를 이동시킵니다. 레이아웃에 영향을 주지 않습니다.

translate (이동)

마우스를 올려보세요 (오른쪽 위로 이동)
transform: translateX(20px); /* X축 이동 */ transform: translateY(-10px); /* Y축 이동 */ transform: translate(20px, -10px); /* X, Y 이동 */
장점: transform은 레이아웃에 영향을 주지 않아 다른 요소의 위치가 변경되지 않습니다. 또한 GPU 가속을 받아 성능이 좋습니다.

3. 트랜스폼 - 회전

요소를 회전시킵니다.

마우스를 올려보세요 (45도 회전)
transform: rotate(45deg); /* 45도 회전 */ transform: rotateX(45deg); /* X축 회전 (3D) */ transform: rotateY(45deg); /* Y축 회전 (3D) */ transform: rotateZ(45deg); /* Z축 회전 */

4. 트랜스폼 - 크기 조절

요소의 크기를 조절합니다.

마우스를 올려보세요 (1.2배 확대)
transform: scale(1.2); /* 1.2배 확대 */ transform: scaleX(1.2); /* X축만 확대 */ transform: scaleY(1.2); /* Y축만 확대 */ transform: scale(1.2, 0.8); /* X 1.2배, Y 0.8배 */

5. 트랜스폼 - 기울임

요소를 기울입니다.

마우스를 올려보세요 (기울임)
transform: skewX(15deg); /* X축 기울임 */ transform: skewY(15deg); /* Y축 기울임 */ transform: skew(15deg, 10deg); /* X, Y 기울임 */

6. 트랜스폼 조합

여러 변형을 조합할 수 있습니다.

마우스를 올려보세요 (이동 + 회전 + 확대)
transform: translate(20px, -10px) rotate(15deg) scale(1.1);
주의: 순서가 중요합니다. 오른쪽부터 적용됩니다. translate() rotate()와 rotate() translate()는 결과가 다릅니다.

7. transform-origin (변형 기준점)

변형의 기준점을 지정합니다.

마우스를 올려보세요 (왼쪽 위 기준 회전)
transform-origin: top left; /* 왼쪽 위 */ transform-origin: center; /* 중앙 (기본값) */ transform-origin: 50% 50%; /* 중앙 (퍼센트) */ transform-origin: 0 0; /* 왼쪽 위 (픽셀) */

8. 실전 예제: 버튼 호버 효과

트랜지션과 트랜스폼을 조합한 버튼 효과입니다.

.button { background: linear-gradient(135deg, #667eea 0%, #764ba2 100%); color: white; padding: 12px 24px; border-radius: 25px; transition: transform 0.2s, box-shadow 0.2s; box-shadow: 0 4px 15px rgba(102, 126, 234, 0.4); } .button:hover { transform: translateY(-2px); box-shadow: 0 6px 20px rgba(102, 126, 234, 0.6); }
