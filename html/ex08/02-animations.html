애니메이션 예제

1. @keyframes 기본

애니메이션의 키프레임을 정의합니다.

@keyframes fadeIn { from { opacity: 0; } to { opacity: 1; } } /* 또는 */ @keyframes fadeIn { 0% { opacity: 0; } 100% { opacity: 1; } }

2. 페이드 인 (Fade In)

페이드 인 애니메이션
@keyframes fadeIn { from { opacity: 0; } to { opacity: 1; } } .element { animation: fadeIn 1s ease-in-out; }

3. 슬라이드 인 (Slide In)

슬라이드 인 애니메이션
@keyframes slideIn { from { transform: translateX(-100%); opacity: 0; } to { transform: translateX(0); opacity: 1; } } .element { animation: slideIn 1s ease-out; }

4. 회전 (Spin)

회전 애니메이션 (무한 반복)
@keyframes spin { from { transform: rotate(0deg); } to { transform: rotate(360deg); } } .element { animation: spin 2s linear infinite; }
사용 예시: 로딩 스피너, 아이콘 회전 등

5. 바운스 (Bounce)

바운스 애니메이션
@keyframes bounce { 0%, 100% { transform: translateY(0); } 50% { transform: translateY(-30px); } } .element { animation: bounce 1s ease-in-out infinite; }

6. 펄스 (Pulse)

펄스 애니메이션
@keyframes pulse { 0%, 100% { transform: scale(1); opacity: 1; } 50% { transform: scale(1.1); opacity: 0.8; } } .element { animation: pulse 2s ease-in-out infinite; }

7. 복합 애니메이션

복합 애니메이션 (이동 + 회전 + 색상 변경)
@keyframes complex { 0% { transform: translateX(0) rotate(0deg); background-color: #3498db; } 25% { transform: translateX(100px) rotate(90deg); background-color: #27ae60; } 50% { transform: translateX(100px) translateY(100px) rotate(180deg); background-color: #e74c3c; } 75% { transform: translateX(0) translateY(100px) rotate(270deg); background-color: #f39c12; } 100% { transform: translateX(0) translateY(0) rotate(360deg); background-color: #3498db; } } .element { animation: complex 4s ease-in-out infinite; }

8. 애니메이션 속성

애니메이션의 세부 속성을 제어할 수 있습니다.

애니메이션 제어 예제
/* 개별 속성 */ .element { animation-name: fadeIn; animation-duration: 1s; animation-timing-function: ease-in-out; animation-delay: 0.5s; animation-iteration-count: 3; /* 3번 반복 */ animation-iteration-count: infinite; /* 무한 반복 */ animation-direction: alternate; /* 왕복 */ animation-fill-mode: forwards; /* 애니메이션 후 상태 유지 */ animation-play-state: running; /* 재생 중 */ animation-play-state: paused; /* 일시정지 */ } /* 단축 속성 */ .element { animation: fadeIn 1s ease-in-out 0.5s 3 alternate forwards; }

9. 실전 애니메이션 예제

로딩 스피너

@keyframes spin { to { transform: rotate(360deg); } } .spinner { width: 50px; height: 50px; border: 5px solid #f3f3f3; border-top: 5px solid #3498db; border-radius: 50%; animation: spin 1s linear infinite; }

모달 애니메이션

모달이 나타나는 애니메이션
@keyframes fadeIn { from { opacity: 0; transform: scale(0.9); } to { opacity: 1; transform: scale(1); } } .modal { animation: fadeIn 0.3s ease-out; }
