CSS 변수 예제

1. CSS 변수 정의

CSS 변수는 :root에서 정의하여 전역으로 사용할 수 있습니다.

:root { /* 색상 변수 */ --primary-color: #3498db; --secondary-color: #2ecc71; /* 폰트 변수 */ --font-size-base: 16px; --font-size-h1: 2.5em; /* 간격 변수 */ --spacing-unit: 8px; --spacing-small: calc(var(--spacing-unit) * 1); --spacing-medium: calc(var(--spacing-unit) * 2); --spacing-large: calc(var(--spacing-unit) * 4); }
특징:
  • --로 시작하는 이름
  • :root에서 정의하면 전역 변수
  • 대소문자 구분

2. CSS 변수 사용

var() 함수로 변수를 사용합니다.

Primary Color

색상: var(--primary-color)

Secondary Color

색상: var(--secondary-color)

.element { color: var(--primary-color); font-size: var(--font-size-base); padding: var(--spacing-medium); }

3. calc()와 조합

calc() 함수와 함께 사용하여 계산된 값을 만들 수 있습니다.

카드 제목

이 카드는 CSS 변수로 스타일이 적용되었습니다.

패딩: var(--spacing-large) = calc(var(--spacing-unit) * 4) = 32px

:root { --spacing-unit: 8px; --spacing-large: calc(var(--spacing-unit) * 4); /* 32px */ } .card { padding: var(--spacing-large); margin: calc(var(--spacing-unit) * 2); /* 16px */ }

4. 기본값 설정

변수가 없을 때 기본값을 설정할 수 있습니다.

.element { color: var(--text-color, #333); /* --text-color가 없으면 #333 사용 */ background-color: var(--bg-color, white); }
사용 시나리오: 변수가 정의되지 않았을 때 대체 값 제공

5. 테마 변경

CSS 변수를 사용하면 테마 변경이 매우 쉽습니다.

테마 데모

버튼을 클릭하여 테마를 변경해보세요.

/* 라이트 테마 (기본) */ :root { --bg-color: #ffffff; --text-color: #333333; } /* 다크 테마 */ [data-theme="dark"] { --bg-color: #1a1a1a; --text-color: #ffffff; } body { background-color: var(--bg-color); color: var(--text-color); }

6. JavaScript로 동적 변경

JavaScript로 CSS 변수를 동적으로 변경할 수 있습니다.

이 박스의 색상을 변경해보세요:

// JavaScript로 변수 변경 document.documentElement.style.setProperty('--primary-color', '#e74c3c'); // 변수 값 읽기 const color = getComputedStyle(document.documentElement) .getPropertyValue('--primary-color');

7. 반응형 변수

미디어 쿼리와 함께 사용하여 반응형 변수를 만들 수 있습니다.

반응형 폰트 크기

화면 크기에 따라 폰트 크기가 변경됩니다.

기본 크기: var(--font-size-base)

:root { --font-size-base: 14px; /* 모바일 */ } @media (min-width: 768px) { :root { --font-size-base: 16px; /* 태블릿 */ } } @media (min-width: 1024px) { :root { --font-size-base: 18px; /* 데스크톱 */ } }

8. 실전 활용 예제

CSS 변수를 활용한 재사용 가능한 컴포넌트입니다.

/* 버튼 컴포넌트 */ .button { background-color: var(--primary-color); color: white; padding: var(--spacing-small) var(--spacing-medium); border-radius: var(--border-radius); box-shadow: var(--shadow-small); } .button-danger { background-color: var(--danger-color); }
장점:
  • 색상 변경이 쉬움 (변수만 수정)
  • 일관된 디자인 시스템
  • 유지보수 용이
