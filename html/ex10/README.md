# 포트폴리오 사이트 예제

이 폴더는 최종 실습을 위한 완성된 포트폴리오 사이트 예제입니다.

## 파일 구조

```
ex10/
├── index.html          # 메인 페이지
├── projects.html       # 프로젝트 페이지
├── contact.html        # 연락처 페이지
├── assets/
│   ├── css/
│   │   └── main.css    # 메인 스타일시트
│   └── js/
│       ├── main.js     # 메인 JavaScript
│       └── form.js     # 폼 유효성 검사
└── README.md           # 이 파일
```

## 주요 기능

### 1. 반응형 디자인
- 모바일 우선 설계
- 태블릿 및 데스크톱 최적화
- 유동형 레이아웃

### 2. 접근성
- 시맨틱 HTML
- 스킵 링크
- 포커스 스타일
- ARIA 속성

### 3. 폼 유효성 검사
- 실시간 검사
- 시각적 피드백
- 에러 메시지 표시
- 성공 메시지

### 4. 인터랙션
- 스무스 스크롤
- 호버 효과
- 애니메이션
- 모바일 메뉴

## 사용 방법

1. 모든 파일을 웹 서버에 업로드
2. 브라우저에서 `index.html` 열기
3. 각 페이지 탐색

## 배포 방법

### GitHub Pages
1. GitHub 저장소 생성
2. 파일 업로드
3. Settings → Pages → Source 선택
4. 배포 완료

### Vercel
1. Vercel 웹사이트 접속
2. GitHub 저장소 연동
3. 자동 배포

## 커스터마이징

### 색상 변경
`assets/css/main.css`의 CSS 변수 수정:
```css
:root {
    --primary-color: #667eea;
    --secondary-color: #764ba2;
}
```

### 콘텐츠 수정
각 HTML 파일의 콘텐츠를 수정하세요.

### 스타일 수정
`assets/css/main.css`를 수정하세요.
