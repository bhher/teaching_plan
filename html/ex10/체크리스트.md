# 유지보수 체크리스트

이 문서는 포트폴리오 사이트의 유지보수를 위한 체크리스트입니다.

## 1. 접근성 체크리스트

### 색상 대비
- [ ] 일반 텍스트 대비: 최소 4.5:1
- [ ] 큰 텍스트 대비: 최소 3:1
- [ ] 링크와 배경 대비: 충분한가?

**도구:**
- WAVE (Web Accessibility Evaluation Tool)
- axe DevTools
- Lighthouse 접근성 점수

### 키보드 접근성
- [ ] Tab 키로 모든 요소 접근 가능한가?
- [ ] 포커스 스타일이 명확한가?
- [ ] 포커스 순서가 논리적인가?
- [ ] 키보드만으로 모든 기능 사용 가능한가?

### 스크린 리더
- [ ] 시맨틱 HTML 사용
- [ ] alt 속성 (이미지)
- [ ] aria-label (아이콘 버튼)
- [ ] 적절한 제목 구조 (h1-h6)
- [ ] 스킵 링크

### 기타
- [ ] 폼 레이블이 명확한가?
- [ ] 에러 메시지가 명확한가?
- [ ] 색상만으로 정보를 전달하지 않는가?
