CSS 선택자 예제

1. 태그 선택자 (Type Selector)

HTML 태그 이름으로 선택합니다.

이 h3는 태그 선택자로 스타일이 적용되었습니다

이 p 태그도 태그 선택자로 스타일이 적용됩니다.

h3 { color: #e74c3c; }
특이성: 1점

2. 클래스 선택자 (Class Selector)

클래스 이름으로 선택합니다. 점(.)으로 시작합니다.

이 문단은 highlight 클래스가 적용되었습니다.

이 문단은 important 클래스가 적용되었습니다.

이 문단은 두 클래스가 모두 적용되었습니다.

.highlight { background-color: #fff3cd; padding: 10px; } .important { font-weight: bold; color: #d32f2f; } /* 여러 클래스 동시 선택 */ .highlight.important { background-color: #ffebee; }
특이성: 10점 (클래스당 10점)

3. ID 선택자 (ID Selector)

ID 이름으로 선택합니다. 샵(#)으로 시작합니다.

이 문단은 ID 선택자로 스타일이 적용되었습니다.

#main-title { color: #1976d2; font-size: 2em; }
주의: 하나의 HTML 문서에서 같은 ID는 한 번만 사용해야 합니다. 가능하면 클래스 선택자를 사용하는 것이 좋습니다.
특이성: 100점

4. 후손 선택자 (Descendant Selector)

특정 요소의 자손(하위) 요소를 선택합니다. 공백으로 구분합니다.

이것은 직접 자식입니다. (자식 선택자와 후손 선택자 모두 적용)

이것은 손자입니다. (후손 선택자만 적용)

/* 후손 선택자 - 모든 하위 p 태그 */ .container p { color: #388e3c; margin-left: 20px; } /* 자식 선택자 - 직접 자식만 */ .container > p { color: #1976d2; font-weight: bold; }
차이점: 후손 선택자는 모든 하위 요소를 선택하고, 자식 선택자는 직접 자식만 선택합니다.

5. 자식 선택자 (Child Selector)

특정 요소의 직접 자식만 선택합니다. `>` 기호를 사용합니다.

직접 자식 - 파란색, 굵게

손자 - 초록색, 일반

.container > p { color: #1976d2; font-weight: bold; }

6. 속성 선택자 (Attribute Selector)

요소의 속성으로 선택합니다.



https로 시작하는 링크
http로 시작하는 링크
.com으로 끝나는 링크
/* 속성이 있는 요소 */ [type="text"] { border: 2px solid #3498db; } /* 속성 값이 시작 */ [href^="https"] { color: #4caf50; font-weight: bold; } /* 속성 값이 끝남 */ [href$=".com"] { text-decoration: underline; } /* 속성 값에 포함 */ [class*="demo"] { background-color: #e3f2fd; }
속성 선택자 종류:
  • [attr]: 속성이 있는 요소
  • [attr="value"]: 속성 값이 정확히 일치
  • [attr*="value"]: 속성 값에 포함
  • [attr^="value"]: 속성 값이 시작
  • [attr$="value"]: 속성 값이 끝남

7. 선택자 조합

여러 선택자를 조합하여 사용할 수 있습니다.

태그 + 클래스 조합

태그 + 클래스 + 후손 선택자 조합

/* 태그 + 클래스 */ p.highlight { font-size: 1.1em; } /* 복합 선택자 */ div.container > p.highlight { background-color: #c8e6c9; } /* 여러 요소 선택 */ h1, h2, h3 { font-family: 'Segoe UI', sans-serif; }

8. 선택자 우선순위 요약

선택자 예시 특이성
태그 선택자 p 1점
클래스 선택자 .highlight 10점
ID 선택자 #main-title 100점
속성 선택자 [type="text"] 10점
후손 선택자 .container p 11점 (10+1)
자식 선택자 .container > p 11점 (10+1)
