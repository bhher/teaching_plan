CSS 적용 방식 예제

1. 인라인 스타일

이것은 인라인 스타일입니다. style 속성에 직접 CSS를 작성합니다.

인라인 스타일은 가장 높은 우선순위를 가집니다.

주의: 인라인 스타일은 유지보수가 어려우므로 가능하면 피하는 것이 좋습니다.

2. 내부 스타일시트

이것은 내부 스타일시트로 스타일이 적용되었습니다. <head> 안의 <style> 태그에 CSS를 작성합니다.

코드 예시:

<head>
    <style>
        .internal-style {
            background-color: #e3f2fd;
            padding: 15px;
        }
    </style>
</head>

3. 외부 스타일시트

이것은 외부 스타일시트로 스타일이 적용되었습니다. 별도의 CSS 파일(external.css)을 만들고 <link> 태그로 연결합니다.

코드 예시:

<head>
    <link rel="stylesheet" href="external.css">
</head>
권장: 외부 스타일시트는 가장 권장되는 방법입니다. HTML과 CSS를 분리하여 유지보수가 쉽고, 여러 페이지에서 재사용할 수 있습니다.

4. 우선순위 (Specificity) 예제

우선순위 테스트

이 문단의 색상은?

적용된 스타일:

  • 태그 선택자 (p): color: blue (특이성: 1점)
  • 클래스 선택자 (.test-class): color: green (특이성: 10점)
  • ID 선택자 (#test): color: red (특이성: 100점)

결과: ID 선택자가 가장 높은 특이성을 가지므로 빨간색이 적용됩니다.

5. 여러 스타일시트 연결

여러 개의 외부 스타일시트를 연결할 수 있습니다.

<head>
    <link rel="stylesheet" href="reset.css">
    <link rel="stylesheet" href="layout.css">
    <link rel="stylesheet" href="theme.css">
</head>
순서: 나중에 연결된 스타일시트가 우선순위가 높습니다. 일반적으로 reset.css → layout.css → theme.css 순서로 연결합니다.

6. !important 사용 (권장하지 않음)

이 문단은 !important가 적용되었습니다.

주의: !important는 모든 우선순위를 무시하므로 가능하면 사용하지 않는 것이 좋습니다. 사용하면 나중에 스타일 수정이 어려워집니다.

7. 우선순위 요약

우선순위 선택자 특이성 점수
1위 인라인 스타일 1000점
2위 ID 선택자 (#id) 100점
3위 클래스 선택자 (.class) 10점
4위 태그 선택자 (tag) 1점
