텍스트 스타일링 예제

1. font-family (글꼴)

글꼴을 지정합니다. 여러 글꼴을 나열하면 앞에서부터 사용 가능한 글꼴을 사용합니다.

Serif 폰트 (Times New Roman)

Sans-serif 폰트 (Arial)

Monospace 폰트 (Courier New)

한글 폰트 (Noto Sans KR)

font-family: 'Noto Sans KR', Arial, sans-serif;
팁: 마지막에 일반 폰트 계열(serif, sans-serif, monospace)을 지정하는 것이 좋습니다.

2. font-size (글자 크기)

글자 크기를 지정합니다.

16px - 픽셀 단위 (고정 크기)

1.5em - 부모 요소의 1.5배

1.5rem - 루트 요소의 1.5배 (권장)

150% - 부모 요소의 150%

font-size: 16px; font-size: 1.5em; font-size: 1.5rem; /* 권장 */ font-size: 150%;
권장: rem 단위를 사용하면 일관된 크기 조절이 가능합니다.

3. font-weight (글자 굵기)

글자 굵기를 지정합니다.

normal (400) - 기본 굵기

300 - 가늘게

bold (700) - 굵게

600 - 중간 굵게

900 - 매우 굵게

font-weight: normal; /* 400 */ font-weight: bold; /* 700 */ font-weight: 300; font-weight: 600;

4. font-style (글자 스타일)

글자 스타일을 지정합니다.

normal - 기본 스타일

italic - 기울임

oblique - 기울임 (약간 다름)

font-style: normal; font-style: italic; font-style: oblique;

5. line-height (줄 간격)

줄 간격을 지정합니다. 가독성에 매우 중요합니다.

line-height: 1.2 - 좁은 간격. 여러 줄의 텍스트가 있을 때 줄 간격이 좁아서 읽기 어려울 수 있습니다.

line-height: 1.5 - 일반적인 간격 (권장). 본문에 적합한 줄 간격으로 읽기 편합니다.

line-height: 2 - 넓은 간격. 줄 간격이 넓어서 여유롭게 느껴집니다.

line-height: 24px - 고정 값. 픽셀 단위로 고정된 줄 간격입니다.

line-height: 1.5; /* 권장 - 글자 크기의 1.5배 */ line-height: 24px; /* 고정 값 */ line-height: 150%; /* 백분율 */
권장 값:
  • 본문: 1.5 ~ 1.8
  • 제목: 1.2 ~ 1.4

6. letter-spacing (자간)

글자 사이 간격을 지정합니다.

normal - 기본 자간

wide - 넓은 자간 (2px)

tight - 좁은 자간 (-1px)

em - 상대적 자간 (0.1em)

letter-spacing: normal; letter-spacing: 2px; letter-spacing: -1px; letter-spacing: 0.1em;
사용 예: 제목이나 로고에 넓은 자간을 주면 고급스러운 느낌을 줄 수 있습니다.

7. text-align (텍스트 정렬)

텍스트 정렬을 지정합니다.

왼쪽 정렬 (left) - 기본값입니다. 대부분의 텍스트는 왼쪽 정렬을 사용합니다.

가운데 정렬 (center) - 제목이나 특별한 강조에 사용합니다.

오른쪽 정렬 (right) - 날짜나 숫자에 사용할 수 있습니다.

양쪽 정렬 (justify) - 양쪽 끝을 맞춰서 정렬합니다. 신문이나 잡지에서 자주 사용합니다.

text-align: left; text-align: center; text-align: right; text-align: justify;

8. text-decoration (텍스트 장식)

텍스트 장식을 지정합니다.

none - 장식 없음

underline - 밑줄

line-through - 취소선

overline - 윗줄

text-decoration: none; text-decoration: underline; text-decoration: line-through; text-decoration: overline;
팁: 링크의 기본 밑줄을 제거할 때 text-decoration: none;을 사용합니다.

9. text-transform (대소문자 변환)

텍스트 대소문자를 변환합니다.

none - 원본 그대로 (Hello World)

uppercase - 대문자 (HELLO WORLD)

lowercase - 소문자 (hello world)

capitalize - 단어 첫 글자 대문자 (Hello World)

text-transform: none; text-transform: uppercase; text-transform: lowercase; text-transform: capitalize;

10. font 단축 속성

여러 폰트 속성을 한 번에 지정할 수 있습니다.

font 단축 속성: italic bold 18px/1.6 'Noto Sans KR', sans-serif

font: italic bold 18px/1.6 'Noto Sans KR', sans-serif; /* font: style weight size/line-height family */
순서: style → weight → size/line-height → family
