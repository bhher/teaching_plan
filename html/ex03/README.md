# CSS 기초와 적용방법 예제

이 폴더는 CSS 기초와 적용방법을 학습하기 위한 예제 파일들을 포함합니다.

## 예제 파일 목록

### 1. `01-css-application-methods.html`
- CSS 적용 방식 (인라인, 내부, 외부 스타일시트)
- CSS 우선순위 (Specificity) 개념
- 여러 스타일시트 연결
- !important 사용법 (권장하지 않음)
- 우선순위 계산 및 예제

### 2. `external.css`
- 외부 스타일시트 예제 파일
- 01-css-application-methods.html에서 사용

### 3. `02-selectors.html`
- 태그 선택자 (Type Selector)
- 클래스 선택자 (Class Selector)
- ID 선택자 (ID Selector)
- 후손 선택자 (Descendant Selector)
- 자식 선택자 (Child Selector)
- 속성 선택자 (Attribute Selector)
- 선택자 조합
- 선택자 우선순위 요약

### 4. `03-text-styling.html`
- font-family (글꼴)
- font-size (글자 크기) - px, em, rem, %
- font-weight (글자 굵기)
- font-style (글자 스타일)
- line-height (줄 간격)
- letter-spacing (자간)
- text-align (텍스트 정렬)
- text-decoration (텍스트 장식)
- text-transform (대소문자 변환)
- font 단축 속성

### 5. `04-colors-backgrounds-borders.html`
- 색상 지정 방법 (키워드, 16진수, RGB, RGBA, HSL, HSLA)
- 색상 속성 (color, background-color, border-color)
- 배경 색상 (background-color)
- 배경 이미지 (background-image)
- 배경 반복 (background-repeat)
- 배경 크기 (background-size: cover, contain)
- 배경 단축 속성
- 테두리 기본 (border)
- 테두리 스타일 (border-style)
- 개별 테두리 (border-top, right, bottom, left)
- border-radius (둥근 모서리)

### 6. `05-complete-example.html`
- 모든 내용을 종합한 완성 예제
- 프로필 카드 형태의 웹페이지
- 외부 폰트, 그라데이션, 그림자, 호버 효과 등 실전 기법 활용

## 사용 방법

1. 각 HTML 파일을 웹 브라우저에서 열어보세요
2. 브라우저의 개발자 도구(F12)를 사용하여 소스 코드를 확인하세요
3. 각 예제의 CSS 코드를 수정해보며 학습하세요
4. 외부 스타일시트 예제는 `external.css` 파일도 함께 확인하세요

## 학습 체크리스트

각 예제를 학습한 후 다음 사항을 확인하세요:

### CSS 적용 방식
- [ ] 인라인, 내부, 외부 스타일시트의 차이를 이해했는가?
- [ ] 외부 스타일시트를 사용할 수 있는가?
- [ ] CSS 우선순위(specificity)를 계산할 수 있는가?
- [ ] 여러 스타일시트를 연결할 수 있는가?

### 선택자
- [ ] 태그, 클래스, ID 선택자를 사용할 수 있는가?
- [ ] 후손 선택자와 자식 선택자의 차이를 이해했는가?
- [ ] 속성 선택자를 사용할 수 있는가?
- [ ] 선택자를 조합할 수 있는가?

### 텍스트 스타일링
- [ ] font-family, font-size, font-weight를 사용할 수 있는가?
- [ ] line-height로 가독성을 향상시킬 수 있는가?
- [ ] letter-spacing을 조절할 수 있는가?
- [ ] text-align, text-decoration을 사용할 수 있는가?
- [ ] font 단축 속성을 사용할 수 있는가?

### 색상/배경/테두리
- [ ] 다양한 색상 지정 방법을 사용할 수 있는가?
- [ ] 배경 색상과 이미지를 설정할 수 있는가?
- [ ] background-size (cover, contain)를 이해했는가?
- [ ] 테두리를 만들고 스타일을 지정할 수 있는가?
- [ ] border-radius로 모서리를 둥글게 만들 수 있는가?
