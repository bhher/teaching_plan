색상, 배경, 테두리 예제

1. 색상 지정 방법

CSS에서 색상을 지정하는 여러 방법이 있습니다.

키워드: color: red;

16진수: color: #ff0000;

RGB: color: rgb(255, 0, 0);

RGBA (반투명): color: rgba(255, 0, 0, 0.5);

HSL: color: hsl(0, 100%, 50%);

HSLA (반투명): color: hsla(0, 100%, 50%, 0.5);

/* 키워드 */ color: red; /* 16진수 */ color: #ff0000; color: #333; /* #333333의 축약형 */ /* RGB */ color: rgb(255, 0, 0); color: rgba(255, 0, 0, 0.5); /* 반투명 */ /* HSL */ color: hsl(0, 100%, 50%); color: hsla(0, 100%, 50%, 0.5); /* 반투명 */
HSL 설명:
  • H (Hue): 색상 (0~360)
  • S (Saturation): 채도 (0~100%)
  • L (Lightness): 명도 (0~100%)

2. 색상 속성

색상을 적용할 수 있는 다양한 속성이 있습니다.

color: 글자 색상

background-color: 배경 색상
border-color: 테두리 색상
color: #333; /* 글자 색상 */ background-color: #e3f2fd; /* 배경 색상 */ border-color: #3498db; /* 테두리 색상 */

3. 배경 색상 (background-color)

요소의 배경 색상을 지정합니다.

background-color: #e3f2fd;
background-color: #e3f2fd; background-color: rgba(0, 0, 0, 0.8); /* 반투명 검정 */

4. 배경 이미지 (background-image)

요소의 배경 이미지를 지정합니다.

background-image 사용 예제
background-image: url('image.jpg');

5. 배경 반복 (background-repeat)

배경 이미지의 반복을 지정합니다.

background-repeat: repeat (기본값)
background-repeat: no-repeat
background-repeat: repeat; /* 반복 (기본값) */ background-repeat: no-repeat; /* 반복 안 함 */ background-repeat: repeat-x; /* 가로만 반복 */ background-repeat: repeat-y; /* 세로만 반복 */

6. 배경 크기 (background-size)

배경 이미지의 크기를 지정합니다.

background-size: cover (영역을 가득 채움)
background-size: contain (이미지 전체가 보이도록)
background-size: cover; /* 영역을 가득 채움 */ background-size: contain; /* 이미지 전체가 보이도록 */ background-size: 100% 100%; /* 가로 세로 100% */ background-size: 200px 150px; /* 고정 크기 */

7. 배경 단축 속성 (background)

여러 배경 속성을 한 번에 지정할 수 있습니다.

background: #f0f0f0 url('image.jpg') no-repeat center/cover; /* background: color image repeat position/size */

8. 테두리 기본 (border)

테두리를 만드는 기본 속성들입니다.

border: 2px solid #3498db;
border: 2px dashed #e74c3c;
border: 2px dotted #27ae60;
border: 4px double #9b59b6;
/* border 단축 속성 */ border: 2px solid #3498db; /* border: width style color */ /* 개별 속성 */ border-width: 2px; border-style: solid; border-color: #3498db;

9. 테두리 스타일 (border-style)

테두리의 스타일을 지정합니다.

solid - 실선
dashed - 점선
dotted - 점
double - 이중선
groove - 홈
ridge - 능선
border-style: solid; /* 실선 */ border-style: dashed; /* 점선 */ border-style: dotted; /* 점 */ border-style: double; /* 이중선 */ border-style: groove; /* 홈 */ border-style: ridge; /* 능선 */ border-style: inset; /* 안으로 */ border-style: outset; /* 밖으로 */ border-style: none; /* 없음 */

10. 개별 테두리

각 방향의 테두리를 개별적으로 지정할 수 있습니다.

border-top: 3px solid red;
border-right: 2px dashed blue;
border-bottom: 4px dotted green;
border-left: 2px solid purple;
border-top: 2px solid red; border-right: 1px dashed blue; border-bottom: 3px dotted green; border-left: 1px solid #333;

11. border-radius (둥근 모서리)

테두리 모서리를 둥글게 만듭니다.

border-radius: 10px (모든 모서리)
원형
border-radius: 20px 5px 20px 5px (각 모서리 개별 지정)
border-radius: 5px; /* 모든 모서리 */ border-radius: 10px 20px; /* 왼쪽위/오른쪽아래 왼쪽아래/오른쪽위 */ border-radius: 5px 10px 15px 20px; /* 왼쪽위 오른쪽위 오른쪽아래 왼쪽아래 */ border-radius: 50%; /* 원형 */
팁: border-radius: 50%를 사용하면 원형을 만들 수 있습니다.
