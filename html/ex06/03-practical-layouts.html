Grid 실전 레이아웃 예제

1. 복잡한 페이지 레이아웃

grid-template-areas를 사용한 완전한 페이지 레이아웃입니다.

Header
Sidebar
Main Content
Aside
Footer
/* 복잡한 페이지 레이아웃 */ .page-layout { display: grid; grid-template-columns: 200px 1fr 200px; grid-template-rows: 80px 1fr 100px; grid-template-areas: "header header header" "sidebar main aside" "footer footer footer"; gap: 20px; min-height: 600px; } header { grid-area: header; } .sidebar { grid-area: sidebar; } main { grid-area: main; } .aside { grid-area: aside; } footer { grid-area: footer; }
장점: grid-template-areas를 사용하면 레이아웃 구조를 시각적으로 파악하기 쉽고, 반응형 디자인에서 영역 순서를 쉽게 변경할 수 있습니다.

2. 반응형 그리드 (auto-fit)

화면 크기에 따라 자동으로 열 개수가 조절됩니다.

/* 반응형 그리드 */ .responsive-grid { display: grid; grid-template-columns: repeat(auto-fit, minmax(250px, 1fr)); gap: 20px; }
설명:
  • auto-fit: 가능한 많은 열 생성, 빈 트랙 축소
  • minmax(250px, 1fr): 최소 250px, 최대 1fr
  • 화면이 넓어지면 열이 늘어나고, 좁아지면 줄어듦

3. 카드 그리드 (auto-fill)

카드 형태의 반응형 그리드입니다.

카드 1

이것은 Grid를 사용한 카드 그리드입니다. auto-fill을 사용하여 화면 크기에 따라 자동으로 열 개수가 조절됩니다.

카드 2

각 카드는 최소 300px 크기를 유지하면서 남은 공간을 균등하게 분배합니다.

카드 3

gap 속성을 사용하여 카드 사이의 간격을 쉽게 조절할 수 있습니다.

카드 4

반응형 디자인에 완벽하게 적합한 레이아웃입니다.

/* 카드 그리드 */ .card-grid { display: grid; grid-template-columns: repeat(auto-fill, minmax(300px, 1fr)); gap: 25px; }
auto-fit vs auto-fill:
  • auto-fit: 빈 트랙을 축소 (콘텐츠에 맞춤)
  • auto-fill: 빈 트랙 유지 (고정 그리드)

4. 잡지형 레이아웃 (Magazine Layout)

다양한 크기의 아이템을 배치한 잡지형 레이아웃입니다.

큰 아이템 1 (2x2)
아이템 2 (2x1)
아이템 3 (1x2)
아이템 4 (1x1)
아이템 5 (1x2)
아이템 6 (2x1)
아이템 7 (3x1)
/* 잡지형 레이아웃 */ .magazine-layout { display: grid; grid-template-columns: repeat(4, 1fr); grid-template-rows: repeat(4, 150px); gap: 15px; } .item-1 { grid-column: 1 / 3; grid-row: 1 / 3; /* 2x2 */ } .item-2 { grid-column: 3 / 5; grid-row: 1 / 2; /* 2x1 */ } .item-3 { grid-column: 3 / 4; grid-row: 2 / 4; /* 1x2 */ } /* ... */
사용 예시: 포트폴리오 갤러리, 블로그 메인 페이지, 뉴스 사이트 등 다양한 크기의 콘텐츠를 배치할 때 유용합니다.
