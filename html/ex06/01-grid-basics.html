Grid 기본 예제

1. display: grid

Grid를 활성화합니다. 기본적으로 한 열로 배치됩니다.

Item 1
Item 2
Item 3
.container { display: grid; /* Grid 활성화 */ }
특징: display: grid만으로는 한 열로 배치됩니다. grid-template-columns로 열을 정의해야 합니다.

2. grid-template-columns (열 정의)

열(Column)의 크기와 개수를 정의합니다.

고정 크기 (200px 200px 200px)

Item 1
Item 2
Item 3
grid-template-columns: 200px 200px 200px;

비율 분배 (1fr 2fr 1fr)

Item 1 (1fr)
Item 2 (2fr)
Item 3 (1fr)
grid-template-columns: 1fr 2fr 1fr; /* 비율: 1:2:1 (25%:50%:25%) */

repeat() 함수 (repeat(3, 1fr))

Item 1
Item 2
Item 3
grid-template-columns: repeat(3, 1fr); /* 3개 열, 균등 분배 */

고정 + 유연 (200px 1fr 200px)

Item 1 (200px)
Item 2 (1fr - 유연)
Item 3 (200px)
grid-template-columns: 200px 1fr 200px; /* 고정 크기 + 유연한 크기 */

minmax() 함수 (minmax(150px, 1fr))

Item 1
Item 2
Item 3
grid-template-columns: repeat(3, minmax(150px, 1fr)); /* 최소 150px, 최대 1fr */
단위 설명:
  • px: 고정 크기
  • fr (fraction): 남은 공간의 비율
  • %: 부모 요소 기준 백분율
  • auto: 콘텐츠 크기
  • minmax(min, max): 최소/최대 크기

3. grid-template-rows (행 정의)

행(Row)의 크기와 개수를 정의합니다.

고정 크기 (100px 150px 100px)

Item 1 (100px)
Item 2 (150px)

grid-template-rows: 100px 150px 100px;

비율 분배 (1fr 2fr 1fr)

Item 1 (1fr)
Item 2 (2fr)
Item 3 (1fr)
grid-template-rows: 1fr 2fr 1fr; height: 300px; /* 높이 지정 필요 */

자동 높이 (auto)

Item 1
콘텐츠에 따라
높이가 결정됩니다
Item 2
Item 3
grid-template-rows: auto; /* 콘텐츠에 따라 자동으로 높이 결정 */
주의: 명시적으로 정의하지 않은 행은 auto로 생성되며, 콘텐츠에 따라 자동으로 높이가 결정됩니다.

4. Grid 구조 이해

주요 개념

  • Grid Container (부모): display: grid를 가진 요소
  • Grid Item (자식): Container의 직접 자식 요소
  • Grid Line (그리드 선): 그리드를 나누는 선
  • Grid Track (트랙): 그리드 선 사이의 공간 (행 또는 열)
  • Grid Cell (셀): 행과 열이 교차하는 영역
  • Grid Area (영역): 하나 이상의 셀로 구성된 영역

시각화

┌─────────────────────────────────┐
│      Grid Container (부모)      │
│  ┌──────┬──────┬──────┐       │
│  │Cell 1│Cell 2│Cell 3│       │
│  ├──────┼──────┼──────┤       │
│  │Cell 4│Cell 5│Cell 6│       │
│  ├──────┼──────┼──────┤       │
│  │Cell 7│Cell 8│Cell 9│       │
│  └──────┴──────┴──────┘       │
│  Column (열)                    │
│  ↑ Row (행)                     │
└─────────────────────────────────┘
