## HTML 1차시 – HTML 입문 & 기본 구조

### 0. 강의 목표

- **HTML이 무엇인지** 설명할 수 있다.
- **HTML 문서의 기본 뼈대 구조**를 이해한다.
- `<!DOCTYPE html>`, `<html>`, `<head>`, `<body>`의 역할을 말할 수 있다.
- HTML **주석**을 작성할 수 있다.
- 직접 `index.html` 파일을 만들고 **브라우저에서 열어볼 수 있다.**

---

### 1. HTML이란?

#### 1) HTML의 정의

- **HTML (HyperText Markup Language)**  
  웹 페이지를 만드는 **뼈대 언어**  
  (컴퓨터에게 “이건 제목이야”, “이건 글이야”, “이건 이미지야”라고 알려주는 언어)

- HTML은 “프로그래밍 언어”라기보다,  
  **문서의 구조를 표시하는 언어(마크업 언어)** 이다.

#### 2) 사람 신체에 비유하면?

- 웹 페이지 = **사람 한 명**
- HTML = **뼈대, 골격**
- CSS = **옷, 헤어스타일, 메이크업 (겉모습, 디자인)**
- JavaScript = **근육, 신경 (움직임, 동작)**

> 강의 멘트 예시  
> “HTML만 있으면 마네킹처럼 서 있는 사람입니다.  
> CSS를 입혀야 예쁘고, JavaScript를 붙여야 움직이고 반응하는 사람이 되는 거죠.”

---

### 2. HTML 문서 기본 구조

먼저 전체 구조를 한 번 보고, 각 부분을 사람 몸에 비유해서 설명합니다.

```html
<!DOCTYPE html>
<html lang="ko">
  <head>
    <meta charset="UTF-8" />
    <title>내 첫 번째 페이지</title>
  </head>
  <body>
    <h1>안녕하세요!</h1>
    <p>HTML 공부를 시작합니다.</p>
  </body>
</html>
```

#### 신체 비유로 전체 이해하기

- `<!DOCTYPE html>` : **“나는 사람이다” 라고 적어 둔 메모**  
  → 이 문서는 **HTML5 문서야!** 라고 브라우저에게 알려주는 선언

- `<html> ... </html>` : **사람 전체 몸을 감싸는 피부**  
  → 이 안에 있는 모든 것이 “웹 페이지”의 구성 요소

- `<head> ... </head>` : **머리 속 (뇌) / 생각, 정보**  
  → 화면에 직접 보이지 않는 정보들 (제목, 설명, 설정 등)

- `<body> ... </body>` : **눈에 보이는 몸**  
  → 실제로 화면에 보여지는 내용(글, 이미지, 버튼 등)

---

### 3. `<!DOCTYPE html>` 의 의미

#### 1) 왜 필요한가?

- 브라우저에게 **“이 문서는 HTML5 규칙을 따릅니다”** 라고 알려주는 문장
- 예전에는 `<!DOCTYPE ...>`이 매우 길었지만,  
  HTML5부터는 **간단히 이렇게만 쓰면 됨**:

```html
<!DOCTYPE html>
```

#### 2) 신체 비유

- 의사에게 **“저는 2025년에 살고 있는 현대인입니다”** 라고 말해주는 느낌
  - 예전 사람과는 몸 검사 기준이 다를 수 있음
  - 마찬가지로, `<!DOCTYPE html>`이 없으면 브라우저가 **옛날 규칙으로 페이지를 해석**할 수 있음

> 실무 팁:  
> HTML 파일을 만들 때 **항상 첫 줄에 그대로 복붙**한다고 생각하면 편합니다.

---

### 4. `<html>`, `<head>`, `<body>` 역할 (신체 비유)

#### 4.1 `<html>` – 몸 전체를 감싸는 큰 껍데기

```html
<!DOCTYPE html>
<html lang="ko">
  ...
</html>
```

- **웹 문서 전체**를 감싸는 최상위 태그
- `lang="ko"`: 이 문서의 기본 언어가 **한국어**라는 의미
  - 한국어: `ko`, 영어: `en` 등

> 비유:  
> 사람 몸 전체를 감싸고 있는 큰 **피부** 같은 역할  
> 피부 안에 머리(head), 몸(body)이 모두 들어있다.

---

#### 4.2 `<head>` – 머리 속(뇌), 보이지 않는 정보

```html
<head>
  <meta charset="UTF-8" />
  <title>내 첫 HTML 페이지</title>
</head>
```

- **화면에 직접 보이지 않는 정보**들을 넣는 곳
  - `<meta>`: 문자 인코딩, 검색 엔진용 설명 등
  - `<title>`: 브라우저 탭에 표시되는 제목
  - CSS, JS 파일 연결도 여기에서 진행하는 것이 일반적

> 비유:  
> 사람의 **생각, 기억, 지식**이 모여 있는 곳이 “머리 속”인 것처럼,  
> `<head>`에는 화면에는 안 보이지만 브라우저와 검색엔진이 알아야 할 정보들이 들어 있다.

---

#### 4.3 `<body>` – 눈에 보이는 몸, 실제 화면 내용

```html
<body>
  <h1>안녕하세요!</h1>
  <p>HTML 공부를 시작합니다.</p>
</body>
```

- 실제로 **사용자가 보는 모든 내용**은 `<body>` 안에 들어간다.
  - 제목: `<h1>`, `<h2>` …
  - 문단: `<p>`
  - 이미지: `<img>`
  - 버튼: `<button>`
  - 리스트: `<ul>`, `<ol>` 등

> 비유:  
> 사람의 **겉모습 전체**(얼굴, 팔, 다리 등)가 `<body>`  
> 우리가 눈으로 보는 것은 모두 body 안에 있는 것들이다.

### 5. 주석 사용법

#### 5.1 주석이란?

- **브라우저는 무시**하지만, 사람(개발자)이 보기 위한 메모
- 코드를 설명하거나, 잠시 끄고 싶은 부분이 있을 때 사용

#### 5.2 HTML 주석 문법

```html
<!-- 이것은 주석입니다. 화면에 보이지 않아요 -->

<!-- 아래는 제목 부분입니다 -->
<h1>안녕하세요</h1>
```

- `<!--` 로 시작해서 `-->` 로 끝난다.

> 추천 습관:  
> `<head>`, `<body>` 안에 영역을 나눌 때 주석으로  
> `<!-- 헤더 영역 -->`, `<!-- 메인 콘텐츠 영역 -->` 같이 표시하면  
> 나중에 코드를 읽기 훨씬 쉽다.

### 6. 실습 1 – `index.html` 만들기

#### 6.1 목표

- 직접 `index.html` 파일을 만들고,
- **브라우저에서 열어서 확인**해 본다.

#### 6.2 단계별 안내

1. **폴더 만들기**
   - 예: `html-practice` 라는 폴더 생성

2. **새 파일 생성**
   - 파일 이름: `index.html`

3. **기본 구조 작성**

```html
<!DOCTYPE html>
<html lang="ko">
  <head>
    <meta charset="UTF-8" />
    <title>내 첫 HTML 페이지</title>
  </head>
  <body>
    <h1>안녕하세요!</h1>
    <p>HTML 1차시 실습 중입니다.</p>
  </body>
</html>
```

4. **파일 저장**

5. **브라우저에서 열기**
   - Windows: 파일 탐색기에서 `index.html` 더블클릭
   - 또는 브라우저에서 `Ctrl + O`로 파일 열기

> 강의 포인트:  
> “우리가 코딩한 결과를 바로 눈으로 확인할 수 있다는 점이 HTML의 큰 장점입니다.  
> 작은 수정 → 저장 → 새로고침 → 결과 확인 흐름에 익숙해져 보세요.”

### 7. 실습 2 – 신체 비유를 사용한 페이지 만들기

#### 7.1 목표

- `<head>`, `<body>` 를 사람의 머리/몸으로 설명하는 텍스트를 HTML로 직접 작성해 본다.

#### 7.2 예시 코드

```html
<!DOCTYPE html>
<html lang="ko">
  <head>
    <meta charset="UTF-8" />
    <title>HTML 신체 비유</title>
  </head>
  <body>
    <!-- 제목 영역 -->
    <h1>HTML을 사람 몸에 비유하면?</h1>

    <!-- 설명 영역 -->
    <p>HTML은 사람의 <strong>뼈대</strong>와 같습니다.</p>
    <p>&lt;head&gt; 태그는 사람의 <strong>머리 속(뇌)</strong>처럼 보이지 않는 정보를 담고,</p>
    <p>&lt;body&gt; 태그는 사람의 <strong>몸</strong>처럼 실제로 보이는 내용을 담습니다.</p>
  </body>
</html>
```
