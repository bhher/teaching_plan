Flex 아이템 속성 예제

1. flex-grow (공간 분배)

남은 공간을 아이템이 얼마나 차지할지 지정합니다.

flex-grow: 0 (기본값: 늘지 않음)

Item 1
Item 2
Item 3

flex-grow: 1 (균등 분배)

Item 1
Item 2
Item 3

flex-grow: 1, 2, 1 (비율 분배)

Item 1 (grow: 1)
Item 2 (grow: 2)
Item 3 (grow: 1)
flex-grow: 0; /* 기본값: 늘지 않음 */ flex-grow: 1; /* 남은 공간을 차지 */ flex-grow: 2; /* 다른 아이템의 2배 */
동작 방식: 숫자가 클수록 더 많은 공간을 차지합니다. 비율로 계산됩니다. (1:2:1 = 25%:50%:25%)

2. flex-shrink (축소 비율)

공간이 부족할 때 아이템이 얼마나 줄어들지 지정합니다.

flex-shrink: 0 (줄어들지 않음)

Item 1 (200px)
Item 2 (200px)
Item 3 (200px)
Item 4 (200px)

flex-shrink: 1 (기본값: 줄어듦)

Item 1 (200px)
Item 2 (200px)
Item 3 (200px)
Item 4 (200px)
flex-shrink: 1; /* 기본값: 줄어듦 */ flex-shrink: 0; /* 줄어들지 않음 (최소 크기 유지) */ flex-shrink: 2; /* 다른 아이템의 2배로 줄어듦 */

3. flex-basis (초기 크기)

아이템의 초기 크기를 지정합니다.

flex-basis: auto (기본값: 콘텐츠 크기)

Item 1
긴 텍스트 Item 2
Item 3

flex-basis: 200px (고정 크기)

Item 1
Item 2
Item 3

flex-basis: 30% (비율)

Item 1
Item 2
Item 3
flex-basis: auto; /* 기본값: 콘텐츠 크기 */ flex-basis: 200px; /* 고정 크기 */ flex-basis: 30%; /* 비율 */ flex-basis: 0; /* 최소 크기 */

4. flex (단축 속성)

flex-grow, flex-shrink, flex-basis를 한 번에 지정합니다.

flex: 1 (균등 분배)

Item 1
Item 2
Item 3

flex: 0 0 200px (고정 크기)

Item 1 (200px)
Item 2 (200px)
Item 3 (200px)

flex: 2, 1, 1 (비율 분배)

Item 1 (flex: 2)
Item 2 (flex: 1)
Item 3 (flex: 1)
flex: 1; /* flex: 1 1 0 (균등 분배) */ flex: 0 0 200px; /* 고정 크기 200px */ flex: 2; /* flex: 2 1 0 (2배) */ flex: 0 1 auto; /* 기본값 */
자주 사용하는 패턴:
- flex: 1: 균등 분배
- flex: 0 0 250px: 고정 크기 (사이드바 등)
- flex: 2, flex: 1: 비율 분배

5. align-self (개별 정렬)

개별 아이템의 교차축 정렬을 지정합니다. 부모의 align-items보다 우선순위가 높습니다.

flex-start
center
flex-end
stretch
align-self: flex-start; /* 시작점 정렬 */ align-self: flex-end; /* 끝점 정렬 */ align-self: center; /* 가운데 정렬 */ align-self: stretch; /* 늘림 */ align-self: baseline; /* 기준선 정렬 */

6. order (순서 변경)

HTML 구조는 변경하지 않고 시각적 순서만 변경합니다.

기본 순서

Item 1 (HTML 순서)
Item 2 (HTML 순서)
Item 3 (HTML 순서)

order로 순서 변경

Item 1 (order: 2)
Item 2 (order: 3)
Item 3 (order: 1)

order: -1 (앞으로 이동)

Item 1
Item 2
Item 3 (order: -1)
order: 0; /* 기본값 */ order: 1; /* 뒤로 이동 */ order: -1; /* 앞으로 이동 */
주의: 숫자가 작을수록 앞에 배치됩니다. HTML 구조는 변경되지 않고 시각적 순서만 변경됩니다.
