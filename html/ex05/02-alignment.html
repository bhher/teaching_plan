4. gap (간격)

아이템 사이의 간격을 지정합니다. margin보다 간편합니다.

gap: 20px (모든 방향)

Item 1
Item 2
Item 3
Item 4

gap: 10px 30px (세로 가로)

Item 1
Item 2
Item 3
Item 4
gap: 20px; /* 모든 방향 간격 */ gap: 10px 30px; /* 세로 가로 */ row-gap: 10px; /* 세로 간격 */ column-gap: 30px; /* 가로 간격 */
특징: 아이템 사이에만 간격이 적용되고, 양끝에는 적용되지 않습니다. margin보다 훨씬 간편합니다.

5. 정렬 속성 조합 예제

완벽한 중앙 정렬

중앙 정렬된 아이템
/* 완벽한 중앙 정렬 */ .container { display: flex; justify-content: center; /* 주축 중앙 */ align-items: center; /* 교차축 중앙 */ min-height: 100vh; /* 전체 화면 높이 */ }
