Flex 컨테이너 속성 예제

1. display: flex

Flexbox를 활성화합니다. 부모 요소에 적용하면 직접 자식 요소들이 Flex Item이 됩니다.

Item 1
Item 2
Item 3
.container { display: flex; /* Flexbox 활성화 */ }
특징: 기본적으로 가로 방향으로 배치되며, 아이템들이 한 줄에 배치됩니다.

2. flex-direction (주축 방향)

주축의 방향을 지정합니다. 아이템이 배치되는 주요 방향을 결정합니다.

row (기본값: 가로, 왼쪽→오른쪽)

Item 1
Item 2
Item 3

row-reverse (가로, 오른쪽→왼쪽)

Item 1
Item 2
Item 3

column (세로, 위→아래)

Item 1
Item 2
Item 3

column-reverse (세로, 아래→위)

Item 1
Item 2
Item 3
flex-direction: row; /* 가로 (왼쪽→오른쪽) */ flex-direction: row-reverse; /* 가로 (오른쪽→왼쪽) */ flex-direction: column; /* 세로 (위→아래) */ flex-direction: column-reverse; /* 세로 (아래→위) */

3. flex-wrap (줄바꿈)

아이템이 넘칠 때 줄바꿈을 허용할지 지정합니다.

nowrap (기본값: 줄바꿈 안 함)

Item 1 (200px)
Item 2 (200px)
Item 3 (200px)
Item 4 (200px)
Item 5 (200px)
주의: nowrap일 때 아이템이 축소될 수 있습니다.

wrap (줄바꿈 허용)

Item 1 (200px)
Item 2 (200px)
Item 3 (200px)
Item 4 (200px)
Item 5 (200px)

wrap-reverse (줄바꿈 허용, 역순)

Item 1 (200px)
Item 2 (200px)
Item 3 (200px)
Item 4 (200px)
Item 5 (200px)
flex-wrap: nowrap; /* 줄바꿈 안 함 (기본값) */ flex-wrap: wrap; /* 줄바꿈 허용 */ flex-wrap: wrap-reverse; /* 줄바꿈 허용 (역순) */

4. flex-flow (단축 속성)

flex-direction과 flex-wrap을 한 번에 지정합니다.

Item 1
Item 2
Item 3
Item 4
flex-flow: row wrap; /* direction wrap */ flex-flow: column nowrap; flex-flow: row-reverse wrap;

5. Flexbox 구조 이해

주요 개념

  • Flex Container (부모): display: flex를 가진 요소
  • Flex Item (자식): Container의 직접 자식 요소
  • Main Axis (주축): 아이템이 배치되는 주요 방향
  • Cross Axis (교차축): 주축에 수직인 방향

시각화

┌─────────────────────────────────┐
│      Flex Container (부모)      │
│  ┌──────┐  ┌──────┐  ┌──────┐  │
│  │Item 1│  │Item 2│  │Item 3│  │
│  └──────┘  └──────┘  └──────┘  │
│  Main Axis (주축) →              │
│  ↑ Cross Axis (교차축)           │
└─────────────────────────────────┘
