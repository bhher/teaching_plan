미디어 쿼리 예제

1. 기본 미디어 쿼리 (모바일 우선)

화면 크기에 따라 스타일이 변경됩니다. 화면 크기를 조절해보세요.

반응형 박스

화면 크기에 따라 색상과 패딩이 변경됩니다.

  • 모바일 (기본): 노란색
  • 작은 태블릿 (480px+): 파란색
  • 태블릿 (768px+): 초록색
  • 데스크톱 (1024px+): 보라색
  • 큰 데스크톱 (1440px+): 빨간색
/* 기본: 모바일 */ .responsive-box { background-color: #fff3cd; padding: 15px; } /* 작은 태블릿 (480px 이상) */ @media (min-width: 480px) { .responsive-box { background-color: #e3f2fd; padding: 20px; } } /* 태블릿 (768px 이상) */ @media (min-width: 768px) { .responsive-box { background-color: #e8f5e9; padding: 25px; } } /* 데스크톱 (1024px 이상) */ @media (min-width: 1024px) { .responsive-box { background-color: #f3e5f5; padding: 30px; } }

2. 브레이크포인트 정의

일반적으로 사용하는 브레이크포인트입니다.

모바일

320px ~ 479px

기본 스타일 (모바일 우선)

작은 태블릿

480px ~ 767px

@media (min-width: 480px)

태블릿

768px ~ 1023px

@media (min-width: 768px)

데스크톱

1024px ~ 1439px

@media (min-width: 1024px)

큰 데스크톱

1440px ~

@media (min-width: 1440px)

/* 일반적인 브레이크포인트 */ /* 모바일: 320px ~ 479px (기본) */ /* 작은 태블릿: 480px ~ 767px */ @media (min-width: 480px) { } /* 태블릿: 768px ~ 1023px */ @media (min-width: 768px) { } /* 데스크톱: 1024px ~ 1439px */ @media (min-width: 1024px) { } /* 큰 데스크톱: 1440px ~ */ @media (min-width: 1440px) { }
팁: 브레이크포인트는 콘텐츠에 따라 조정할 수 있습니다. 고정된 값보다는 콘텐츠가 자연스럽게 깨지는 지점을 찾는 것이 좋습니다.

3. 화면 방향 (Orientation)

세로 모드와 가로 모드를 구분할 수 있습니다.

방향별 스타일

기기를 회전시켜보세요:

  • 세로 모드 (Portrait): 노란색
  • 가로 모드 (Landscape): 초록색
/* 세로 모드 */ @media (orientation: portrait) { .box { background-color: #fff3cd; } } /* 가로 모드 */ @media (orientation: landscape) { .box { background-color: #e8f5e9; } }

4. 복합 조건

여러 조건을 조합하여 사용할 수 있습니다.

복합 조건 예제

태블릿 크기 (768px~1023px)일 때 노란색으로 변경됩니다.

데스크톱 + 가로 모드일 때 초록색으로 변경됩니다.

/* AND 조건: 태블릿 크기 */ @media (min-width: 768px) and (max-width: 1023px) { .box { background-color: #fff3cd; } } /* AND 조건: 데스크톱 + 가로 모드 */ @media (min-width: 1024px) and (orientation: landscape) { .box { background-color: #e8f5e9; } } /* OR 조건 */ @media (min-width: 768px), (orientation: landscape) { .box { /* 768px 이상 또는 가로 모드 */ } } /* NOT 조건 */ @media not (max-width: 767px) { .box { /* 768px 이상 */ } }

5. 미디어 쿼리 적용 방법

방법 1: CSS 파일 내부

/* 기본 스타일 */ .container { width: 100%; } /* 미디어 쿼리 */ @media (min-width: 768px) { .container { width: 750px; margin: 0 auto; } }

방법 2: 별도 CSS 파일

<link rel="stylesheet" href="mobile.css"> <link rel="stylesheet" href="tablet.css" media="(min-width: 768px)"> <link rel="stylesheet" href="desktop.css" media="(min-width: 1024px)">
권장: 방법 1 (CSS 파일 내부)을 사용하는 것이 HTTP 요청을 줄이고 성능에 유리합니다.

6. 실전 패턴

패턴 1: 폰트 크기 조절

body { font-size: 14px; /* 모바일 */ } @media (min-width: 768px) { body { font-size: 16px; /* 태블릿 */ } } @media (min-width: 1024px) { body { font-size: 18px; /* 데스크톱 */ } }

패턴 2: 레이아웃 전환

/* 모바일: 세로 배치 */ .container { display: flex; flex-direction: column; } /* 태블릿 이상: 가로 배치 */ @media (min-width: 768px) { .container { flex-direction: row; } }

패턴 3: 그리드 열 개수 조절

/* 모바일: 1열 */ .grid { display: grid; grid-template-columns: 1fr; } /* 태블릿: 2열 */ @media (min-width: 768px) { .grid { grid-template-columns: repeat(2, 1fr); } } /* 데스크톱: 3열 */ @media (min-width: 1024px) { .grid { grid-template-columns: repeat(3, 1fr); } }
