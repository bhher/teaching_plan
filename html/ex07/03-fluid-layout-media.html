유동형 레이아웃과 반응형 미디어 예제

1. 고정 너비 vs 유동 너비

고정 너비 (문제 있음)

width: 1200px (고정)

작은 화면에서 가로 스크롤이 발생합니다.

.fixed-width { width: 1200px; /* 고정 크기 */ }
문제점:
  • 작은 화면에서 가로 스크롤 발생
  • 큰 화면에서 공간 낭비
  • 반응형이 아님

유동 너비 (권장)

width: 100%; max-width: 1200px;

모든 화면 크기에 적응하며 최대 크기는 1200px입니다.

.fluid-width { width: 100%; /* 부모의 100% */ max-width: 1200px; /* 최대 크기 제한 */ margin: 0 auto; /* 중앙 정렬 */ }
장점:
  • 모든 화면 크기에 적응
  • 가로 스크롤 없음
  • 공간 효율적 사용

2. 반응형 이미지

max-width: 100%

반응형 이미지 예제
(max-width: 100%)
img { max-width: 100%; height: auto; }
효과:
  • 이미지가 컨테이너보다 크지 않음
  • 비율 유지 (height: auto)
  • 가로 스크롤 방지

object-fit

이미지가 컨테이너를 채우는 방식을 지정합니다.

object-fit: cover (영역을 가득 채움)

cover: 영역을 가득 채움

object-fit: contain (전체가 보이도록)

contain: 전체가 보이도록
img { width: 100%; height: 300px; object-fit: cover; /* 영역을 가득 채움 */ object-fit: contain; /* 전체가 보이도록 */ object-fit: fill; /* 늘림 */ }

3. 반응형 비디오

비디오를 반응형으로 만드는 방법입니다.

비디오 영역
(16:9 비율 유지)
/* 반응형 비디오 */ .video-container { position: relative; padding-bottom: 56.25%; /* 16:9 비율 (9/16 = 0.5625) */ height: 0; overflow: hidden; } .video-container iframe, .video-container video { position: absolute; top: 0; left: 0; width: 100%; height: 100%; }
비율 계산:
  • 16:9 → padding-bottom: 56.25% (9/16 = 0.5625)
  • 4:3 → padding-bottom: 75% (3/4 = 0.75)
  • 1:1 → padding-bottom: 100% (1/1 = 1.0)

4. 유동형 레이아웃 (Flexbox)

Flexbox를 사용한 유동형 레이아웃입니다.

사이드바

flex: 0 0 250px

모바일에서는 위로 이동합니다.

메인 콘텐츠

flex: 1

남은 공간을 모두 차지합니다.

/* 유동형 레이아웃 */ .fluid-layout { display: flex; gap: 20px; } .fluid-sidebar { flex: 0 0 250px; /* 고정 크기 */ } .fluid-main { flex: 1; /* 유연한 크기 */ min-width: 0; /* 축소 가능 */ } /* 모바일: 세로 배치 */ @media (max-width: 768px) { .fluid-layout { flex-direction: column; } }

5. 퍼센트 레이아웃

퍼센트를 사용한 유동형 레이아웃입니다.

사이드바

width: 30%

메인 콘텐츠

width: 70%

/* 퍼센트 레이아웃 */ .percent-layout { display: flex; gap: 20px; } .percent-sidebar { width: 30%; } .percent-main { width: 70%; } /* 모바일: 세로 배치 */ @media (max-width: 768px) { .percent-sidebar, .percent-main { width: 100%; } }

6. calc() 사용

calc() 함수로 정확한 크기를 계산할 수 있습니다.

사이드바

250px (고정)

메인

calc(100% - 270px)

전체에서 사이드바(250px) + gap(20px) 제외

.sidebar { width: 250px; } .main { width: calc(100% - 270px); /* 전체 - 사이드바 - gap */ }

7. 반응형 이미지 고급 기법

srcset과 sizes

다양한 해상도에 맞는 이미지를 제공합니다.

<img src="image-small.jpg" srcset="image-small.jpg 480w, image-medium.jpg 768w, image-large.jpg 1024w" sizes="(max-width: 480px) 100vw, (max-width: 768px) 50vw, 33vw" alt="반응형 이미지">

picture 요소

조건에 따라 다른 이미지를 표시합니다.

<picture> <source media="(min-width: 1024px)" srcset="desktop.jpg"> <source media="(min-width: 768px)" srcset="tablet.jpg"> <img src="mobile.jpg" alt="반응형 이미지"> </picture>
사용 시나리오:
  • 고해상도 디스플레이 지원
  • 모바일 데이터 절약
  • 최적의 이미지 제공
