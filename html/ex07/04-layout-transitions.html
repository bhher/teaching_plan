레이아웃 전환 사례 예제

1. 네비게이션 메뉴 전환

모바일: 햄버거 메뉴 → 태블릿 이상: 가로 메뉴

로고
홈 소개 서비스 포트폴리오 연락처
브레이크포인트:
  • 모바일 (< 768px): 햄버거 메뉴, 세로 메뉴
  • 태블릿 이상 (≥ 768px): 가로 메뉴, 햄버거 버튼 숨김
/* 모바일: 햄버거 메뉴 */ .nav { display: block; } .menu-toggle { display: block; } .menu { display: none; } /* 태블릿 이상: 가로 메뉴 */ @media (min-width: 768px) { .nav { display: flex; justify-content: space-between; } .menu-toggle { display: none; } .menu { display: flex; gap: 20px; } }

2. 카드 그리드 전환

화면 크기에 따라 열 개수가 변경됩니다.

카드 1

모바일: 1열, 태블릿: 2열, 데스크톱: 3열, 큰 화면: 4열

카드 2

화면 크기를 조절하여 열 개수 변화를 확인하세요.

카드 3

Grid의 auto-fit과 minmax를 활용했습니다.

카드 4

완벽한 반응형 그리드입니다.

브레이크포인트:
  • 모바일 (< 480px): 1열
  • 작은 태블릿 (480px ~ 767px): 2열
  • 태블릿 (768px ~ 1023px): 2열
  • 데스크톱 (1024px ~ 1439px): 3열
  • 큰 데스크톱 (≥ 1440px): 4열
/* 모바일: 1열 */ .card-grid { display: grid; grid-template-columns: 1fr; } /* 작은 태블릿: 2열 */ @media (min-width: 480px) { .card-grid { grid-template-columns: repeat(2, 1fr); } } /* 데스크톱: 3열 */ @media (min-width: 1024px) { .card-grid { grid-template-columns: repeat(3, 1fr); } } /* 큰 데스크톱: 4열 */ @media (min-width: 1440px) { .card-grid { grid-template-columns: repeat(4, 1fr); } }

3. 사이드바 레이아웃 전환

모바일: 사이드바 하단 → 태블릿 이상: 사이드바 왼쪽

사이드바

모바일에서는 메인 아래에 위치합니다.

태블릿 이상에서는 왼쪽에 위치합니다.

메인 콘텐츠

모바일에서는 위에 위치합니다.

태블릿 이상에서는 오른쪽에 위치합니다.

브레이크포인트:
  • 모바일 (< 768px): 세로 배치, 사이드바 하단
  • 태블릿 이상 (≥ 768px): 가로 배치, 사이드바 왼쪽
/* 모바일: 세로 배치 */ .sidebar-layout { display: flex; flex-direction: column; } .sidebar { order: 2; /* 아래 */ } .main { order: 1; /* 위 */ } /* 태블릿 이상: 가로 배치 */ @media (min-width: 768px) { .sidebar-layout { flex-direction: row; } .sidebar { flex: 0 0 250px; order: 1; /* 왼쪽 */ } .main { flex: 1; order: 2; /* 오른쪽 */ } }

4. 텍스트 크기 전환

화면 크기에 따라 텍스트 크기가 조절됩니다.

반응형 텍스트

이 텍스트는 화면 크기에 따라 크기가 변경됩니다. 모바일에서는 14px, 태블릿에서는 16px, 데스크톱에서는 18px로 표시됩니다.

가독성을 위해 화면 크기에 맞는 텍스트 크기를 사용하는 것이 중요합니다. 작은 화면에서는 작은 텍스트, 큰 화면에서는 큰 텍스트를 사용하면 사용자 경험이 향상됩니다.

/* 모바일 */ .responsive-text { font-size: 14px; } /* 작은 태블릿 */ @media (min-width: 480px) { .responsive-text { font-size: 15px; } } /* 태블릿 */ @media (min-width: 768px) { .responsive-text { font-size: 16px; } } /* 데스크톱 */ @media (min-width: 1024px) { .responsive-text { font-size: 18px; } }

5. 복합 레이아웃 전환

여러 요소가 함께 전환되는 복합 예제입니다.

헤더

모바일: 중앙 정렬, 태블릿 이상: Flexbox 레이아웃

사이드바

모바일: 전체 너비, 태블릿 이상: 고정 너비

메인 콘텐츠

모바일: 전체 너비, 태블릿 이상: 유연한 너비

푸터

모바일: 세로 배치, 태블릿 이상: 가로 배치

/* 모바일: 세로 배치 */ .layout { display: grid; grid-template-columns: 1fr; } .sidebar, .main { width: 100%; } /* 태블릿 이상: 가로 배치 */ @media (min-width: 768px) { .layout { grid-template-columns: 250px 1fr; } .sidebar { width: 250px; } }
