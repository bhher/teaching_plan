# 3장 예제 파일 실행 방법

## 명령줄에서 실행하기

### 1. ArithmeticOperators.java
```bash
javac ArithmeticOperators.java
java ArithmeticOperators
```

### 2. ComparisonOperators.java
```bash
javac ComparisonOperators.java
java ComparisonOperators
```

### 3. LogicalOperators.java
```bash
javac LogicalOperators.java
java LogicalOperators
```

### 4. AssignmentOperators.java
```bash
javac AssignmentOperators.java
java AssignmentOperators
```

### 5. IncrementDecrement.java
```bash
javac IncrementDecrement.java
java IncrementDecrement
```

### 6. OperatorPrecedence.java
```bash
javac OperatorPrecedence.java
java OperatorPrecedence
```

### 7. OperatorComprehensive.java
```bash
javac OperatorComprehensive.java
java OperatorComprehensive
```

## VS Code에서 실행하기

1. 각 `.java` 파일을 열기
2. `main` 메서드 위에 나타나는 "Run" 버튼 클릭
3. 또는 `F5` 키를 눌러 실행

## 주의사항

- 클래스 이름과 파일 이름이 반드시 일치해야 합니다
- Java는 대소문자를 구분합니다
- 각 파일은 하나의 public 클래스를 포함해야 합니다
- 연산자 우선순위를 이해하고 괄호를 적절히 사용하세요
