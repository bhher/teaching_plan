# 4장 예제 파일 실행 방법

## 명령줄에서 실행하기

### 1. IfStatement.java
```bash
javac IfStatement.java
java IfStatement
```

### 2. SwitchStatement.java
```bash
javac SwitchStatement.java
java SwitchStatement
```

### 3. ForLoop.java
```bash
javac ForLoop.java
java ForLoop
```

### 4. WhileLoop.java
```bash
javac WhileLoop.java
java WhileLoop
```

### 5. NestedControl.java
```bash
javac NestedControl.java
java NestedControl
```

### 6. BreakContinue.java
```bash
javac BreakContinue.java
java BreakContinue
```

### 7. ControlComprehensive.java
```bash
javac ControlComprehensive.java
java ControlComprehensive
```

## VS Code에서 실행하기

1. 각 `.java` 파일을 열기
2. `main` 메서드 위에 나타나는 "Run" 버튼 클릭
3. 또는 `F5` 키를 눌러 실행

## 주의사항

- 클래스 이름과 파일 이름이 반드시 일치해야 합니다
- Java는 대소문자를 구분합니다
- 각 파일은 하나의 public 클래스를 포함해야 합니다
- 무한 루프에 빠지지 않도록 주의하세요
- break와 continue의 차이를 이해하세요

## 학습 포인트

- **조건문**: if-else와 switch의 차이와 사용 시기
- **반복문**: for, while, do-while의 차이와 선택 기준
- **중첩 제어문**: 구구단, 별 찍기 등 패턴 만들기
- **break/continue**: 반복문의 흐름 제어
