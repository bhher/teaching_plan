## 문제: Product 클래스 작성 (20점)

### 문제 설명
상품 정보를 저장하는 `Product` 클래스를 작성하세요.

### 요구사항

1. **필드**
   - `name` (String): 상품명
   - `price` (int): 상품 가격
   - `stock` (int): 재고 수량

2. **생성자**
   - 매개변수가 없는 기본 생성자
   - `name`, `price`, `stock`을 받는 매개변수 생성자

3. **메서드**
   - `toString()`: 상품 정보를 다음 형식으로 반환
     ```
     "상품명: [name], 가격: [price]원, 재고: [stock]개"
     ```
   - `getTotalValue()`: 상품의 총 가치를 계산하는 메서드
     - 총 가치 = 가격 × 재고 수량
     - 반환값: int
   - `isAvailable()`: 재고가 있는지 확인하는 메서드
     - 재고가 0보다 크면 true, 0이면 false 반환
     - 반환값: boolean

### 실행 코드 (제공됨)
```java
public class ProductTest {
    public static void main(String[] args) {
        Product p1 = new Product("노트북", 1200000, 5);
        Product p2 = new Product("마우스", 25000, 0);
        
        System.out.println(p1);
        System.out.println("총 가치: " + p1.getTotalValue() + "원");
        System.out.println("판매 가능: " + p1.isAvailable());
        
        System.out.println();
        
        System.out.println(p2);
        System.out.println("총 가치: " + p2.getTotalValue() + "원");
        System.out.println("판매 가능: " + p2.isAvailable());
    }
}
```

### 실행 결과 예시
```

총 가치: 6000000원

총 가치: 0원
판매 가능: false
```

### 채점 기준
- 필드 선언 (name, price, stock): 5점
- 기본 생성자: 2점
- 매개변수 생성자: 3점
- toString() 메서드: 5점
- getTotalValue() 메서드: 3점
- isAvailable() 메서드: 2점

## 정답 예시

```java
public class Product {
    String name;
    int price;
    int stock;
    
    // 기본 생성자
    public Product() {
    }
    
    // 매개변수 생성자
    public Product(String name, int price, int stock) {
        this.name = name;
        this.price = price;
        this.stock = stock;
    }
    
    // toString() 메서드
    @Override
    public String toString() {
        return "상품명: " + name + ", 가격: " + price + "원, 재고: " + stock + "개";
    }
    
    // 총 가치 계산 메서드
    public int getTotalValue() {
        return price * stock;
    }
    
    // 재고 확인 메서드
    public boolean isAvailable() {
        return stock > 0;
    }
}
```

## 학습 포인트

### 1. 클래스 기본 구조
- 필드 선언: 클래스의 속성을 나타내는 변수
- 생성자: 객체를 생성할 때 호출되는 메서드
- 메서드: 클래스의 기능을 나타내는 함수

### 2. 생성자
- 기본 생성자: 매개변수가 없는 생성자
- 매개변수 생성자: 필드를 초기화하는 생성자
- `this` 키워드: 현재 객체를 가리킴

### 3. 메서드 작성
- `toString()`: 객체를 문자열로 표현
- 계산 메서드: 필드 값을 이용한 계산
- 조건 메서드: boolean 값을 반환하는 메서드

### 4. 주의사항
- 메서드 반환 타입을 정확히 지정
- `toString()`은 `@Override` 어노테이션 사용 권장
- boolean 반환 시 조건식을 직접 반환 가능 (`return stock > 0;`)
