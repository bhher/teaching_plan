# 기본 자료형 (Primitive Data Types)

## 기본 자료형이란?

**기본 자료형(Primitive Data Types)**은 Java에서 제공하는 8가지 기본 데이터 타입입니다. 이들은 실제 값을 메모리에 직접 저장하며, 객체가 아닌 단순한 값입니다.

### 기본 자료형의 특징

- **실제 값을 저장**: 메모리에 직접 값이 저장됨
- **스택(Stack) 메모리 사용**: 빠른 접근 속도
- **기본값 제공**: 클래스 멤버 변수는 자동으로 기본값 할당
- **8가지 타입**: 정수형 4개, 실수형 2개, 문자형 1개, 논리형 1개

---

## 1. 정수형 (Integer Types)

정수형은 소수점이 없는 정수를 저장하는 자료형입니다. 크기에 따라 4가지 타입이 있습니다.

### 정수형 종류

| 자료형 | 크기 | 범위 | 기본값 | 사용 예시 |
|--------|------|------|--------|----------|
| `byte` | 1 byte (8 bits) | -128 ~ 127 | 0 | 작은 범위의 정수 |
| `short` | 2 bytes (16 bits) | -32,768 ~ 32,767 | 0 | 중간 범위의 정수 |
| `int` | 4 bytes (32 bits) | -2,147,483,648 ~ 2,147,483,647 | 0 | **가장 많이 사용** |
| `long` | 8 bytes (64 bits) | -9,223,372,036,854,775,808 ~ 9,223,372,036,854,775,807 | 0L | 큰 정수 |

### 정수형 사용 예제

```java
public class IntegerTypes {
    public static void main(String[] args) {
        // byte: 가장 작은 정수형
        byte byteValue = 100;
        byte maxByte = 127;
        byte minByte = -128;
        
        // short: 작은 정수형
        short shortValue = 1000;
        short year = 2024;
        
        // int: 가장 많이 사용하는 정수형 (기본값)
        int intValue = 100000;
        int age = 25;
        int score = 85;
        
        // long: 큰 정수형 (L 또는 l을 붙여야 함)
        long longValue = 10000000000L;  // L 또는 l 필수!
        long population = 7800000000L;
        long bigNumber = 9223372036854775807L;  // long의 최대값
        
        // 출력
        System.out.println("byte: " + byteValue);
        System.out.println("short: " + shortValue);
        System.out.println("int: " + intValue);
        System.out.println("long: " + longValue);
    }
}
```

### 정수형 주의사항

1. **long 타입 리터럴**: `long` 타입은 숫자 뒤에 `L` 또는 `l`을 붙여야 합니다.
   ```java
   long num1 = 10000000000L;  // 올바름
   long num2 = 10000000000;   // 오류! int 범위를 초과
   ```

2. **범위 초과**: 각 타입의 범위를 벗어나면 컴파일 오류가 발생합니다.
   ```java
   byte value = 128;  // 오류! byte 범위는 -128 ~ 127
   ```

3. **기본 타입**: 정수 리터럴은 기본적으로 `int` 타입입니다.

## 2. 실수형 (Floating-Point Types)

실수형은 소수점이 있는 실수를 저장하는 자료형입니다. 정밀도에 따라 2가지 타입이 있습니다.

### 실수형 종류

| 자료형 | 크기 | 범위 | 정밀도 | 기본값 | 사용 예시 |
|--------|------|------|--------|--------|----------|
| `float` | 4 bytes (32 bits) | 약 ±3.4×10³⁸ | 약 7자리 | 0.0f | 단정밀도 실수 |
| `double` | 8 bytes (64 bits) | 약 ±1.7×10³⁰⁸ | 약 15자리 | 0.0 | **기본값, 가장 많이 사용** |

### 실수형 사용 예제

```java
public class FloatingPointTypes {
    public static void main(String[] args) {
        // float: 단정밀도 실수 (f 또는 F를 붙여야 함)
        float floatValue = 3.14f;      // f 또는 F 필수!
        float weight = 70.5f;
        float temperature = -5.3f;
        
        // double: 배정밀도 실수 (기본값, 접미사 생략 가능)
        double doubleValue = 3.141592653589793;
        double height = 175.5;           // 접미사 없이 사용 가능
        double pi = 3.14159265358979323846;
        double price = 12500.99;
        
        // 과학적 표기법
        double bigNumber = 1.5e10;       // 1.5 × 10¹⁰
        double smallNumber = 2.5e-5;     // 2.5 × 10⁻⁵
        
        // 출력
        System.out.println("float: " + floatValue);
        System.out.println("double: " + doubleValue);
        System.out.println("큰 수: " + bigNumber);
        System.out.println("작은 수: " + smallNumber);
    }
}
```

### 실수형 주의사항

1. **float 타입 리터럴**: `float` 타입은 숫자 뒤에 `f` 또는 `F`를 붙여야 합니다.
   ```java
   float num1 = 3.14f;   // 올바름
   float num2 = 3.14;    // 오류! double로 인식됨
   ```

2. **기본 타입**: 실수 리터럴은 기본적으로 `double` 타입입니다.

3. **정밀도 손실**: `float`는 `double`보다 정밀도가 낮으므로, 정확한 계산이 필요한 경우 `double`을 사용합니다.

4. **부동소수점 오차**: 실수 연산 시 미세한 오차가 발생할 수 있습니다.
   ```java
   double result = 0.1 + 0.2;  // 0.30000000000000004 (미세한 오차)
   ```

## 3. 문자형 (Character Type)

문자형은 단일 문자를 저장하는 자료형입니다. Java는 유니코드(Unicode)를 사용하여 전 세계의 문자를 표현할 수 있습니다.

### 문자형 종류

| 자료형 | 크기 | 범위 | 기본값 | 설명 |
|--------|------|------|--------|------|
| `char` | 2 bytes (16 bits) | 0 ~ 65,535 (유니코드) | '\u0000' | 단일 문자 |

### 문자형 사용 예제

```java
public class CharacterType {
    public static void main(String[] args) {
        // 영문자
        char charValue = 'A';
        char lowercase = 'a';
        
        // 한글
        char koreanChar = '가';
        char koreanWord = '한';
        
        // 숫자 문자
        char digit = '5';
        
        // 특수 문자
        char space = ' ';
        char exclamation = '!';
        
        // 유니코드로 표현
        char unicodeChar = '\u0041';  // 'A' (16진수 유니코드)
        char unicodeKorean = '\uAC00'; // '가'
        
        // 이스케이프 시퀀스
        char newline = '\n';      // 줄바꿈
        char tab = '\t';          // 탭
        char quote = '\'';        // 작은따옴표
        char backslash = '\\';    // 백슬래시
        
        // 출력
        System.out.println("영문자: " + charValue);
        System.out.println("한글: " + koreanChar);
        System.out.println("유니코드: " + unicodeChar);
        System.out.println("줄바꿈" + newline + "테스트");
    }
}
```

### 문자형 특징

1. **유니코드 지원**: 전 세계 모든 문자를 표현 가능
2. **문자 리터럴**: 작은따옴표(`'`)로 감싸야 함
3. **숫자와의 관계**: `char`는 내부적으로 정수로 저장되므로 산술 연산 가능
   ```java
   char ch = 'A';
   int ascii = ch;  // 65 (ASCII 값)
   char next = (char)(ch + 1);  // 'B'
   ```

### 이스케이프 시퀀스

| 이스케이프 시퀀스 | 의미 |
|-----------------|------|
| `\n` | 줄바꿈 (Newline) |
| `\t` | 탭 (Tab) |
| `\'` | 작은따옴표 |
| `\"` | 큰따옴표 |
| `\\` | 백슬래시 |
| `\r` | 캐리지 리턴 |
| `\uXXXX` | 유니코드 문자 (16진수) |

## 4. 논리형 (Boolean Type)

논리형은 참(`true`) 또는 거짓(`false`) 값을 저장하는 자료형입니다. 조건문과 반복문에서 주로 사용됩니다.

### 논리형 종류

| 자료형 | 크기 | 값 | 기본값 | 설명 |
|--------|------|-----|--------|------|
| `boolean` | 1 bit (실제로는 1 byte 사용) | `true` 또는 `false` | `false` | 논리값 |

### 논리형 사용 예제

```java
public class BooleanType {
    public static void main(String[] args) {
        // 직접 값 할당
        boolean isTrue = true;
        boolean isFalse = false;
        
        // 비교 연산 결과
        boolean result1 = (10 > 5);      // true
        boolean result2 = (10 < 5);      // false
        boolean result3 = (10 == 10);    // true
        boolean result4 = (10 != 5);     // true
        
        // 논리 연산 결과
        boolean andResult = (true && false);  // false
        boolean orResult = (true || false);   // true
        boolean notResult = !true;            // false
        
        // 실용적인 예제
        int age = 20;
        boolean isAdult = (age >= 18);        // true
        boolean canVote = (age >= 18);        // true
        
        int score = 85;
        boolean isPass = (score >= 60);      // true
        boolean isExcellent = (score >= 90); // false
        
        // 출력
        System.out.println("isTrue: " + isTrue);
        System.out.println("10 > 5: " + result1);
        System.out.println("성인 여부: " + isAdult);
        System.out.println("합격 여부: " + isPass);
    }
}
```

### 논리형 특징

1. **두 가지 값만 가능**: `true` 또는 `false`만 저장 가능
2. **숫자와 호환 안 됨**: C/C++와 달리 숫자로 변환 불가
   ```java
   // Java에서는 불가능
   // boolean flag = 1;  // 오류!
   // int num = true;     // 오류!
   ```
3. **조건문과 반복문**: `if`, `while`, `for` 등에서 필수적으로 사용

## 기본값 (Default Values)

클래스의 멤버 변수(필드)는 자동으로 기본값이 할당됩니다. 하지만 **지역 변수(메서드 내부 변수)는 반드시 초기화해야 합니다!**

### 기본값 정리

| 자료형 | 기본값 |
|--------|--------|
| `byte` | `0` |
| `short` | `0` |
| `int` | `0` |
| `long` | `0L` |
| `float` | `0.0f` |
| `double` | `0.0` |
| `char` | `'\u0000'` (null 문자) |
| `boolean` | `false` |

### 기본값 예제

```java
public class DefaultValues {
    // 클래스 멤버 변수 - 기본값 자동 할당
    byte byteField;
    short shortField;
    int intField;
    long longField;
    float floatField;
    double doubleField;
    char charField;
    boolean booleanField;
    
    public static void main(String[] args) {
        DefaultValues obj = new DefaultValues();
        
        System.out.println("byte 기본값: " + obj.byteField);      // 0
        System.out.println("int 기본값: " + obj.intField);        // 0
        System.out.println("double 기본값: " + obj.doubleField); // 0.0
        System.out.println("char 기본값: " + (int)obj.charField); // 0
        System.out.println("boolean 기본값: " + obj.booleanField); // false
        
        // 지역 변수는 반드시 초기화해야 함!
        int localVar;
        // System.out.println(localVar);  // 오류! 초기화되지 않음
        
        int initializedVar = 10;  // 올바름
        System.out.println(initializedVar);
    }
}
```

    }

## 기본 자료형 선택 가이드

### 정수형 선택

- **byte**: 매우 작은 범위의 정수 (예: 점수, 나이 등 작은 값)
- **short**: 작은 범위의 정수 (거의 사용 안 함)
- **int**: **일반적인 정수 (가장 많이 사용)**
- **long**: 큰 정수 (예: 인구 수, 시간(밀리초) 등)

### 실수형 선택

- **float**: 메모리가 제한적인 경우, 정밀도가 중요하지 않은 경우
- **double**: **일반적인 실수 (가장 많이 사용, 기본값)**

### 실용적인 권장사항

```java
// 권장: 일반적으로 사용하는 타입
int age = 25;              // 정수는 int
double height = 175.5;      // 실수는 double
char grade = 'A';          // 문자는 char
boolean isActive = true;   // 논리는 boolean

// 특수한 경우에만 사용
byte score = 100;          // 메모리가 매우 중요한 경우
long timestamp = 1234567890L;  // 큰 정수가 필요한 경우
float weight = 70.5f;      // 메모리가 제한적인 경우
```

## 주의사항 및 팁

### 1. 리터럴 접미사

```java
long num = 100L;      // long은 L 또는 l
float num = 3.14f;    // float는 f 또는 F
double num = 3.14;    // double은 접미사 없음 (기본값)
```

### 2. 범위 확인

각 자료형의 범위를 벗어나면 컴파일 오류나 런타임 오류가 발생할 수 있습니다.

```java
byte value = 128;  // 오류! byte 범위는 -128 ~ 127
int maxInt = 2147483647;  // int의 최대값
```

### 3. 지역 변수 초기화

지역 변수는 반드시 초기화해야 합니다.

```java
public void method() {
    int num;  // 초기화 안 함
    // System.out.println(num);  // 오류!
    
    int initialized = 10;  // 올바름
    System.out.println(initialized);
}
```

### 4. 타입 변환

작은 타입에서 큰 타입으로는 자동 변환되지만, 큰 타입에서 작은 타입으로는 명시적 변환이 필요합니다.

```java
int num = 100;
double decimal = num;  // 자동 변환 (int → double)

double pi = 3.14;
int intPi = (int) pi;  // 강제 변환 (double → int), 소수점 손실
```
