## 문제 1: Student 클래스 작성 (20점)

### 문제 설명
학생 정보를 저장하는 `Student` 클래스를 작성하세요.

### 요구사항
1. **필드**
   - `name` (String): 학생 이름
   - `age` (int): 학생 나이
   - `score` (int): 학생 점수

2. **생성자**
   - 매개변수가 없는 기본 생성자
   - `name`, `age`, `score`를 받는 매개변수 생성자

3. **메서드**
   - `toString()`: 학생 정보를 다음 형식으로 반환
     ```
     "이름: [name], 나이: [age], 점수: [score]"
     ```
   - `getGrade()`: 점수에 따라 등급을 반환하는 메서드
     - 90점 이상: "A"
     - 80점 이상: "B"
     - 70점 이상: "C"
     - 60점 이상: "D"
     - 60점 미만: "F"

### 실행 코드 (제공됨)
```java
public class Problem1 {
    public static void main(String[] args) {
        Student s1 = new Student("홍길동", 20, 85);
        Student s2 = new Student("김철수", 22, 95);
        
        System.out.println(s1);
        System.out.println("등급: " + s1.getGrade());
        System.out.println(s2);
        System.out.println("등급: " + s2.getGrade());
    }
}
```

### 실행 결과 예시
```
이름: 홍길동, 나이: 20, 점수: 85
등급: B
이름: 김철수, 나이: 22, 점수: 95
등급: A
```

### 채점 기준
- 필드 선언: 5점
- 생성자 작성: 5점
- toString() 메서드: 5점
- getGrade() 메서드: 5점

## 문제 2: Product 클래스 작성 (20점)

### 문제 설명
상품 정보를 저장하는 `Product` 클래스를 작성하세요.

### 요구사항
1. **필드**
   - `name` (String): 상품명
   - `price` (int): 상품 가격
   - `stock` (int): 재고 수량

2. **생성자**
   - 매개변수가 없는 기본 생성자
   - `name`, `price`, `stock`을 받는 매개변수 생성자

3. **메서드**
   - `toString()`: 상품 정보를 다음 형식으로 반환
     ```
     "상품명: [name], 가격: [price]원, 재고: [stock]개"
     ```
   - `getTotalValue()`: 상품의 총 가치를 계산하는 메서드
     - 총 가치 = 가격 × 재고 수량
     - 반환값: int
   - `isAvailable()`: 재고가 있는지 확인하는 메서드
     - 재고가 0보다 크면 true, 0이면 false 반환
     - 반환값: boolean

### 실행 코드 (제공됨)
```java
public class Problem2 {
    public static void main(String[] args) {
        Product p1 = new Product("노트북", 1200000, 5);
        Product p2 = new Product("마우스", 25000, 0);
        
        System.out.println(p1);
        System.out.println("총 가치: " + p1.getTotalValue() + "원");
        System.out.println("판매 가능: " + p1.isAvailable());
        
        System.out.println();
        
        System.out.println(p2);
        System.out.println("총 가치: " + p2.getTotalValue() + "원");
        System.out.println("판매 가능: " + p2.isAvailable());
    }
}
```

### 실행 결과 예시
```
상품명: 노트북, 가격: 1200000원, 재고: 5개
총 가치: 6000000원
판매 가능: true

상품명: 마우스, 가격: 25000원, 재고: 0개
총 가치: 0원
판매 가능: false
```

### 채점 기준
- 필드 선언: 5점
- 생성자 작성: 5점
- toString() 메서드: 5점
- getTotalValue() 메서드: 3점
- isAvailable() 메서드: 2점

## 문제 3: Employee 클래스 작성 (20점)

### 문제 설명
직원 정보를 저장하는 `Employee` 클래스를 작성하세요.

### 요구사항
1. **필드**
   - `name` (String): 직원 이름
   - `department` (String): 부서명
   - `salary` (int): 급여

2. **생성자**
   - 매개변수가 없는 기본 생성자
   - `name`, `department`, `salary`를 받는 매개변수 생성자

3. **메서드**
   - `toString()`: 직원 정보를 다음 형식으로 반환
     ```
     "이름: [name], 부서: [department], 급여: [salary]원"
     ```
   - `getAnnualSalary()`: 연봉을 계산하는 메서드
     - 연봉 = 급여 × 12
     - 반환값: int
   - `getSalaryLevel()`: 급여 수준을 반환하는 메서드
     - 500만원 이상: "고급"
     - 300만원 이상: "중급"
     - 300만원 미만: "초급"

### 실행 코드 (제공됨)
```java
public class Problem3 {
    public static void main(String[] args) {
        Employee e1 = new Employee("홍길동", "개발팀", 4500000);
        Employee e2 = new Employee("김영희", "디자인팀", 2500000);
        
        System.out.println(e1);
        System.out.println("연봉: " + e1.getAnnualSalary() + "원");
        System.out.println("급여 수준: " + e1.getSalaryLevel());
        
        System.out.println();
        
        System.out.println(e2);
        System.out.println("연봉: " + e2.getAnnualSalary() + "원");
        System.out.println("급여 수준: " + e2.getSalaryLevel());
    }
}
```

### 실행 결과 예시
```
이름: 홍길동, 부서: 개발팀, 급여: 4500000원
연봉: 54000000원
급여 수준: 중급

이름: 김영희, 부서: 디자인팀, 급여: 2500000원
연봉: 30000000원
급여 수준: 초급
```

### 채점 기준
- 필드 선언: 5점
- 생성자 작성: 5점
- toString() 메서드: 5점
- getAnnualSalary() 메서드: 3점
- getSalaryLevel() 메서드: 2점

## 문제 4: Book 클래스 작성 (20점)

### 문제 설명
도서 정보를 저장하는 `Book` 클래스를 작성하세요.

### 요구사항
1. **필드**
   - `title` (String): 도서 제목
   - `author` (String): 저자명
   - `pages` (int): 페이지 수

2. **생성자**
   - 매개변수가 없는 기본 생성자
   - `title`, `author`, `pages`를 받는 매개변수 생성자

3. **메서드**
   - `toString()`: 도서 정보를 다음 형식으로 반환
     ```
     "제목: [title], 저자: [author], 페이지: [pages]"
     ```
   - `getReadingTime()`: 읽는 데 걸리는 시간을 계산하는 메서드
     - 읽는 시간(분) = 페이지 수 × 2 (페이지당 2분 가정)
     - 반환값: int
   - `getBookType()`: 도서 유형을 반환하는 메서드
     - 500페이지 이상: "장편"
     - 200페이지 이상: "중편"
     - 200페이지 미만: "단편"

### 실행 코드 (제공됨)
```java
public class Problem4 {
    public static void main(String[] args) {
        Book b1 = new Book("자바의 정석", "남궁성", 800);
        Book b2 = new Book("혼자 공부하는 자바", "신용권", 350);
        
        System.out.println(b1);
        System.out.println("예상 읽는 시간: " + b1.getReadingTime() + "분");
        System.out.println("도서 유형: " + b1.getBookType());
        
        System.out.println();
        
        System.out.println(b2);
        System.out.println("예상 읽는 시간: " + b2.getReadingTime() + "분");
        System.out.println("도서 유형: " + b2.getBookType());
    }
}
```

### 실행 결과 예시
```
제목: 자바의 정석, 저자: 남궁성, 페이지: 800
예상 읽는 시간: 1600분
도서 유형: 장편

제목: 혼자 공부하는 자바, 저자: 신용권, 페이지: 350
예상 읽는 시간: 700분
도서 유형: 중편
```

### 채점 기준
- 필드 선언: 5점
- 생성자 작성: 5점
- toString() 메서드: 5점
- getReadingTime() 메서드: 3점
- getBookType() 메서드: 2점
