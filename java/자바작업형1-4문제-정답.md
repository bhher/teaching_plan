## 문제 1 정답: Student 클래스

```java
public class Student {
    String name;
    int age;
    int score;
    
    // 기본 생성자
    public Student() {
    }
    
    // 매개변수 생성자
    public Student(String name, int age, int score) {
        this.name = name;
        this.age = age;
        this.score = score;
    }
    
    // toString() 메서드
    @Override
    public String toString() {
        return "이름: " + name + ", 나이: " + age + ", 점수: " + score;
    }
    
    // 등급 반환 메서드
    public String getGrade() {
        if (score >= 90) {
            return "A";
        } else if (score >= 80) {
            return "B";
        } else if (score >= 70) {
            return "C";
        } else if (score >= 60) {
            return "D";
        } else {
            return "F";
        }
    }
}
```

### 실행 결과
```
이름: 홍길동, 나이: 20, 점수: 85
등급: B
이름: 김철수, 나이: 22, 점수: 95
등급: A
```

## 문제 2 정답: Product 클래스

```java
public class Product {
    String name;
    int price;
    int stock;
    
    // 기본 생성자
    public Product() {
    }
    
    // 매개변수 생성자
    public Product(String name, int price, int stock) {
        this.name = name;
        this.price = price;
        this.stock = stock;
    }
    
    // toString() 메서드
    @Override
    public String toString() {
        return "상품명: " + name + ", 가격: " + price + "원, 재고: " + stock + "개";
    }
    
    // 총 가치 계산 메서드
    public int getTotalValue() {
        return price * stock;
    }
    
    // 재고 확인 메서드
    public boolean isAvailable() {
        return stock > 0;
    }
}
```

### 실행 결과
```
상품명: 노트북, 가격: 1200000원, 재고: 5개
총 가치: 6000000원
판매 가능: true

상품명: 마우스, 가격: 25000원, 재고: 0개
총 가치: 0원
판매 가능: false
```

## 문제 3 정답: Employee 클래스

public class Employee {
    String name;
    String department;
    int salary;
    
    // 기본 생성자
    public Employee() {
    }

    public Employee(String name, String department, int salary) {
        this.name = name;
        this.department = department;
        this.salary = salary;
    }

    @Override
    public String toString() {

    // 연봉 계산 메서드

            return "고급";

            return "중급";
        } else {
            return "초급";
        }
    }
}

## 문제 4 정답: Book 클래스

```java
public class Book {
    String title;
    String author;
    int pages;
    
    // 기본 생성자
    public Book() {
    }
    
    // 매개변수 생성자
    public Book(String title, String author, int pages) {
        this.title = title;
        this.author = author;
        this.pages = pages;
    }
    
    // toString() 메서드
    @Override
    public String toString() {
        return "제목: " + title + ", 저자: " + author + ", 페이지: " + pages;
    }
    
    // 읽는 시간 계산 메서드
    public int getReadingTime() {
        return pages * 2;
    }
    
    // 도서 유형 반환 메서드
    public String getBookType() {
        if (pages >= 500) {
            return "장편";
        } else if (pages >= 200) {
            return "중편";
        } else {
            return "단편";
        }
    }
}
```

## 공통 학습 포인트

### 1. 클래스 기본 구조
- **필드**: 클래스의 속성을 나타내는 변수
- **생성자**: 객체를 생성할 때 호출되는 메서드
- **메서드**: 클래스의 기능을 나타내는 함수

### 2. 생성자 작성 패턴
```java
// 기본 생성자
public ClassName() {
}

// 매개변수 생성자
public ClassName(Type1 param1, Type2 param2, Type3 param3) {
    this.field1 = param1;
    this.field2 = param2;
    this.field3 = param3;
}
```

### 3. toString() 메서드 패턴
```java
@Override
public String toString() {
    return "필드1: " + field1 + ", 필드2: " + field2 + ", 필드3: " + field3;
}
```

### 4. 조건문 메서드 패턴
```java
// if-else if 구조
public String getLevel() {
    if (condition1) {
        return "값1";
    } else if (condition2) {
        return "값2";
    } else {
        return "값3";
    }
}
```

### 5. 계산 메서드 패턴
```java
// 단순 계산
public int calculate() {
    return value1 * value2;
}

// 조건부 계산
public int calculate() {
    if (condition) {
        return value1 * value2;
    }
    return 0;
}
```

## 자주 하는 실수

### 1. 필드명 오타
```java
// 잘못된 예
String nmae;  // name 오타
int prcie;    // price 오타
```

### 2. 생성자에서 this 미사용
```java
// 잘못된 예
public Student(String name, int age, int score) {
    name = name;  // 필드가 아닌 매개변수에 대입됨
}

// 올바른 예
public Student(String name, int age, int score) {
    this.name = name;  // this로 필드 구분
}
```

### 3. toString() 형식 불일치
```java
// 잘못된 예
return name + age + score;  // 형식이 다름

// 올바른 예
return "이름: " + name + ", 나이: " + age + ", 점수: " + score;
```

### 4. 반환 타입 오류
```java
// 잘못된 예
public String getTotalValue() {  // int여야 함
    return price * stock;
}

public int isAvailable() {  // boolean이어야 함
    return stock > 0;
}
```

### 5. 조건문 순서 오류
```java
// 잘못된 예 (낮은 값부터 확인)
if (score >= 60) {
    return "D";
} else if (score >= 90) {  // 이 조건은 절대 실행되지 않음
    return "A";
}

// 올바른 예 (높은 값부터 확인)
if (score >= 90) {
    return "A";
} else if (score >= 60) {
    return "D";
}
```
