# 학생 관리 시스템 실습 문제 (ArrayList 버전, 점수 포함)

## 📌 문제 설명

은행 애플리케이션과 유사한 구조로 **학생 관리 시스템**을 구현하세요.

**ArrayList**를 사용하여 동적으로 학생 정보를 관리하는 프로그램을 작성하세요.

**점수 포함 버전**: 학생의 국어, 영어, 수학 점수를 포함하여 총점과 평균을 자동으로 계산합니다.

> **배열 버전과의 차이점**: 배열은 크기가 고정되어 있지만, ArrayList는 동적으로 크기가 조정되므로 학생 수 제한이 없습니다.

---

## 1️⃣ Student 클래스 작성

### 요구사항

다음 조건을 만족하는 `Student` 클래스를 작성하세요.

#### 필드 (모두 private)

- `String studentId` : 학번
- `String name` : 이름
- `int age` : 나이
- `String major` : 전공
- `int kor` : 국어 점수
- `int eng` : 영어 점수
- `int math` : 수학 점수

#### 생성자

- 모든 필드를 매개변수로 받아 초기화하는 생성자

#### Getter 메서드

- `getStudentId()` : 학번 반환
- `getName()` : 이름 반환
- `getAge()` : 나이 반환
- `getMajor()` : 전공 반환
- `getKor()` : 국어 점수 반환
- `getEng()` : 영어 점수 반환
- `getMath()` : 수학 점수 반환

#### Setter 메서드

- `setName(String name)` : 이름 설정
- `setAge(int age)` : 나이 설정
- `setMajor(String major)` : 전공 설정
- `setKor(int kor)` : 국어 점수 설정
- `setEng(int eng)` : 영어 점수 설정
- `setMath(int math)` : 수학 점수 설정

#### 계산 메서드

- `calculateTotal()` : 총점 계산 (국어 + 영어 + 수학)
- `calculateAverage()` : 평균 계산 (총점 / 3.0)

## 2️⃣ StudentApplication 클래스 작성

### 요구사항

다음 기능을 가진 `StudentApplication` 클래스를 작성하세요.

#### 필드

- `private static ArrayList<Student> studentsList = new ArrayList<>()` : 모든 학생 정보를 저장하는 ArrayList
- `private static Scanner scanner = new Scanner(System.in)` : 사용자 입력을 받기 위한 Scanner

#### 메뉴 시스템

프로그램 실행 시 다음과 같은 메뉴를 출력하고 사용자 선택에 따라 기능을 실행합니다.

```
-----------------------------------------------------------------------
1. 학생등록 | 2. 학생목록 | 3. 학생검색 | 4. 학생수정 | 5. 학생삭제 | 6. 종료
-----------------------------------------------------------------------
선택> 
```

#### 기능 1: 학생등록 (registerStudent)

- 학번, 이름, 나이, 전공, 국어, 영어, 수학 점수를 입력받습니다.
- **중복 체크**: 이미 존재하는 학번인지 확인합니다.
- **점수 유효성 검사**: 점수는 0 이상 100 이하의 정수만 허용합니다.
- 중복이 아니고 점수가 유효한 경우 새 Student 객체를 생성하고 ArrayList에 추가합니다.
- 등록 완료 메시지를 출력합니다.

**입력 예시:**
```
----------- 
학생등록
----------- 
학번: 2024001
이름: 홍길동
나이: 20
전공: 컴퓨터공학
국어: 85
영어: 90
수학: 88
결과: 학생이 등록되었습니다.
```

**에러 처리:**
- 이미 존재하는 학번인 경우: "이미 존재하는 학번입니다."
- 나이 입력 오류: "잘못된 입력입니다. 숫자를 입력하세요."
- 점수 입력 오류: "잘못된 입력입니다. 숫자를 입력하세요."
- 점수 범위 오류: "점수는 0 이상 100 이하의 정수여야 합니다."

#### 기능 2: 학생목록 (studentList)

- 등록된 모든 학생의 정보를 출력합니다.
- 학번, 이름, 나이, 전공, 국어, 영어, 수학, 총점, 평균을 표 형태로 출력합니다.
- 평균은 소수점 둘째 자리까지 출력합니다.
- 등록된 학생이 없으면 아무것도 출력하지 않습니다.

**출력 예시:**
```
----------- 
학생목록
----------- 
2024001 홍길동   20 컴퓨터공학   국어:85 영어:90 수학:88 총점:263 평균:87.67
2024002 김철수   21 소프트웨어   국어:92 영어:88 수학:95 총점:275 평균:91.67
2024003 이영희   19 정보통신     국어:78 영어:85 수학:82 총점:245 평균:81.67
```

#### 기능 3: 학생검색 (searchStudent)

- 학번을 입력받습니다.
- 해당 학번의 학생을 찾아 정보를 출력합니다 (점수, 총점, 평균 포함).
- 학생을 찾지 못한 경우: "결과: 학생을 찾을 수 없습니다."

**입력/출력 예시:**
```
----------- 
학생검색
----------- 
학번: 2024001
결과: 학생을 찾았습니다.
학번: 2024001, 이름: 홍길동, 나이: 20, 전공: 컴퓨터공학
국어: 85, 영어: 90, 수학: 88, 총점: 263, 평균: 87.67
```

#### 기능 4: 학생수정 (updateStudent)

- 학번을 입력받습니다.
- 해당 학번의 학생을 찾습니다.
- 이름, 나이, 전공, 국어, 영어, 수학을 새로 입력받아 수정합니다.
- 점수 수정 시 유효성 검사를 수행합니다.
- 학생을 찾지 못한 경우: "결과: 학생을 찾을 수 없습니다."
- 수정 후 총점과 평균이 자동으로 재계산됩니다.

**입력/출력 예시:**
```
----------- 
학생수정
----------- 
학번: 2024001
이름(수정): 홍길동
나이(수정): 21
전공(수정): 소프트웨어
국어(수정): 90
영어(수정): 92
수학(수정): 88
결과: 학생 정보가 수정되었습니다.
```

#### 기능 5: 학생삭제 (deleteStudent)

- 학번을 입력받습니다.
- 해당 학번의 학생을 찾아 ArrayList에서 제거합니다.
- 학생을 찾지 못한 경우: "결과: 학생을 찾을 수 없습니다."
- 삭제 완료 메시지를 출력합니다.

**입력/출력 예시:**
```
----------- 
학생삭제
----------- 
학번: 2024001
결과: 학생이 삭제되었습니다.
```

#### 보조 메서드: findStudent

- 학번(String)을 매개변수로 받습니다.
- ArrayList에서 해당 학번을 가진 Student 객체를 찾아 반환합니다.
- 찾지 못한 경우 `null`을 반환합니다.
- `private static Student findStudent(String studentId)` 형태로 작성합니다.

#### 종료 (6번 선택)

- "프로그램 종료" 메시지를 출력하고 프로그램을 종료합니다.

## 3️⃣ 추가 요구사항

### 예외 처리

- 나이 및 점수 입력 시 숫자가 아닌 값이 입력되면 `try-catch`를 사용하여 처리합니다.
- 점수는 0 이상 100 이하의 정수만 허용합니다.
- 잘못된 입력 시 적절한 메시지를 출력하고 다시 입력을 받을 수 있도록 합니다.

### 점수 계산

- 총점과 평균은 Student 클래스의 메서드로 계산합니다.
- 평균은 `double` 타입으로 계산하되, 출력 시 소수점 둘째 자리까지 표시합니다.

### ArrayList 관리

- ArrayList는 동적으로 크기가 조정되므로 빈 자리를 찾을 필요가 없습니다.
- 학생 추가 시 `add()` 메서드로 바로 추가 가능합니다.
- 학생 삭제 시 `remove()` 메서드로 바로 삭제 가능합니다 (배열과 달리 null 처리 불필요).

### 메뉴 반복

- 6번(종료)을 선택할 때까지 메뉴를 반복 출력합니다.
- `while` 반복문을 사용합니다.

## 5️⃣ 힌트

<details>
<summary>힌트 보기</summary>

1. **ArrayList import**: 
   ```java
   import java.util.ArrayList;
   ```

2. **점수 입력 및 유효성 검사**: 
   ```java
   int kor = 0;
   try {
       kor = Integer.parseInt(scanner.nextLine());
       if (kor < 0 || kor > 100) {
           System.out.println("점수는 0 이상 100 이하의 정수여야 합니다.");
           return;
       }
   } catch (Exception e) {
       System.out.println("잘못된 입력입니다. 숫자를 입력하세요.");
       return;
   }
   ```

3. **총점과 평균 계산**: 
   ```java
   // Student 클래스의 메서드 사용
   int total = student.calculateTotal();
   double average = student.calculateAverage();
   ```

4. **평균 출력 (소수점 둘째 자리)**: 
   ```java
   System.out.printf("평균: %.2f\n", average);
   // 또는
   System.out.printf("평균: %.2f\n", student.calculateAverage());
   ```

5. **배열 vs ArrayList 비교**:
   - 배열: 빈 자리 찾아야 함, null 체크 필요
   - ArrayList: 바로 추가 가능, null 체크 불필요

6. **점수 수정 시 자동 재계산**: 
   - Setter로 점수를 수정하면, `calculateTotal()`과 `calculateAverage()`를 다시 호출하면 자동으로 재계산됨
