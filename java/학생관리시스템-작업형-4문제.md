## 문제 1: 학생 관리 시스템 - 메뉴 기반 (20점)

### 문제 설명

학생 정보를 관리하는 메뉴 기반 프로그램을 작성하세요. 
사용자가 메뉴를 선택하여 학생을 등록, 검색, 수정, 삭제, 전체 출력할 수 있도록 구현하세요.

### 요구사항

#### 1. Student 클래스 작성
- **필드**
  - `name` (String): 학생 이름
  - `age` (int): 학생 나이
- **생성자**
  - `Student(String name, int age)`: 이름과 나이를 받는 생성자
- **메서드**
  - `toString()`: 학생 정보를 문자열로 반환
    - 형식: `"Student{name='이름', age=나이}"`

#### 2. StudentApp 클래스 작성

**메뉴 시스템**
다음 메뉴를 반복해서 표시하고 사용자 입력을 받아 처리하세요:

```
------------------------
1. 학생 등록
2. 학생 검색
3. 학생 수정
4. 학생 삭제
5. 전체 출력
6. 종료
선택 > 
```

**메뉴별 기능**
- **1. 학생 등록**: 이름과 나이를 입력받아 학생을 등록
- **2. 학생 검색**: 검색할 이름을 입력받아 학생을 검색 (찾으면 출력, 없으면 "학생이 없습니다")
- **3. 학생 수정**: 수정할 이름과 새로운 나이를 입력받아 수정 (성공: "수정 완료", 실패: "해당 학생이 없습니다")
- **4. 학생 삭제**: 삭제할 이름을 입력받아 삭제 (성공: "삭제 완료", 실패: "삭제할 학생이 없습니다")
- **5. 전체 출력**: 등록된 모든 학생 정보 출력 (없으면 "등록된 학생이 없습니다")
- **6. 종료**: 프로그램 종료

**예외 처리**
- 메뉴 선택 시 숫자가 아닌 값 입력 시 "숫자만 입력하세요" 출력 후 다시 메뉴 표시
- 잘못된 메뉴 번호 입력 시 "1~6번 중에 선택하세요" 출력
- 나이 입력 시 숫자가 아닌 값 입력 시 "나이는 숫자로 입력하세요" 출력 후 다시 입력

**필요한 메서드**
```java
// 학생 검색
static Student findStudent(ArrayList<Student> list, String name)

// 학생 수정
static boolean updateStudent(ArrayList<Student> list, String name, int newAge)

// 학생 삭제
static boolean deleteStudent(ArrayList<Student> list, String name)
```

## 문제 2: Product 클래스 작성 (20점)

### 문제 설명

상품 정보를 저장하는 `Product` 클래스를 작성하세요.

### 요구사항

1. **필드**
   - `name` (String): 상품 이름
   - `price` (int): 상품 가격
   - `quantity` (int): 상품 수량

2. **생성자**
   - 매개변수가 없는 기본 생성자
   - `name`, `price`, `quantity`를 받는 매개변수 생성자

3. **메서드**
   - `toString()`: 상품 정보를 다음 형식으로 반환
     ```
     "상품명: [name], 가격: [price]원, 수량: [quantity]개"
     ```
   - `getTotalPrice()`: 가격과 수량을 곱한 총 가격을 반환하는 메서드
     - 반환값: int (총 가격)

### 실행 코드 (제공됨)

```java
public class Problem2 {
    public static void main(String[] args) {
        Product p1 = new Product("노트북", 1000000, 2);
        Product p2 = new Product("마우스", 15000, 5);
        
        System.out.println(p1);
        System.out.println("총 가격: " + p1.getTotalPrice() + "원");
        System.out.println(p2);
        System.out.println("총 가격: " + p2.getTotalPrice() + "원");
    }
}
```

### 실행 결과 예시

```
상품명: 노트북, 가격: 1000000원, 수량: 2개
총 가격: 2000000원
상품명: 마우스, 가격: 15000원, 수량: 5개
총 가격: 75000원
```

## 문제 3: ArrayList를 이용한 숫자 관리 (20점)

### 문제 설명

ArrayList를 사용하여 정수 리스트를 관리하는 프로그램을 작성하세요.

### 요구사항

다음 static 메서드들을 `NumberManager` 클래스에 작성하세요:

1. **`addNumber(ArrayList<Integer> list, int num)`**
   - 숫자를 리스트에 추가하는 메서드
   - 반환값: 없음

2. **`findMax(ArrayList<Integer> list)`**
   - 리스트에서 최댓값을 찾는 메서드
   - 반환값: int (최댓값)
   - 리스트가 비어있으면 -1 반환

3. **`findMin(ArrayList<Integer> list)`**
   - 리스트에서 최솟값을 찾는 메서드
   - 반환값: int (최솟값)
   - 리스트가 비어있으면 -1 반환

4. **`getSum(ArrayList<Integer> list)`**
   - 리스트에 있는 모든 숫자의 합을 계산하는 메서드
   - 반환값: int (합계)

### 실행 코드 (제공됨)

```java
import java.util.ArrayList;

public class Problem3 {
    public static void main(String[] args) {
        ArrayList<Integer> list = new ArrayList<>();
        
        NumberManager.addNumber(list, 10);
        NumberManager.addNumber(list, 5);
        NumberManager.addNumber(list, 20);
        NumberManager.addNumber(list, 8);
        NumberManager.addNumber(list, 15);
        
        System.out.println("최댓값: " + NumberManager.findMax(list));
        System.out.println("최솟값: " + NumberManager.findMin(list));
        System.out.println("합계: " + NumberManager.getSum(list));
    }
}
```

### 실행 결과 예시

## 문제 4: 문자열 처리 유틸리티 (20점)

### 문제 설명

문자열을 처리하는 유틸리티 메서드들을 작성하세요.

### 요구사항

다음 static 메서드들을 `StringUtil` 클래스에 작성하세요:

1. **`countChar(String text, char ch)`**
   - 문자열에서 특정 문자가 나타나는 횟수를 세는 메서드
   - 반환값: int (문자 개수)
   - null이면 0 반환
   - 예: `countChar("Hello", 'l')` → 2

2. **`reverseString(String text)`**
   - 문자열을 뒤집는 메서드
   - 반환값: String (뒤집은 문자열)
   - null이면 null 반환
   - 예: `reverseString("자바")` → "바자"

3. **`toUpperCase(String text)`**
   - 문자열을 대문자로 변환하는 메서드
   - 반환값: String (대문자로 변환된 문자열)
   - null이면 null 반환
   - 예: `toUpperCase("hello")` → "HELLO", `toUpperCase("Java")` → "JAVA"

4. **`removeSpaces(String text)`**
   - 문자열에서 모든 공백을 제거하는 메서드
   - 반환값: String (공백 제거된 문자열)
   - null이면 null 반환
   - 예: `removeSpaces("Hello World")` → "HelloWorld"

### 실행 코드 (제공됨)

```java
public class Problem4 {
    public static void main(String[] args) {
        // 1. 문자 개수 세기
        System.out.println("'l' 개수: " + StringUtil.countChar("Hello", 'l'));
        
        // 2. 문자열 뒤집기
        System.out.println("뒤집기: " + StringUtil.reverseString("자바"));
        
        // 3. 대문자 변환
        System.out.println("대문자: " + StringUtil.toUpperCase("hello"));
        System.out.println("대문자: " + StringUtil.toUpperCase("Java"));
        
        // 4. 공백 제거
        System.out.println("공백 제거: " + StringUtil.removeSpaces("Hello World"));
    }
}
```

### 실행 결과 예시

```

뒤집기: 바자
대문자: HELLO
대문자: JAVA

```

### 채점 기준
- countChar() 메서드: 5점
- reverseString() 메서드: 5점
- toUpperCase() 메서드: 5점
- removeSpaces() 메서드: 5점

### 평가 내용

#### countChar() 메서드 (5점)
- **메서드 시그니처 (1점)**
  - `public static int countChar(String text, char ch)`: 1점
- **null 처리 (1점)**
  - `if (text == null) return 0;`: 1점
- **문자 개수 세기 로직 (3점)**
  - 카운터 변수 초기화 (`int count = 0`): 0.5점
  - 반복문으로 문자열 순회: 1점
  - `text.charAt(i) == ch` 비교: 1점
  - 일치하면 카운터 증가 (`count++`): 0.5점

#### reverseString() 메서드 (5점)
- **메서드 시그니처 (1점)**
  - `public static String reverseString(String text)`: 1점
- **null 처리 (1점)**
  - `if (text == null) return null;`: 1점
- **문자열 뒤집기 로직 (3점)**
  - StringBuilder 사용: 1점
  - 역순 반복문 (`i = text.length() - 1; i >= 0; i--`): 1점
  - 문자 추가 (`sb.append(text.charAt(i))`): 1점
  - `sb.toString()` 반환: 포함

#### toUpperCase() 메서드 (5점)
- **메서드 시그니처 (1점)**
  - `public static String toUpperCase(String text)`: 1점
- **null 처리 (1점)**
  - `if (text == null) return null;`: 1점
- **대문자 변환 로직 (3점)**
  - StringBuilder 사용: 0.5점
  - 반복문으로 문자열 순회: 0.5점
  - 소문자 판별 (`ch >= 'a' && ch <= 'z'`): 1점
  - 대문자 변환 (`(char)(ch - 32)`): 1점
  - 다른 문자는 그대로 추가: 포함

#### removeSpaces() 메서드 (5점)
- **메서드 시그니처 (1점)**
  - `public static String removeSpaces(String text)`: 1점
- **null 처리 (1점)**
  - `if (text == null) return null;`: 1점
- **공백 제거 로직 (3점)**
  - StringBuilder 사용: 0.5점
  - 반복문으로 문자열 순회: 0.5점
  - 공백 판별 (`ch != ' '`): 1점
  - 공백이 아닌 문자만 추가: 1점
  - `sb.toString()` 반환: 포함

**감점 사항:**
- 컴파일 오류: 해당 문제 전체 0점
- 클래스명/메서드명 불일치: 각 항목당 1점 감점
- null 처리 누락: 각 메서드당 1점 감점
- 로직 오류: 각 메서드당 2점 감점
- 반환 타입 불일치: 각 메서드당 1점 감점
