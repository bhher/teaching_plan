# 학생 관리 시스템 실습 문제 (ArrayList 버전)

## 📌 문제 설명

은행 애플리케이션과 유사한 구조로 **학생 관리 시스템**을 구현하세요.

**ArrayList**를 사용하여 동적으로 학생 정보를 관리하는 프로그램을 작성하세요.

> **배열 버전과의 차이점**: 배열은 크기가 고정되어 있지만, ArrayList는 동적으로 크기가 조정되므로 학생 수 제한이 없습니다.

---

## 1️⃣ Student 클래스 작성

### 요구사항

다음 조건을 만족하는 `Student` 클래스를 작성하세요.

#### 필드 (모두 private)

- `String studentId` : 학번
- `String name` : 이름
- `int age` : 나이
- `String major` : 전공

#### 생성자

- 모든 필드를 매개변수로 받아 초기화하는 생성자

#### Getter 메서드

- `getStudentId()` : 학번 반환
- `getName()` : 이름 반환
- `getAge()` : 나이 반환
- `getMajor()` : 전공 반환

#### Setter 메서드

- `setName(String name)` : 이름 설정
- `setAge(int age)` : 나이 설정
- `setMajor(String major)` : 전공 설정

## 2️⃣ StudentApplication 클래스 작성

### 요구사항

다음 기능을 가진 `StudentApplication` 클래스를 작성하세요.

#### 필드

- `private static ArrayList<Student> studentsList = new ArrayList<>()` : 모든 학생 정보를 저장하는 ArrayList
- `private static Scanner scanner = new Scanner(System.in)` : 사용자 입력을 받기 위한 Scanner

> **배열과의 차이점**: 
> - 배열: `Student[] studentsArray = new Student[100]` (크기 고정)
> - ArrayList: `ArrayList<Student> studentsList = new ArrayList<>()` (동적 크기)

#### 메뉴 시스템

프로그램 실행 시 다음과 같은 메뉴를 출력하고 사용자 선택에 따라 기능을 실행합니다.

```
-----------------------------------------------------------------------
1. 학생등록 | 2. 학생목록 | 3. 학생검색 | 4. 학생수정 | 5. 학생삭제 | 6. 종료
-----------------------------------------------------------------------
선택> 
```

#### 기능 1: 학생등록 (registerStudent)

- 학번, 이름, 나이, 전공을 입력받습니다.
- **중복 체크**: 이미 존재하는 학번인지 확인합니다.
- 중복이 아닌 경우 새 Student 객체를 생성하고 ArrayList에 추가합니다.
- 등록 완료 메시지를 출력합니다.

**입력 예시:**
```
----------- 
학생등록
----------- 
학번: 2024001
이름: 홍길동
나이: 20
전공: 컴퓨터공학
결과: 학생이 등록되었습니다.
```

**에러 처리:**
- 이미 존재하는 학번인 경우: "이미 존재하는 학번입니다."
- 나이 입력 오류: "잘못된 입력입니다. 숫자를 입력하세요."

**ArrayList 사용법:**
```java
// ArrayList에 추가
studentsList.add(newStudent);
// 배열과 달리 빈 자리를 찾을 필요 없이 바로 추가 가능!
```

#### 기능 2: 학생목록 (studentList)

- 등록된 모든 학생의 정보를 출력합니다.
- 학번, 이름, 나이, 전공을 표 형태로 출력합니다.
- 등록된 학생이 없으면 아무것도 출력하지 않습니다.

**출력 예시:**
```
----------- 
학생목록
----------- 
2024001 홍길동   20 컴퓨터공학
2024002 김철수   21 소프트웨어
2024003 이영희   19 정보통신
```

**ArrayList 사용법:**
```java
// ArrayList 크기 확인
int size = studentsList.size();  // 배열의 .length 대신 .size() 사용

// ArrayList 순회
for (int i = 0; i < studentsList.size(); i++) {
    Student student = studentsList.get(i);  // 배열의 [i] 대신 .get(i) 사용
    // 출력
}

// 또는 향상된 for문
for (Student student : studentsList) {
    // 출력
}
```

#### 기능 3: 학생검색 (searchStudent)

- 학번을 입력받습니다.
- 해당 학번의 학생을 찾아 정보를 출력합니다.
- 학생을 찾지 못한 경우: "결과: 학생을 찾을 수 없습니다."

**입력/출력 예시:**
```
----------- 
학생검색
----------- 
학번: 2024001
결과: 학생을 찾았습니다.
학번: 2024001, 이름: 홍길동, 나이: 20, 전공: 컴퓨터공학
```

#### 기능 4: 학생수정 (updateStudent)

- 학번을 입력받습니다.
- 해당 학번의 학생을 찾습니다.
- 이름, 나이, 전공을 새로 입력받아 수정합니다.
- 학생을 찾지 못한 경우: "결과: 학생을 찾을 수 없습니다."

**입력/출력 예시:**
```
----------- 
학생수정
----------- 
학번: 2024001
이름(수정): 홍길동
나이(수정): 21
전공(수정): 소프트웨어
결과: 학생 정보가 수정되었습니다.
```

**ArrayList 사용법:**
```java
// Student 객체 찾기 (findStudent 메서드 사용)
Student student = findStudent(studentId);

// Setter 메서드로 수정
student.setName(newName);
student.setAge(newAge);
student.setMajor(newMajor);
```

#### 기능 5: 학생삭제 (deleteStudent)

- 학번을 입력받습니다.
- 해당 학번의 학생을 찾아 ArrayList에서 제거합니다.
- 학생을 찾지 못한 경우: "결과: 학생을 찾을 수 없습니다."
- 삭제 완료 메시지를 출력합니다.

**입력/출력 예시:**
```
----------- 
학생삭제
----------- 
학번: 2024001
결과: 학생이 삭제되었습니다.
```

**ArrayList 사용법:**
```java
// 방법 1: 인덱스로 삭제
studentsList.remove(index);

// 방법 2: 객체로 삭제
Student student = findStudent(studentId);
if (student != null) {
    studentsList.remove(student);  // equals()로 비교하여 삭제
}

// 배열과 달리 null로 설정할 필요 없이 바로 삭제 가능!
```

#### 보조 메서드: findStudent

- 학번(String)을 매개변수로 받습니다.
- ArrayList에서 해당 학번을 가진 Student 객체를 찾아 반환합니다.
- 찾지 못한 경우 `null`을 반환합니다.
- `private static Student findStudent(String studentId)` 형태로 작성합니다.

**ArrayList 사용법:**
```java
private static Student findStudent(String studentId) {
    // 방법 1: 반복문으로 찾기
    for (int i = 0; i < studentsList.size(); i++) {
        Student student = studentsList.get(i);
        if (student.getStudentId().equals(studentId)) {
            return student;
        }
    }
    return null;
    
    // 방법 2: 향상된 for문 사용
    for (Student student : studentsList) {
        if (student.getStudentId().equals(studentId)) {
            return student;
        }
    }
    return null;
}
```

#### 종료 (6번 선택)

- "프로그램 종료" 메시지를 출력하고 프로그램을 종료합니다.

## 3️⃣ 추가 요구사항

### 예외 처리

- 나이 입력 시 숫자가 아닌 값이 입력되면 `try-catch`를 사용하여 처리합니다.
- 잘못된 입력 시 적절한 메시지를 출력하고 다시 입력을 받을 수 있도록 합니다.

### ArrayList 관리

- ArrayList는 동적으로 크기가 조정되므로 빈 자리를 찾을 필요가 없습니다.
- 학생 추가 시 `add()` 메서드로 바로 추가 가능합니다.
- 학생 삭제 시 `remove()` 메서드로 바로 삭제 가능합니다 (배열과 달리 null 처리 불필요).

### 메뉴 반복

- 6번(종료)을 선택할 때까지 메뉴를 반복 출력합니다.
- `while` 반복문을 사용합니다.

## 5️⃣ 힌트

<details>
<summary>힌트 보기</summary>

1. **ArrayList import**: 
   ```java
   import java.util.ArrayList;
   ```

2. **ArrayList 생성**: 
   ```java
   ArrayList<Student> studentsList = new ArrayList<>();
   ```

3. **배열 vs ArrayList 비교**:
   | 작업 | 배열 | ArrayList |
   |------|------|-----------|
   | 크기 확인 | `array.length` | `list.size()` |
   | 요소 접근 | `array[i]` | `list.get(i)` |
   | 요소 추가 | 불가능 (크기 고정) | `list.add(element)` |
   | 요소 삭제 | 불가능 (null로 설정) | `list.remove(index)` 또는 `list.remove(object)` |
   | 빈 자리 찾기 | 필요 | 불필요 |

4. **학생 등록**: 
   ```java
   // 배열 버전: 빈 자리 찾아야 함
   for (int i = 0; i < studentsArray.length; i++) {
       if (studentsArray[i] == null) {
           studentsArray[i] = newStudent;
           break;
       }
   }
   
   // ArrayList 버전: 바로 추가 가능!
   studentsList.add(newStudent);
   ```

5. **학생 삭제**: 
   ```java
   // 배열 버전: null로 설정
   studentsArray[i] = null;
   
   // ArrayList 버전: 바로 삭제
   studentsList.remove(student);  // 또는 studentsList.remove(index);
   ```

6. **학생 목록**: 
   ```java
   // 배열 버전: null 체크 필요
   for (int i = 0; i < studentsArray.length; i++) {
       if (studentsArray[i] != null) {
           // 출력
       }
   }
   
   // ArrayList 버전: null 체크 불필요 (모든 요소가 유효함)
   for (Student student : studentsList) {
       // 출력
   }
   ```

7. **findStudent 메서드**: 배열 버전과 동일하게 작성하되, `studentsList.get(i)` 사용

## 7️⃣ 배열 버전과의 차이점 비교

| 구분 | 배열 버전 | ArrayList 버전 |
|------|----------|---------------|
| **선언** | `Student[] studentsArray = new Student[100]` | `ArrayList<Student> studentsList = new ArrayList<>()` |
| **크기 제한** | 최대 100명 (고정) | 제한 없음 (동적) |
| **크기 확인** | `.length` (필드) | `.size()` (메서드) |
| **요소 접근** | `array[i]` | `list.get(i)` |
| **요소 추가** | 빈 자리(null) 찾아서 저장 | `list.add(student)` 바로 추가 |
| **요소 삭제** | `array[i] = null` (빈 자리 생성) | `list.remove(student)` (실제 삭제) |
| **순회 시 null 체크** | 필요 | 불필요 |
| **메모리 효율** | 사용 안 하는 공간 낭비 가능 | 필요한 만큼만 사용 |
| **코드 복잡도** | 높음 (빈 자리 찾기 등) | 낮음 (간단한 메서드 호출) |
