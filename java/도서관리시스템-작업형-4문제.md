## 문제 1: 도서 관리 시스템 - 메뉴 기반 (20점)

### 문제 설명

도서 정보를 관리하는 메뉴 기반 프로그램을 작성하세요. 
사용자가 메뉴를 선택하여 도서를 등록, 검색, 수정, 삭제, 전체 출력할 수 있도록 구현하세요.

### 요구사항

#### 1. Book 클래스 작성
- **필드**
  - `title` (String): 도서 제목
  - `author` (String): 저자 이름
- **생성자**
  - `Book(String title, String author)`: 제목과 저자를 받는 생성자
- **메서드**
  - `toString()`: 도서 정보를 문자열로 반환
    - 형식: `"Book{title='제목', author='저자'}"`

#### 2. BookApp 클래스 작성

**메뉴 시스템**
다음 메뉴를 반복해서 표시하고 사용자 입력을 받아 처리하세요:

```
------------------------
1. 도서 등록
2. 도서 검색
3. 도서 수정
4. 도서 삭제
5. 전체 출력
6. 종료
선택 > 
```

**메뉴별 기능**
- **1. 도서 등록**: 제목과 저자를 입력받아 도서를 등록
- **2. 도서 검색**: 검색할 제목을 입력받아 도서를 검색 (찾으면 출력, 없으면 "도서가 없습니다")
- **3. 도서 수정**: 수정할 제목과 새로운 저자를 입력받아 수정 (성공: "수정 완료", 실패: "해당 도서가 없습니다")
- **4. 도서 삭제**: 삭제할 제목을 입력받아 삭제 (성공: "삭제 완료", 실패: "삭제할 도서가 없습니다")
- **5. 전체 출력**: 등록된 모든 도서 정보 출력 (없으면 "등록된 도서가 없습니다")
- **6. 종료**: 프로그램 종료

**예외 처리**
- 메뉴 선택 시 숫자가 아닌 값 입력 시 "숫자만 입력하세요" 출력 후 다시 메뉴 표시
- 잘못된 메뉴 번호 입력 시 "1~6번 중에 선택하세요" 출력

**필요한 메서드**
```java
// 도서 검색
static Book findBook(ArrayList<Book> list, String title)

// 도서 수정
static boolean updateBook(ArrayList<Book> list, String title, String newAuthor)

// 도서 삭제
static boolean deleteBook(ArrayList<Book> list, String title)
```

### 실행 결과 예시

```

2. 도서 검색
3. 도서 수정
4. 도서 삭제

6. 종료
선택 > 1
제목 입력 : 자바의 정석
저자 입력 : 남궁성
도서 등록 완료
------------------------
1. 도서 등록

Book{title='자바의 정석', author='남궁성'}
```

- Book 클래스 작성: 5점

## 문제 2: Car 클래스 작성 (20점)

### 문제 설명

자동차 정보를 저장하는 `Car` 클래스를 작성하세요.

### 요구사항

1. **필드**
   - `brand` (String): 자동차 브랜드
   - `model` (String): 자동차 모델
   - `year` (int): 제조년도

2. **생성자**
   - 매개변수가 없는 기본 생성자
   - `brand`, `model`, `year`를 받는 매개변수 생성자

3. **메서드**
   - `toString()`: 자동차 정보를 다음 형식으로 반환
     ```
     "브랜드: [brand], 모델: [model], 제조년도: [year]년"
     ```
   - `getAge(int currentYear)`: 현재 연도를 받아 자동차 나이를 계산하는 메서드
     - 반환값: int (자동차 나이 = 현재 연도 - 제조년도)

### 실행 코드 (제공됨)

```java
public class Problem2 {
    public static void main(String[] args) {
        Car c1 = new Car("현대", "소나타", 2020);
        Car c2 = new Car("기아", "K5", 2022);
        
        System.out.println(c1);
        System.out.println("나이: " + c1.getAge(2024) + "년");
        System.out.println(c2);
        System.out.println("나이: " + c2.getAge(2024) + "년");
    }
}
```

### 실행 결과 예시

```
브랜드: 현대, 모델: 소나타, 제조년도: 2020년
나이: 4년
브랜드: 기아, 모델: K5, 제조년도: 2022년
나이: 2년
```

### 채점 기준
- 필드 선언: 5점
- 생성자 작성: 5점
- toString() 메서드: 5점
- getAge() 메서드: 5점

## 문제 3: ArrayList를 이용한 점수 관리 (20점)

### 문제 설명

ArrayList를 사용하여 학생 점수를 관리하는 프로그램을 작성하세요.

### 요구사항

다음 static 메서드들을 `ScoreManager` 클래스에 작성하세요:

1. **`addScore(ArrayList<Integer> list, int score)`**
   - 점수를 리스트에 추가하는 메서드
   - 반환값: 없음

2. **`getAverage(ArrayList<Integer> list)`**
   - 리스트에 있는 모든 점수의 평균을 계산하는 메서드
   - 반환값: double (평균 점수)
   - 리스트가 비어있으면 0.0 반환

3. **`countPass(ArrayList<Integer> list, int passScore)`**
   - 합격 점수 이상인 점수의 개수를 세는 메서드
   - 반환값: int (합격자 수)
   - 예: `countPass(list, 60)` → 60점 이상인 점수의 개수

4. **`getMaxScore(ArrayList<Integer> list)`**
   - 리스트에서 최고 점수를 찾는 메서드
   - 반환값: int (최고 점수)
   - 리스트가 비어있으면 -1 반환

### 실행 코드 (제공됨)

```java
import java.util.ArrayList;

public class Problem3 {
    public static void main(String[] args) {
        ArrayList<Integer> list = new ArrayList<>();
        
        ScoreManager.addScore(list, 85);
        ScoreManager.addScore(list, 92);
        ScoreManager.addScore(list, 78);
        ScoreManager.addScore(list, 65);
        ScoreManager.addScore(list, 88);
        
        System.out.println("평균: " + ScoreManager.getAverage(list));
        System.out.println("합격자 수 (60점 이상): " + ScoreManager.countPass(list, 60));
        System.out.println("최고 점수: " + ScoreManager.getMaxScore(list));
    }
}
```

### 실행 결과 예시

```
평균: 81.6
합격자 수 (60점 이상): 5
최고 점수: 92
```

### 채점 기준
- addScore() 메서드: 5점
- getAverage() 메서드: 5점
- countPass() 메서드: 5점
- getMaxScore() 메서드: 5점

## 문제 4: 문자열 처리 유틸리티 (20점)

### 문제 설명

문자열을 처리하는 유틸리티 메서드들을 작성하세요.

### 요구사항

다음 static 메서드들을 `StringUtil` 클래스에 작성하세요:

1. **`countVowels(String text)`**
   - 문자열에서 모음(a, e, i, o, u)의 개수를 세는 메서드 (대소문자 구분 없음)
   - 반환값: int (모음 개수)
   - null이면 0 반환
   - 예: `countVowels("Hello")` → 2

2. **`toUpperCase(String text)`**
   - 문자열을 대문자로 변환하는 메서드
   - 반환값: String (대문자로 변환된 문자열)
   - null이면 null 반환
   - 예: `toUpperCase("hello")` → "HELLO"

3. **`containsWord(String text, String word)`**
   - 문자열에 특정 단어가 포함되어 있는지 확인하는 메서드
   - 반환값: boolean (포함되면 true, 아니면 false)
   - null이면 false 반환
   - 예: `containsWord("Hello World", "World")` → true, `containsWord("Hello World", "Java")` → false

4. **`replaceChar(String text, char oldChar, char newChar)`**
   - 문자열에서 특정 문자를 다른 문자로 교체하는 메서드
   - 반환값: String (교체된 문자열)
   - null이면 null 반환
   - 예: `replaceChar("Hello", 'l', 'L')` → "HeLLo"

### 실행 코드 (제공됨)

```java
public class Problem4 {
    public static void main(String[] args) {
        // 1. 모음 개수 세기
        System.out.println("모음 개수: " + StringUtil.countVowels("Hello"));
        
        // 2. 대문자 변환
        System.out.println("대문자: " + StringUtil.toUpperCase("hello world"));
        
        // 3. 단어 포함 확인
        System.out.println("World 포함: " + StringUtil.containsWord("Hello World", "World"));
        System.out.println("Java 포함: " + StringUtil.containsWord("Hello World", "Java"));
        
        // 4. 문자 교체
        System.out.println("문자 교체: " + StringUtil.replaceChar("Hello", 'l', 'L'));
    }
}
```

### 실행 결과 예시

```
모음 개수: 2
대문자: HELLO WORLD
World 포함: true
Java 포함: false
문자 교체: HeLLo
```

### 채점 기준
- countVowels() 메서드: 5점
- toUpperCase() 메서드: 5점
- containsWord() 메서드: 5점
- replaceChar() 메서드: 5점
