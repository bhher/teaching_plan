## 문제 1: Student 클래스 작성 (20점)

### 문제 설명
학생 정보를 저장하는 `Student` 클래스를 작성하세요.

### 요구사항
1. **필드**
   - `name` (String): 학생 이름
   - `age` (int): 학생 나이
   - `score` (int): 학생 점수

2. **생성자**
   - 매개변수가 없는 기본 생성자
   - `name`, `age`, `score`를 받는 매개변수 생성자

3. **메서드**
   - `toString()`: 학생 정보를 다음 형식으로 반환
     ```
     "이름: [name], 나이: [age], 점수: [score]"
     ```
   - `getGrade()`: 점수에 따라 등급을 반환하는 메서드
     - 90점 이상: "A"
     - 80점 이상: "B"
     - 70점 이상: "C"
     - 60점 이상: "D"
     - 60점 미만: "F"

### 실행 코드 (제공됨)
```java
public class Problem1 {
    public static void main(String[] args) {
        Student s1 = new Student("홍길동", 20, 85);
        Student s2 = new Student("김철수", 22, 95);
        
        System.out.println(s1);
        System.out.println("등급: " + s1.getGrade());
        System.out.println(s2);
        System.out.println("등급: " + s2.getGrade());
    }
}
```

### 실행 결과 예시
```
이름: 홍길동, 나이: 20, 점수: 85
등급: B
이름: 김철수, 나이: 22, 점수: 95
등급: A
```

### 채점 기준
- 필드 선언: 5점
- 생성자 작성: 5점
- toString() 메서드: 5점
- getGrade() 메서드: 5점

## 문제 2: ArrayList를 이용한 학생 관리 (20점)

### 문제 설명
ArrayList를 사용하여 학생 정보를 관리하는 프로그램을 작성하세요.

### 요구사항
다음 메서드들을 `StudentManager` 클래스에 작성하세요:

1. **`addStudent(ArrayList<Student> list, String name, int age, int score)`**
   - 학생 정보를 리스트에 추가하는 메서드
   - 반환값: 없음

2. **`findStudent(ArrayList<Student> list, String name)`**
   - 이름으로 학생을 찾는 메서드
   - 찾으면 Student 객체 반환, 못 찾으면 null 반환

3. **`getAverageScore(ArrayList<Student> list)`**
   - 리스트에 있는 모든 학생의 평균 점수를 계산하는 메서드
   - 반환값: double (평균 점수)

4. **`printAll(ArrayList<Student> list)`**
   - 리스트에 있는 모든 학생 정보를 출력하는 메서드
   - 반환값: 없음

### 실행 코드 (제공됨)
```java
import java.util.ArrayList;

public class Problem2 {
    public static void main(String[] args) {
        ArrayList<Student> list = new ArrayList<>();
        
        StudentManager.addStudent(list, "홍길동", 20, 85);
        StudentManager.addStudent(list, "김철수", 22, 95);
        StudentManager.addStudent(list, "이영희", 21, 75);
        
        System.out.println("=== 전체 학생 정보 ===");
        StudentManager.printAll(list);
        
        System.out.println("\n=== 학생 검색 ===");
        Student s = StudentManager.findStudent(list, "김철수");
        if (s != null) {
            System.out.println("찾은 학생: " + s);
        }
        
        System.out.println("\n=== 평균 점수 ===");
        double avg = StudentManager.getAverageScore(list);
        System.out.println("평균: " + avg);
    }
}
```

### 실행 결과 예시
```
=== 전체 학생 정보 ===
이름: 홍길동, 나이: 20, 점수: 85
이름: 김철수, 나이: 22, 점수: 95
이름: 이영희, 나이: 21, 점수: 75

=== 학생 검색 ===
찾은 학생: 이름: 김철수, 나이: 22, 점수: 95

=== 평균 점수 ===
평균: 85.0
```

### 채점 기준
- addStudent() 메서드: 5점
- findStudent() 메서드: 5점
- getAverageScore() 메서드: 5점
- printAll() 메서드: 5점

## 문제 3: 문자열 처리 및 배열 활용 (20점)

### 문제 설명
문자열을 처리하고 배열을 활용하는 메서드들을 작성하세요.

### 요구사항
다음 메서드들을 `StringUtil` 클래스에 작성하세요:

1. **`countWords(String text)`**
   - 문자열에서 단어의 개수를 세는 메서드
   - 단어는 공백으로 구분됨
   - 예: "안녕하세요 자바입니다" → 2개

2. **`reverseString(String text)`**
   - 문자열을 뒤집는 메서드
   - 예: "자바" → "바자"

3. **`findMax(int[] arr)`**
   - 정수 배열에서 최댓값을 찾는 메서드
   - 배열이 비어있으면 -1 반환

4. **`contains(int[] arr, int value)`**
   - 배열에 특정 값이 있는지 확인하는 메서드
   - 있으면 true, 없으면 false 반환

### 실행 코드 (제공됨)
```java
public class Problem3 {
    public static void main(String[] args) {
        // 1. 단어 개수 세기
        String text1 = "안녕하세요 자바입니다";
        System.out.println("단어 개수: " + StringUtil.countWords(text1));
        
        // 2. 문자열 뒤집기
        String text2 = "자바";
        System.out.println("뒤집기: " + StringUtil.reverseString(text2));
        
        // 3. 최댓값 찾기
        int[] arr = {10, 5, 20, 8, 15};
        System.out.println("최댓값: " + StringUtil.findMax(arr));
        
        // 4. 값 포함 여부 확인
        System.out.println("15 포함: " + StringUtil.contains(arr, 15));
        System.out.println("100 포함: " + StringUtil.contains(arr, 100));
    }
}
```

### 실행 결과 예시
```
단어 개수: 2
뒤집기: 바자
최댓값: 20
15 포함: true
100 포함: false
```

### 채점 기준
- countWords() 메서드: 5점
- reverseString() 메서드: 5점
- findMax() 메서드: 5점
- contains() 메서드: 5점

## 문제 4: 종합 문제 - 도서 관리 시스템 (20점)

### 문제 설명
도서 정보를 관리하는 간단한 시스템을 구현하세요.

### 요구사항

#### 1. Book 클래스 작성
- **필드**
  - `title` (String): 도서 제목
  - `author` (String): 저자
  - `price` (int): 가격
- **생성자**: `title`, `author`, `price`를 받는 생성자
- **메서드**: `toString()` - "제목: [title], 저자: [author], 가격: [price]원" 형식

#### 2. BookManager 클래스에 메서드 작성

1. **`addBook(ArrayList<Book> list, String title, String author, int price)`**
   - 도서를 리스트에 추가

2. **`findBooksByAuthor(ArrayList<Book> list, String author)`**
   - 특정 저자의 도서를 모두 찾아서 리스트로 반환
   - 없으면 빈 리스트 반환

3. **`getTotalPrice(ArrayList<Book> list)`**
   - 모든 도서의 총 가격을 계산하여 반환

4. **`removeBook(ArrayList<Book> list, String title)`**
   - 제목으로 도서를 찾아 삭제
   - 성공하면 true, 실패하면 false 반환

### 실행 코드 (제공됨)
```java
import java.util.ArrayList;

public class Problem4 {
    public static void main(String[] args) {
        ArrayList<Book> list = new ArrayList<>();
        
        BookManager.addBook(list, "자바의 정석", "남궁성", 30000);
        BookManager.addBook(list, "혼자 공부하는 자바", "신용권", 28000);
        BookManager.addBook(list, "이것이 자바다", "신용권", 32000);
        
        System.out.println("=== 전체 도서 ===");
        for (Book b : list) {
            System.out.println(b);
        }
        
        System.out.println("\n=== 신용권 저자 도서 ===");
        ArrayList<Book> authorBooks = BookManager.findBooksByAuthor(list, "신용권");
        for (Book b : authorBooks) {
            System.out.println(b);
        }
        
        System.out.println("\n=== 총 가격 ===");
        int total = BookManager.getTotalPrice(list);
        System.out.println("총 가격: " + total + "원");
        
        System.out.println("\n=== 도서 삭제 ===");
        boolean removed = BookManager.removeBook(list, "자바의 정석");
        System.out.println("삭제 결과: " + removed);
        
        System.out.println("\n=== 삭제 후 전체 도서 ===");
        for (Book b : list) {
            System.out.println(b);
        }
    }
}
```

### 실행 결과 예시
