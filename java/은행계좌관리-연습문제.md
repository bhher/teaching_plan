# 은행 계좌 관리 시스템 연습문제

## 문제 설명

StudentManager 코드를 참고하여 **은행 계좌 관리 시스템**을 구현하세요.

### 요구사항

1. **Account 클래스**를 만들어야 합니다.
   - 계좌번호 (accountNumber: String)
   - 예금주명 (ownerName: String)
   - 잔액 (balance: int)

2. **AccountManager 클래스**를 만들어야 합니다.
   - 계좌 목록을 `ArrayList<Account>`로 관리
   - 파일 경로: `c:/Users/TJ/memo/account.txt`
   - 디렉토리 자동 생성 기능 포함

3. **필수 기능**
   - 계좌 생성 (계좌번호, 예금주명, 초기 잔액 입력)
   - 입금 (계좌번호로 찾아서 입금)
   - 출금 (계좌번호로 찾아서 출금, 잔액 부족 시 오류 메시지)
   - 계좌 조회 (계좌번호로 조회)
   - 전체 계좌 조회
   - 계좌 삭제 (계좌번호로 삭제)
   - 파일 저장
   - 파일 불러오기

### 파일 저장 형식

```
계좌번호|예금주명|잔액
```

**예시:**
```
123-456-789|홍길동|100000
987-654-321|김철수|50000
```

### 메뉴 구성

```
=== 은행 계좌 관리 시스템 ===
1. 계좌 생성
2. 입금
3. 출금
4. 계좌 조회
5. 전체 계좌 조회
6. 계좌 삭제
7. 파일 저장
8. 파일 불러오기
9. 종료
선택 > 
```

### 기능 상세 설명

#### 1. 계좌 생성
- 계좌번호, 예금주명, 초기 잔액을 입력받아 계좌를 생성합니다.
- 계좌번호가 이미 존재하면 "이미 존재하는 계좌번호입니다." 메시지를 출력합니다.

#### 2. 입금
- 계좌번호를 입력받아 해당 계좌를 찾습니다.
- 입금할 금액을 입력받아 잔액에 추가합니다.
- 계좌를 찾지 못하면 "계좌를 찾을 수 없습니다." 메시지를 출력합니다.
- 입금 후 자동으로 파일에 저장합니다.

#### 3. 출금
- 계좌번호를 입력받아 해당 계좌를 찾습니다.
- 출금할 금액을 입력받아 잔액에서 차감합니다.
- 잔액이 부족하면 "잔액이 부족합니다." 메시지를 출력합니다.
- 계좌를 찾지 못하면 "계좌를 찾을 수 없습니다." 메시지를 출력합니다.
- 출금 후 자동으로 파일에 저장합니다.

#### 4. 계좌 조회
- 계좌번호를 입력받아 해당 계좌의 정보를 출력합니다.
- 계좌번호, 예금주명, 잔액을 출력합니다.
- 계좌를 찾지 못하면 "계좌를 찾을 수 없습니다." 메시지를 출력합니다.

#### 5. 전체 계좌 조회
- 등록된 모든 계좌의 정보를 출력합니다.
- 계좌가 없으면 "등록된 계좌가 없습니다." 메시지를 출력합니다.

#### 6. 계좌 삭제
- 계좌번호를 입력받아 해당 계좌를 삭제합니다.
- 계좌를 찾지 못하면 "계좌를 찾을 수 없습니다." 메시지를 출력합니다.
- 삭제 후 자동으로 파일에 저장합니다.

#### 7. 파일 저장
- 현재 메모리에 있는 모든 계좌 정보를 파일에 저장합니다.
- 디렉토리가 없으면 자동으로 생성합니다.

#### 8. 파일 불러오기
- 파일에서 계좌 정보를 읽어와 메모리에 로드합니다.
- 파일이 없으면 "파일이 없습니다. 새로 시작합니다." 메시지를 출력합니다.
