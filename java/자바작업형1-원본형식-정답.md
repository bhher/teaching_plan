## 문제 1 정답: Student 클래스

```java
public class Student {
    String name;
    int age;
    int score;
    
    // 기본 생성자
    public Student() {
    }
    
    // 매개변수 생성자
    public Student(String name, int age, int score) {
        this.name = name;
        this.age = age;
        this.score = score;
    }
    
    // toString() 메서드
    @Override
    public String toString() {
        return "이름: " + name + ", 나이: " + age + ", 점수: " + score;
    }
    
    // 등급 반환 메서드
    public String getGrade() {
        if (score >= 90) {
            return "A";
        } else if (score >= 80) {
            return "B";
        } else if (score >= 70) {
            return "C";
        } else if (score >= 60) {
            return "D";
        } else {
            return "F";
        }
    }
}
```

## 문제 2 정답: StudentManager 클래스

```java
import java.util.ArrayList;

public class StudentManager {
    
    // 학생 추가
    public static void addStudent(ArrayList<Student> list, String name, int age, int score) {
        list.add(new Student(name, age, score));
    }
    
    // 학생 검색
    public static Student findStudent(ArrayList<Student> list, String name) {
        for (Student s : list) {
            if (s.name.equals(name)) {
                return s;
            }
        }
        return null;
    }
    
    // 평균 점수 계산
    public static double getAverageScore(ArrayList<Student> list) {
        if (list.isEmpty()) {
            return 0.0;
        }
        
        int sum = 0;
        for (Student s : list) {
            sum += s.score;
        }
        
        return (double) sum / list.size();
    }
    
    // 전체 출력
    public static void printAll(ArrayList<Student> list) {
        for (Student s : list) {
            System.out.println(s);
        }
    }
}
```

## 문제 3 정답: StringUtil 클래스

```java
public class StringUtil {
    
    // 단어 개수 세기
    public static int countWords(String text) {
        if (text == null || text.trim().isEmpty()) {
            return 0;
        }
        
        String[] words = text.trim().split("\\s+");
        return words.length;
    }
    
    // 문자열 뒤집기
    public static String reverseString(String text) {
        if (text == null) {
            return null;
        }
        
        StringBuilder sb = new StringBuilder();
        for (int i = text.length() - 1; i >= 0; i--) {
            sb.append(text.charAt(i));
        }
        return sb.toString();
    }
    
    // 최댓값 찾기
    public static int findMax(int[] arr) {
        if (arr == null || arr.length == 0) {
            return -1;
        }
        
        int max = arr[0];
        for (int i = 1; i < arr.length; i++) {
            if (arr[i] > max) {
                max = arr[i];
            }
        }
        return max;
    }
    
    // 값 포함 여부 확인
    public static boolean contains(int[] arr, int value) {
        if (arr == null) {
            return false;
        }
        
        for (int num : arr) {
            if (num == value) {
                return true;
            }
        }
        return false;
    }
}
```

## 문제 4 정답: Book 클래스 및 BookManager 클래스

```java
import java.util.ArrayList;

// Book 클래스
class Book {
    String title;
    String author;
    int price;
    
    // 생성자
    public Book(String title, String author, int price) {
        this.title = title;
        this.author = author;
        this.price = price;
    }
    
    // toString() 메서드
    @Override
    public String toString() {
        return "제목: " + title + ", 저자: " + author + ", 가격: " + price + "원";
    }
}

// BookManager 클래스
class BookManager {
    
    // 도서 추가
    public static void addBook(ArrayList<Book> list, String title, String author, int price) {
        list.add(new Book(title, author, price));
    }
    
    // 저자로 도서 찾기
    public static ArrayList<Book> findBooksByAuthor(ArrayList<Book> list, String author) {
        ArrayList<Book> result = new ArrayList<>();
        for (Book b : list) {
            if (b.author.equals(author)) {
                result.add(b);
            }
        }
        return result;
    }
    
    // 총 가격 계산
    public static int getTotalPrice(ArrayList<Book> list) {
        int total = 0;
        for (Book b : list) {
            total += b.price;
        }
        return total;
    }
    
    // 도서 삭제
    public static boolean removeBook(ArrayList<Book> list, String title) {
        for (int i = 0; i < list.size(); i++) {
            if (list.get(i).title.equals(title)) {
                list.remove(i);
                return true;
            }
        }
        return false;
    }
}
```

## 추가 설명

### 문제 1 - getGrade() 메서드
- if-else if 구조를 사용하여 점수 범위에 따라 등급 반환
- 조건문의 순서가 중요 (높은 점수부터 확인)

### 문제 2 - getAverageScore() 메서드
- 빈 리스트 체크 필요
- 정수 나눗셈을 피하기 위해 `(double)` 캐스팅 필요

### 문제 2 - findStudent() 메서드
- 향상된 for문 사용
- 문자열 비교는 `equals()` 사용
- 찾지 못하면 null 반환

### 문제 3 - countWords() 메서드
- `split("\\s+")` 사용: 하나 이상의 공백으로 분리
- null 체크 및 빈 문자열 체크 필요

### 문제 3 - reverseString() 메서드
- StringBuilder를 사용하여 문자열 뒤집기
- 또는 charAt()을 사용하여 역순으로 문자 추가

### 문제 4 - findBooksByAuthor() 메서드
- 새로운 ArrayList를 생성하여 결과를 저장
- 원본 리스트를 수정하지 않음
- 조건에 맞는 모든 도서를 찾아 추가

### 문제 4 - removeBook() 메서드
- 인덱스를 사용한 삭제 (일반 for문 필요)
- 향상된 for문 사용 시 ConcurrentModificationException 발생 가능
- 삭제 후 즉시 반환 (break)
