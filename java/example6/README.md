# 6장 예제 파일 실행 방법

## 명령줄에서 실행하기

### 1. MethodBasics.java
```bash
javac MethodBasics.java
java MethodBasics
```

### 2. ParameterReturn.java
```bash
javac ParameterReturn.java
java ParameterReturn
```

### 3. MethodOverloading.java
```bash
javac MethodOverloading.java
java MethodOverloading
```

### 4. StaticMethods.java
```bash
javac StaticMethods.java
java StaticMethods
```

### 5. Calculator.java
```bash
javac Calculator.java
java Calculator
```

### 6. ArrayProcessor.java
```bash
javac ArrayProcessor.java
java ArrayProcessor
```

### 7. MethodComprehensive.java
```bash
javac MethodComprehensive.java
java MethodComprehensive
```

## VS Code에서 실행하기

1. 각 `.java` 파일을 열기
2. `main` 메서드 위에 나타나는 "Run" 버튼 클릭
3. 또는 `F5` 키를 눌러 실행

## 주의사항

- 클래스 이름과 파일 이름이 반드시 일치해야 합니다
- Java는 대소문자를 구분합니다
- 각 파일은 하나의 public 클래스를 포함해야 합니다
- 메서드는 클래스 안에 정의되어야 합니다
- static 메서드는 객체 생성 없이 호출할 수 있습니다
- 메서드 오버로딩은 매개변수의 개수나 타입이 달라야 합니다

## 학습 포인트

- **메서드의 개념**: 코드를 재사용하고 구조화하는 방법
- **매개변수와 반환값**: 메서드에 값을 전달하고 결과를 받는 방법
- **메서드 오버로딩**: 같은 이름의 메서드를 여러 개 정의하는 방법
- **static 메서드**: 객체 생성 없이 호출할 수 있는 메서드
- **메서드 작성 원칙**: 하나의 메서드는 하나의 기능만 수행

## 메서드 작성 팁

1. 메서드명은 동사로 시작
2. 하나의 메서드는 하나의 기능만 수행
3. 매개변수는 적절한 개수로 (3~4개 이하 권장)
4. 반환값은 명확하게
5. 메서드가 너무 길면 여러 메서드로 분리
