# 학생 관리 시스템 - 메뉴 기반 실습 문제

## 📚 문제 설명

학생 정보를 관리하는 메뉴 기반 프로그램을 작성하세요. 
사용자가 메뉴를 선택하여 학생을 등록, 검색, 수정, 삭제, 전체 출력할 수 있도록 구현하세요.

---

## 📋 요구사항

### 1. Student 클래스 작성
- **필드**
  - `name` (String): 학생 이름
  - `age` (int): 학생 나이
- **생성자**
  - `Student(String name, int age)`: 이름과 나이를 받는 생성자
- **메서드**
  - `toString()`: 학생 정보를 문자열로 반환
    - 형식: `"Student{name='이름', age=나이}"`

### 2. StudentApp 클래스 작성

#### 2.1 메뉴 시스템
다음 메뉴를 반복해서 표시하고 사용자 입력을 받아 처리하세요:

```
------------------------
1. 학생 등록
2. 학생 검색
3. 학생 수정
4. 학생 삭제
5. 전체 출력
6. 종료
선택 > 
```

#### 2.2 메뉴별 기능

**1. 학생 등록**
- 이름과 나이를 입력받아 학생을 등록
- 등록 완료 메시지 출력

**2. 학생 검색**
- 검색할 이름을 입력받아 학생을 검색
- 찾으면 학생 정보 출력, 없으면 "학생이 없습니다" 출력

**3. 학생 수정**
- 수정할 이름과 새로운 나이를 입력받아 수정
- 성공하면 "수정 완료", 실패하면 "해당 학생이 없습니다" 출력

**4. 학생 삭제**
- 삭제할 이름을 입력받아 삭제
- 성공하면 "삭제 완료", 실패하면 "삭제할 학생이 없습니다" 출력

**5. 전체 출력**
- 등록된 모든 학생 정보 출력
- 등록된 학생이 없으면 "등록된 학생이 없습니다" 출력

**6. 종료**
- 프로그램 종료

#### 2.3 예외 처리
- 메뉴 선택 시 숫자가 아닌 값 입력 시 "숫자만 입력하세요" 출력 후 다시 메뉴 표시
- 잘못된 메뉴 번호 입력 시 "1~6번 중에 선택하세요" 출력
- 나이 입력 시 숫자가 아닌 값 입력 시 "나이는 숫자로 입력하세요" 출력 후 다시 입력

#### 2.4 필요한 메서드
다음 static 메서드들을 구현하세요:

```java
// 학생 검색
static Student findStudent(ArrayList<Student> list, String name)

// 학생 수정
static boolean updateStudent(ArrayList<Student> list, String name, int newAge)

// 학생 삭제
static boolean deleteStudent(ArrayList<Student> list, String name)
```

## 💡 힌트

1. **Scanner 사용**: `Scanner sc = new Scanner(System.in);`
2. **while 루프**: `boolean run = true;`로 반복 제어
3. **메뉴 입력 처리**: `Integer.parseInt(sc.nextLine())` 사용
4. **예외 처리**: `try-catch`로 숫자 입력 예외 처리
5. **문자열 비교**: `equals()` 메서드 사용
6. **삭제 메서드**: 인덱스가 필요하므로 일반 for문 사용
7. **나이 입력**: `Integer.parseInt()` 사용 시 예외 처리 필요
