# 정보처리기사 파이썬 프로그래밍 언어 정리

## 1. 파이썬 기본 문법

### 1.1 변수와 자료형

**변수 선언**
- 파이썬은 변수 선언 시 자료형을 명시하지 않음 (동적 타입)
- 변수명은 영문자, 숫자, 언더스코어(_)로 구성
- 대소문자 구분

```python
x = 10          # 정수형
y = 3.14        # 실수형
name = "Python" # 문자열
is_true = True  # 불린형
```

**기본 자료형**
- `int`: 정수형
- `float`: 실수형
- `str`: 문자열
- `bool`: 불린형 (True, False)
- `None`: None 타입

**자료형 확인**
```python
type(x)  # 변수의 자료형 확인
```

### 1.2 연산자

**산술 연산자**
- `+`, `-`, `*`, `/`: 덧셈, 뺄셈, 곱셈, 나눗셈
- `//`: 몫 연산자
- `%`: 나머지 연산자
- `**`: 거듭제곱

```python
10 // 3  # 결과: 3
10 % 3   # 결과: 1
2 ** 3   # 결과: 8
```

**비교 연산자**
- `==`, `!=`, `<`, `>`, `<=`, `>=`

**논리 연산자**
- `and`: 논리곱
- `or`: 논리합
- `not`: 논리부정

**할당 연산자**
- `=`, `+=`, `-=`, `*=`, `/=`, `//=`, `%=`, `**=`

### 1.3 문자열 처리

**문자열 생성**
```python
str1 = "Hello"
str2 = 'World'
str3 = """여러 줄
문자열"""
```

**문자열 연산**
```python
"Hello" + "World"  # "HelloWorld"
"Hi" * 3          # "HiHiHi"
```

**문자열 인덱싱과 슬라이싱**
```python
s = "Python"
s[0]      # 'P'
s[-1]     # 'n' (마지막 문자)
s[0:3]    # 'Pyt' (0부터 2까지)
s[:3]     # 'Pyt' (처음부터 2까지)
s[3:]     # 'hon' (3부터 끝까지)
s[::-1]   # 'nohtyP' (역순)
```

**문자열 메서드**
```python
s.upper()        # 대문자 변환
s.lower()        # 소문자 변환
s.strip()        # 양쪽 공백 제거
s.split(',')     # 구분자로 분리
s.replace('a', 'b')  # 문자열 치환
s.find('x')      # 문자열 찾기 (인덱스 반환)
s.count('x')     # 문자열 개수 세기
len(s)           # 문자열 길이
```

**문자열 포맷팅**
```python
# f-string (Python 3.6+)
name = "홍길동"
age = 20
f"이름: {name}, 나이: {age}"

# format() 메서드
"이름: {}, 나이: {}".format(name, age)
"이름: {0}, 나이: {1}".format(name, age)

# % 포맷팅
"이름: %s, 나이: %d" % (name, age)
```

## 2. 제어문

### 2.1 조건문 (if문)

```python
if 조건1:
    실행문1
elif 조건2:
    실행문2
else:
    실행문3
```

**예제**
```python
score = 85
if score >= 90:
    grade = 'A'
elif score >= 80:
    grade = 'B'
else:
    grade = 'C'
```

**삼항 연산자**
```python
result = "합격" if score >= 60 else "불합격"
```

### 2.2 반복문

**for문**
```python
# 리스트 순회
for item in [1, 2, 3]:
    print(item)

# range() 사용
for i in range(5):        # 0부터 4까지
    print(i)

for i in range(1, 6):     # 1부터 5까지
    print(i)

for i in range(0, 10, 2): # 0부터 9까지 2씩 증가
    print(i)

# enumerate() 사용 (인덱스와 값 함께)
for idx, value in enumerate(['a', 'b', 'c']):
    print(idx, value)
```

**while문**
```python
i = 0
while i < 5:
    print(i)
    i += 1
```

**반복문 제어**
- `break`: 반복문 종료
- `continue`: 다음 반복으로 건너뛰기
- `else`: 반복문이 정상 종료될 때 실행

```python
for i in range(5):
    if i == 3:
        break
    print(i)
else:
    print("반복 완료")  # break로 종료되면 실행 안됨
```

## 3. 자료구조

### 3.1 리스트 (List)

**리스트 생성**
```python
list1 = [1, 2, 3]
list2 = list(range(5))
list3 = []  # 빈 리스트
```

**리스트 메서드**
```python
list.append(x)      # 끝에 추가
list.insert(i, x)   # i번째 위치에 삽입
list.remove(x)      # 첫 번째 x 제거
list.pop()          # 마지막 요소 제거 및 반환
list.pop(i)         # i번째 요소 제거 및 반환
list.index(x)       # x의 인덱스 반환
list.count(x)       # x의 개수
list.sort()         # 정렬 (원본 변경)
list.reverse()      # 역순 정렬
list.extend(iter)   # 리스트 확장
len(list)           # 리스트 길이
```

**리스트 컴프리헨션**
```python
# 기본 형태
[x for x in range(5)]  # [0, 1, 2, 3, 4]

# 조건 포함
[x for x in range(10) if x % 2 == 0]  # [0, 2, 4, 6, 8]

# 중첩
[(x, y) for x in range(3) for y in range(2)]
```

### 3.2 튜플 (Tuple)

**튜플 생성**
```python
tuple1 = (1, 2, 3)
tuple2 = 1, 2, 3  # 괄호 생략 가능
tuple3 = (1,)      # 단일 요소 튜플 (쉼표 필요)
```

**특징**
- 불변(immutable) 자료형
- 리스트와 달리 수정 불가
- 인덱싱, 슬라이싱 가능

### 3.3 딕셔너리 (Dictionary)

**딕셔너리 생성**
```python
dict1 = {'name': '홍길동', 'age': 20}
dict2 = dict(name='홍길동', age=20)
dict3 = {}  # 빈 딕셔너리
```

**딕셔너리 메서드**
```python
dict.keys()        # 모든 키 반환
dict.values()     # 모든 값 반환
dict.items()      # (키, 값) 쌍 반환
dict.get(key)     # 키로 값 가져오기 (없으면 None)
dict.get(key, default)  # 기본값 지정
dict.pop(key)     # 키로 항목 제거 및 반환
dict.update(other) # 다른 딕셔너리로 업데이트
'key' in dict     # 키 존재 확인
```

**딕셔너리 순회**
```python
for key in dict:
    print(key, dict[key])

for key, value in dict.items():
    print(key, value)
```

### 3.4 집합 (Set)

**집합 생성**
```python
set1 = {1, 2, 3}
set2 = set([1, 2, 3])
set3 = set()  # 빈 집합
```

**집합 연산**
```python
set1 | set2   # 합집합
set1 & set2   # 교집합
set1 - set2   # 차집합
set1 ^ set2   # 대칭차집합
```

**집합 메서드**
```python
set.add(x)        # 요소 추가
set.remove(x)     # 요소 제거 (없으면 에러)
set.discard(x)    # 요소 제거 (없어도 에러 없음)
set.pop()         # 임의 요소 제거 및 반환
set.clear()       # 모든 요소 제거
```

### 4.2 매개변수

```python
def greet(name, message="안녕하세요"):
    return f"{message}, {name}!"

greet("홍길동")  # "안녕하세요, 홍길동!"
greet("홍길동", "반갑습니다")  # "반갑습니다, 홍길동!"
```

**가변 매개변수**
```python

    return sum(args)

# **kwargs: 딕셔너리로 받음
def print_info(**kwargs):
    for key, value in kwargs.items():
        print(f"{key}: {value}")

print_info(name="홍길동", age=20)
```

### 4.3 람다 함수

```python
# 일반 함수
def square(x):
    return x ** 2

# 람다 함수
square = lambda x: x ** 2

# 사용 예
result = square(5)  # 25

# map, filter와 함께 사용
numbers = [1, 2, 3, 4, 5]
squared = list(map(lambda x: x**2, numbers))  # [1, 4, 9, 16, 25]
evens = list(filter(lambda x: x % 2 == 0, numbers))  # [2, 4]
```

## 5. 파일 입출력

### 5.1 파일 읽기

```python
# 파일 열기
file = open("file.txt", "r", encoding="utf-8")
content = file.read()  # 전체 읽기
file.close()

# with 문 사용 (자동으로 파일 닫기)
with open("file.txt", "r", encoding="utf-8") as file:
    content = file.read()
    lines = file.readlines()  # 줄 단위로 읽기
    for line in file:          # 줄 단위로 순회
        print(line.strip())
```

**파일 모드**
- `"r"`: 읽기 모드
- `"w"`: 쓰기 모드 (덮어쓰기)
- `"a"`: 추가 모드
- `"x"`: 생성 모드 (파일이 없을 때만)
- `"b"`: 바이너리 모드

### 5.2 파일 쓰기

```python
with open("file.txt", "w", encoding="utf-8") as file:
    file.write("첫 번째 줄\n")
    file.write("두 번째 줄\n")
    file.writelines(["세 번째 줄\n", "네 번째 줄\n"])
```

## 6. 예외 처리

### 6.1 try-except

```python
try:
    result = 10 / 0
except ZeroDivisionError:
    print("0으로 나눌 수 없습니다")
except Exception as e:
    print(f"에러 발생: {e}")
else:
    print("에러가 없을 때 실행")
finally:
    print("항상 실행")
```

**예외 종류**
- `ZeroDivisionError`: 0으로 나누기
- `ValueError`: 잘못된 값
- `TypeError`: 잘못된 타입
- `IndexError`: 인덱스 범위 초과
- `KeyError`: 딕셔너리 키 없음
- `FileNotFoundError`: 파일 없음

### 6.2 예외 발생시키기

```python
if age < 0:
```

## 10. 기타 중요 개념

### 10.1 리스트 vs 튜플 vs 딕셔너리 vs 집합

| 자료형 | 순서 | 중복 | 수정 | 용도 |
|--------|------|------|------|------|
| 리스트 | O | O | O | 순서 있는 데이터 |
| 튜플 | O | O | X | 불변 데이터 |
| 딕셔너리 | X | 키: X, 값: O | O | 키-값 쌍 |
| 집합 | X | X | O | 중복 제거, 집합 연산 |

### 10.2 얕은 복사 vs 깊은 복사

```python
# 얕은 복사
list1 = [1, 2, 3]
list2 = list1[:]  # 또는 list(list1)
list2[0] = 100
# list1은 변경되지 않음

# 깊은 복사 (중첩 리스트)
import copy
list1 = [[1, 2], [3, 4]]
list2 = copy.deepcopy(list1)
```

### 10.3 문자열과 리스트 변환

```python
# 문자열 → 리스트
s = "hello"
list(s)  # ['h', 'e', 'l', 'l', 'o']
s.split()  # 공백으로 분리

# 리스트 → 문자열
list1 = ['h', 'e', 'l', 'l', 'o']
''.join(list1)  # "hello"
','.join(list1)  # "h,e,l,l,o"
```
