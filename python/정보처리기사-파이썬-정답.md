## 상세 해설

### 문제 1
```python
x = 10
y = 3
print(x // y, x % y)  # 3 1
```
- `//`: 몫 연산자 (10 // 3 = 3)
- `%`: 나머지 연산자 (10 % 3 = 1)

### 문제 2
```python

```
- 슬라이싱: 인덱스 1부터 3까지 (4는 포함 안됨)

### 문제 3
```python

```
- `list2 = list1`은 참조 복사이므로 같은 객체를 가리킴

## 주요 개념 정리

### 1. 리스트 복사
- `list2 = list1`: 참조 복사 (같은 객체)
- `list2 = list1[:]`: 얕은 복사 (새 객체)
- `list2 = list1.copy()`: 얕은 복사
- `list2 = list1 + []`: 새 리스트 생성

### 2. 문자열 슬라이싱
- `s[start:end]`: start부터 end-1까지
- `s[:end]`: 처음부터 end-1까지
- `s[start:]`: start부터 끝까지
- `s[::-1]`: 역순

### 3. 딕셔너리 접근
- `dict[key]`: 키 없으면 KeyError
- `dict.get(key)`: 키 없으면 None
- `dict.get(key, default)`: 키 없으면 default 반환

### 4. 리스트 메서드
- `append(x)`: 끝에 추가
- `extend(iter)`: 리스트 확장
- `insert(i, x)`: i번째에 삽입
- `remove(x)`: 첫 번째 x 제거
- `pop()`: 마지막 제거 및 반환
- `sort()`: 원본 정렬
- `reverse()`: 원본 역순

### 5. 반복문 제어
- `break`: 반복문 종료
- `continue`: 다음 반복으로
- `else`: 정상 종료 시 실행

### 6. 함수 매개변수
- 기본값: `def func(x=10)`
- 가변: `def func(*args)` (튜플)
- 키워드: `def func(**kwargs)` (딕셔너리)
