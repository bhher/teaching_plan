# 7단계. 최종 프로젝트

## 학습 목표
- React의 모든 개념을 종합하여 프로젝트를 완성할 수 있다
- 게시판 CRUD 기능을 구현할 수 있다
- 로그인/로그아웃 기능을 구현할 수 있다
- API 연동을 완료할 수 있다
- 프로젝트를 배포할 수 있다

---

## 프로젝트 개요

### 필수 기능

1. **게시판 (CRUD)**
   - 게시글 목록 조회
   - 게시글 작성
   - 게시글 수정
   - 게시글 삭제

2. **로그인/로그아웃**
   - 로그인 기능
   - 로그아웃 기능
   - 인증 상태 관리

3. **API 연동**
   - REST API 호출
   - 데이터 CRUD

4. **배포**
   - Vercel 배포
   - 환경 변수 설정

## 단계별 구현

### 1단계: 프로젝트 설정

```bash
npm create vite@latest my-blog -- --template react
cd my-blog
npm install
npm install react-router-dom axios
```

### 2단계: 기본 구조

- 라우팅 설정
- 네비게이션 컴포넌트
- 기본 페이지들

### 3단계: API 연동

- API 함수 작성
- 게시글 CRUD 구현

### 4단계: 인증 기능

- AuthContext 생성
- 로그인/로그아웃 구현

### 5단계: 스타일링

- CSS 또는 styled-components
- 반응형 디자인

### 6단계: 배포

- 빌드
- Vercel 배포
- 환경 변수 설정
